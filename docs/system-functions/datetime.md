# Дата и время

в языке EME-L функции группы DateTime отвечают за работу с датами, временем, календарными периодами и часовыми поясами. в системе EME.WMS эти функции используются для форматирования дат, расчета периодов, преобразования между временными зонами и извлечения компонентов даты/времени.

### is_fdatetime — форматирование даты/времени

в EME БД функция `is_fdatetime` форматирует дату/время в строку согласно заданному формату или выполняет обратное преобразование строки в дату/время.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | DateTime / String | Да | Дата/время для форматирования, или текстовая строка для обратного преобразования |
| args[1] | String | Нет | Строка формата вывода даты/времени. При обратном преобразовании задает формат входной строки. Если не указан, дата выводится прописью в формате DD месяца YYYY г. (по умолчанию) |

**Возвращает**: `String` — отформатированная строка даты/времени, или `DateTime` при обратном преобразовании.

```EME-L
' Форматировать дату прописью: 15 марта 2025 г. '
TextDate = is_fdatetime(Dt)

' Форматировать с явным форматом '
Formatted = is_fdatetime(Dt, "%d.%m.%Y %H:%M")

' Обратное преобразование строки в дату '
Dt = is_fdatetime("15.03.2025", "%d.%m.%Y")
```

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Год для проверки |

**Возвращает**: `Integer` — 1, если год високосный; 0 — если невисокосный.

```EME-L

' Leap = 1 '

' Leap = 1 - 2000 делится на 400 '
```

### is_parsing_date — разбор строки даты

в EME БД функция `is_parsing_date` преобразует текстовую строку даты в значение типа дата/время.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Строка даты для разбора |
| args[1] | String | Нет | Формат входной строки: ddmmyyyy (по умолчанию), yyyymmdd, yyyy-mm-dd, dd-mm-yyyy, dd.mm.yyyy, ddmmyy |
| args[2] | Integer | Нет | Формат возвращаемого значения: 0 — строка DD.MM.YYYY, иное — дата (по умолчанию) |

**Возвращает**: `Date` или `String` — разобранная дата.

```EME-L
' Разбор даты в формате ddmmyyyy '
Dt = is_parsing_date("15032025")

' Разбор даты в формате yyyymmdd '
Dt = is_parsing_date("20250315", "yyyymmdd")

' Разбор даты с дефисами '
Dt = is_parsing_date("15-03-2025", "dd-mm-yyyy")

' Вернуть строкой '
Str = is_parsing_date("15032025", "ddmmyyyy", 0)
' Str = "15.03.2025" '
```
