# Сообщения и диалоги

> Synonyms for search: сообщения и диалоги, is_msg, is_message, is_message_beep, is_timer_message, is_input, is_multiinput, is_get_password, is_show_popup_window, is_show_email_icon, MessageBox, звук, email, ввод текста, всплывающее окно

### is_msg — текст сообщения проекта

Возвращает текст сообщения проекта по его имени из словаря сообщений. Выполняет поиск через GetProjectMessage.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя сообщения проекта для поиска в словаре |

**Возвращает**: `String` — текст найденного сообщения проекта. Если сообщение не найдено, возвращает пустую строку.

```EME-L
text = is_msg("MessageName")
```

### is_message — вывод MessageBox

Выводит модальное диалоговое окно MessageBox с заданным заголовком, текстом, набором кнопок, иконкой и опциональным таймаутом. Поддерживает как числовые, так и строковые значения для кнопок и иконок. В системном режиме (systemMode = TRUE) диалог выводится в любом потоке, включая транзакции (в этом случае таймаут должен быть > 0).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Заголовок окна. При отсутствии — пустой заголовок |
| args[1] | String | Нет | Текст сообщения. При отсутствии — пустое сообщение |
| args[2] | Integer/String | Нет | Набор кнопок (по умолчанию 2 / OKCANCEL). Числа: 1 = OK, 2 = OKCANCEL, 3 = YESNOCANCEL, 4 = YESNO, 5 = YES_NO_YESTOALL_NOTOALL_CANCEL. Строки: "OK", "OKCANCEL", "YESNOCANCEL", "YESNO", "ABORTRETRYIGNORE", "RETRYCANCEL" |
| args[3] | Integer/String | Нет | Тип иконки (по умолчанию INFORMATION). Числа: 0 = INFORMATION, 1 = EXCLAMATION, 2 = QUESTION, 3 = STOP. Строки: "INFORMATION", "EXCLAMATION", "QUESTION", "STOP" |
| args[4] | Integer | Нет | Номер кнопки по умолчанию (по умолчанию 0): 0 = первая кнопка, 1 = вторая кнопка, 2 = третья кнопка |
| args[5] | Boolean | Нет | Системный режим (по умолчанию FALSE). Если TRUE, диалог выводится в любом потоке и в транзакции |
| args[6] | Integer | Нет | Таймаут в миллисекундах (по умолчанию 0 — бесконечность). При истечении таймаута возвращается значение кнопки по умолчанию |

**Возвращает**: при текстовом args[2] — `String` с именем нажатой кнопки ("OK", "CANCEL", "ABORT", "RETRY", "IGNORE", "YES", "NO", "CLOSE", "HELP"). При YESNOCANCEL (3) и числовом args[2] — `Integer` (Yes = -1, No = 0, Cancel = 1). При YES_NO_YESTOALL_NOTOALL_CANCEL (5) — `String` ("YES", "NO", "YESTOALL", "NOTOALL", "CANCEL"). В остальных случаях — `Boolean` (OK/Yes = TRUE, Cancel/No = FALSE).

```EME-L
' Показать MessageBox с заголовком и текстом, TRUE — нажали OK '
result = is_message("Заголовок", "Текст сообщения")

' Диалог «Да/Нет/Отмена» с иконкой вопроса, YES=-1, NO=0, Cancel=1 '
result = is_message("Вопрос", "Продолжить?", 3, 2)

' Диалог «Да/Нет» с иконкой вопроса, строковые параметры, результат «YES» или «NO» '
result = is_message("Заголовок", "Текст", "YESNO", "QUESTION")
```

### is_message_beep — звуковой сигнал

Воспроизводит системный звуковой сигнал через API MessageBeep. Поддерживает как числовые, так и строковые идентификаторы типа звука.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer/String | Да | Тип звукового сигнала. Числа: 0 = ICONASTERISK, 1 = ICONEXCLAMATION, 2 = ICONERROR, 3 = ICONHAND, 4 = ICONINFORMATION, 5 = ICONQUESTION, 6 = ICONSTOP, 7 = OK. Строки: "ICONASTERISK", "ICONEXCLAMATION", "ICONERROR", "ICONHAND", "ICONINFORMATION", "ICONQUESTION", "ICONSTOP", "OK" |

**Возвращает**: `Integer` — результат вызова MessageBeep (ненулевое при успехе).

```EME-L
is_message_beep("ICONEXCLAMATION")
is_message_beep(0)
```

### is_timer_message — MessageBox с автозакрытием

Выводит модальное диалоговое окно MessageBox с заданным заголовком, текстом, набором кнопок, иконкой и автоматическим закрытием по таймауту. Поддерживает как числовые, так и строковые значения для кнопок и иконок. В системном режиме (systemMode = TRUE) диалог выводится в любом потоке, включая транзакции (в этом случае таймаут должен быть > 0). При истечении таймаута окно закрывается автоматически и возвращается значение кнопки по умолчанию.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Заголовок окна. При отсутствии — пустой заголовок |
| args[1] | String | Нет | Текст сообщения. При отсутствии — пустое сообщение |
| args[2] | Integer/String | Нет | Набор кнопок (по умолчанию 2 / OKCANCEL). Числа: 1 = OK, 2 = OKCANCEL, 3 = YESNOCANCEL, 4 = YESNO, 5 = YES_NO_YESTOALL_NOTOALL_CANCEL. Строки: "OK", "OKCANCEL", "YESNOCANCEL", "YESNO", "ABORTRETRYIGNORE", "RETRYCANCEL" |
| args[3] | Integer/String | Нет | Тип иконки (по умолчанию INFORMATION). Числа: 0 = INFORMATION, 1 = EXCLAMATION, 2 = QUESTION, 3 = STOP. Строки: "INFORMATION", "EXCLAMATION", "QUESTION", "STOP" |
| args[4] | Integer | Нет | Номер кнопки по умолчанию (по умолчанию 0): 0 = первая кнопка, 1 = вторая кнопка, 2 = третья кнопка |
| args[5] | Boolean | Нет | Системный режим (по умолчанию FALSE). Если TRUE, диалог выводится в любом потоке и в транзакции |
| args[6] | Integer | Нет | Таймаут в миллисекундах (по умолчанию 10000). При значении 0 — бесконечность (без автозакрытия) |

**Возвращает**: при текстовом args[2] — `String` с именем нажатой кнопки ("OK", "CANCEL", "ABORT", "RETRY", "IGNORE", "YES", "NO", "CLOSE", "HELP"). При YESNOCANCEL (3) и числовом args[2] — `Integer` (Yes = -1, No = 0, Cancel = 1). При YES_NO_YESTOALL_NOTOALL_CANCEL (5) — `String` ("YES", "NO", "YESTOALL", "NOTOALL", "CANCEL"). В остальных случаях — `Boolean` (OK/Yes = TRUE, Cancel/No = FALSE). При срабатывании таймаута возвращается значение кнопки по умолчанию (args[4]).

```EME-L
result = is_timer_message("Заголовок", "Автозакрытие через 5с", 2, 0, 0, FALSE, 5000)
```

### is_input — ввод однострочного текста

Выводит диалоговое окно для ввода однострочного текста с кнопками OK и Отмена. Вызов InputStringBox.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Подсказка (prompt), отображаемая над полем ввода |
| args[1] | String | Да | Стартовое значение, предзаполняющее поле ввода |
| args[2] | String | Нет | Заголовок окна диалога (по умолчанию "EME") |

**Возвращает**: `String` — введённая пользователем строка на нажатие OK. Пустую строку на нажатие Отмена или при ошибке.

```EME-L
result = is_input("Введите значение:", "по умолчанию", "Мой диалог")
```

### is_multiinput — ввод многострочного текста

Выводит диалоговое окно для ввода многострочного текста с кнопками OK и Отмена. Аналогична is_input, но использует многострочное поле ввода. Вызов InputMessageText.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Подсказка (prompt), отображаемая над полем ввода |
| args[1] | String | Да | Стартовое значение, предзаполняющее поле ввода |
| args[2] | String | Нет | Заголовок окна диалога (по умолчанию "EME") |

**Возвращает**: `String` — введённая пользователем многострочная строка на нажатие OK. Пустую строку на нажатие Отмена или при ошибке.

```EME-L
text = is_multiinput("Введите комментарий:", "", "Комментарий")
```

### is_get_password — ввод пароля

Выводит диалоговое окно для ввода пароля с маскированием вводимых символов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Заголовок диалогового окна |
| args[1] | String | Да | Текст-подсказка, отображаемый в диалоге |

**Возвращает**: `String` — введённый пользователем пароль. Если поддержка диалога ввода пароля отсутствует, возвращает пустое значение.

```EME-L
pwd = is_get_password("Авторизация", "Введите пароль:")
```

### is_show_popup_window — всплывающее уведомление

Показывает всплывающее окно в правой нижней части экрана. Вызов EMEShowPopupWindow.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Текст сообщения |
| args[1] | String | Да | Имя запускаемого диалога при нажатии |
| args[2] | Integer | Да | Номер строки диалога |
| args[3] | Integer | Да | Код запуска (OVL-код) |

**Возвращает**: пустое значение.

```EME-L
is_show_popup_window("Новое задание", "DialogOrders", 1, 0)
```

### is_show_email_icon — иконка email

Управляет отображением иконки email в статусной строке главного окна.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Boolean | Да | TRUE — показать иконку, FALSE — скрыть |

**Возвращает**: пустое значение.

```EME-L
is_show_email_icon(TRUE)
```
