# Права доступа и пользователи

> Synonyms for search: права доступа, пользователи, is_administrator, is_access, is_password, is_change_password, is_set_user_read_only, is_clr_user_read_only, is_disconnect_user, is_impersonate_user, is_security_event, администратор, пароль, безопасность

### is_administrator — проверка администратора

Проверяет, обладает ли текущий оператор правами администратора. Вызов `user_as_administrator`.

**Параметры**: нет.

**Возвращает**: `Boolean` — `TRUE`, если текущий оператор — администратор, иначе `FALSE`.

```EME-L
' Проверить, является ли оператор администратором (TRUE — администратор) '
result = is_administrator()
```

---

### is_access — проверка прав доступа

Проверяет наличие у текущего оператора права доступа по указанному биту в 32-битном поле флагов доступа. Вызов `get_flag_access` с маской `1 << bitNo`.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Номер бита в поле доступа (0..31) |

**Возвращает**: `Boolean` — `TRUE`, если бит установлен (оператор имеет право), иначе `FALSE`.

```EME-L
' Проверить право доступа по биту 5 (TRUE — есть право) '
result = is_access(5)
```

---

### is_password — диалог ввода пароля

Выводит диалог ввода пароля для идентификации текущего оператора. Вызов `password(TRUE)`.

**Параметры**: нет.

**Возвращает**: `Boolean` — `TRUE` при успешной идентификации, `FALSE` при ошибке.

```EME-L
' Проверить пароль оператора (TRUE — пароль верный) '
result = is_password()
```

---

### is_change_password — смена пароля

Выводит диалог смены пароля для указанного пользователя.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Нет | Ссылка на пользователя (целое число, по умолчанию `NULL_REF` — текущий пользователь) |
| args[1] | Boolean | Нет | `TRUE` — требовать ввод старого пароля (по умолчанию `TRUE`) |

**Возвращает**: пустое значение.

```EME-L
' Смена пароля текущего пользователя '
is_change_password()
' С указанием пользователя '
is_change_password(user_ref, TRUE)
```

---

### is_set_user_read_only — режим только чтение

Устанавливает в памяти у текущего пользователя флаг «только чтение» (бит номер 1 в поле 32 бита допусков). Изменение не сохраняется в БД. При установленном флаге запись в базу данных запрещена.

**Параметры**: нет.

**Возвращает**: пустое значение.

```EME-L
is_set_user_read_only()
```

---

### is_clr_user_read_only — снять режим только чтение

Снимает в памяти у текущего пользователя флаг «только чтение» (бит номер 1 в поле 32 бита допусков). Изменение не сохраняется в БД. При снятом флаге запись в базу данных разрешена.

**Параметры**: нет.

**Возвращает**: пустое значение.

```EME-L
is_clr_user_read_only()
```

---

### is_disconnect_user — отключение пользователя

Отправляет на все рабочие станции команду отключить указанного пользователя. Вызов `DisconnectUser`.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Ссылка на пользователя в записи «Список пользователей» |

**Возвращает**: пустое значение.

```EME-L
is_disconnect_user(user_ref)
```

---

### is_impersonate_user — установка пользователя

Устанавливает текущего пользователя в неинтерфейсном потоке (HTTP, CLR и т.д.). Вызов `impersonate_user`.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Ссылка на пользователя в записи «Список пользователей» |

**Возвращает**: пустое значение.

```EME-L
is_impersonate_user(admin_ref)
```

---

### is_security_event — событие безопасности

Генерирует произвольное событие информационной безопасности. Обработка события выполняется в методе `OnSecurityEvent` класса `Resident`.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Ссылка на пользователя |
| args[1] | Integer | Да | Код события |
| args[2] | String | Да | Описание события |

**Возвращает**: пустое значение.

```EME-L
is_security_event(user_ref, 1001, "Попытка несанкционированного доступа")
```
