# Строковые функции

> Synonyms for search: строки, is_string, is_regexp, is_find, is_replace, is_format, is_upper, is_lower, is_trim, is_split, is_join, is_length, is_char, is_hex, is_base64, is_SHA1Sum, is_urlencode, is_urldecode, is_mid, is_left, is_right, is_piece, is_wildcmp, is_masked, is_stemplate, is_count_CRC, подстрока, поиск, замена, кодировка, хеш, регулярное выражение

в языке EME-L строковые функции обеспечивают манипуляцию текстом: поиск, замену, форматирование, преобразование регистра, кодировки, хеширование и работу с подстроками. в системе EME.WMS строковые функции используются для обработки данных штрих-кодов, формирования отчётов, валидации ввода и подготовки данных для EDI-обмена.

### is_regexp — проверка по регулярному выражению

проверяет, содержит ли строка подстроку, соответствующую регулярному выражению. При наличии третьего параметра заполняет массив найденными подстроками (до 10 элементов).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка для проверки |
| args[1] | String | Да | Шаблон регулярного выражения |
| args[2] | CIArray | Нет | Массив для сохранения найденных подстрок |

**Возвращает**: `Boolean` — результат проверки соответствия:
- `TRUE` — подстрока, соответствующая регулярному выражению, найдена
- `FALSE` — соответствие не найдено или исходная строка пуста

```EME-L
' Проверка формата email '
Result = is_regexp("user@example.com", "^[\w.-]+@[\w.-]+\.\w+$")
' Result = TRUE '

' Извлечение групп совпадений '
Dim arr(10)
Found = is_regexp("Order #12345", "#(\d+)", arr)
' Found = TRUE, arr[0] = "12345" '
```

### is_regexparr — поиск всех вхождений регулярного выражения

выполняет поиск всех вхождений регулярного выражения в строке с использованием синтаксиса QRegExp Qt и возвращает массив найденных подстрок.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка для поиска |
| args[1] | String | Да | Регулярное выражение в синтаксисе QRegExp |
| args[2] | Boolean | Нет | Если `TRUE`, используется минимальный (ленивый) поиск. По умолчанию `FALSE` — жадный, максимальный поиск |

**Возвращает**: `CIArray` — массив найденных подстрок.

```EME-L
' Поиск всех чисел в строке '
Dim Numbers()
Numbers = is_regexparr("Order 100, Item 200, Total 300", "\d+")
' Numbers = ["100", "200", "300"] '

' Ленивый поиск '
Dim Tags()
Tags = is_regexparr("<b>bold</b><i>italic</i>", "<.*>", TRUE)
' Tags = ["<b>", "</b>", "<i>", "</i>"] — при ленивом режиме '
```

### is_regexparrpos — поиск вхождений с позициями

выполняет поиск всех вхождений регулярного выражения в строке и возвращает массив найденных подстрок, а также заполняет переданный массив позициями вхождений.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка для поиска |
| args[1] | String | Да | Регулярное выражение в синтаксисе QRegExp |
| args[2] | CIArray | Да | Массив для сохранения позиций найденных вхождений |
| args[3] | Boolean | Нет | Если `TRUE`, используется минимальный поиск. По умолчанию `FALSE` |
| args[4] | Boolean | Нет | Если `FALSE`, регистр не учитывается. По умолчанию `TRUE` |
| args[5] | Integer | Нет | Индекс захватываемой группы. По умолчанию 0 |

**Возвращает**: `CIArray` — массив найденных подстрок.

```EME-L
' Поиск с сохранением позиций '
Dim Results()
Dim Positions()
Results = is_regexparrpos("ABC def GHI", "[A-Z]{3}", Positions)
' Results = ["ABC", "GHI"], Positions = [0, 8] '
```

заменяет часть строки, начиная с указанной позиции, на заданную подстроку.

```EME-L
' Result = "A123BC" '
```

### is_masked — проверка по маске символов

проверяет соответствие строки заданной маске. В маске в фигурных скобках `{}` задаётся допустимый набор символов, в круглых скобках `()` определяется собственно маска. Доступна только маска `*` — все символы должны быть из допустимого набора.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка для проверки |
| args[1] | String | Да | Строка маски с описанием допустимых символов и шаблона |

**Возвращает**: `Boolean` — результат проверки:
- `TRUE` — строка соответствует маске
- `FALSE` — строка не соответствует маске или маска задана неверно

```EME-L
' Проверка, что строка содержит только цифры '
Result = is_masked("12345", "{ '0'-'9' }(*)")
' Result = TRUE '

' Проверка на буквы и цифры '
Result = is_masked("A1B2", "{ 'A'-'Z', '0'-'9' }(*)")
' Result = TRUE '
```

### is_char — преобразование символа и кода

выполняет преобразование между символом и его кодом. Если аргумент — строка, возвращает код первого символа (0-255). Если аргумент — целое число, возвращает строку из одного или нескольких символов с указанным кодом.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String или Integer | Да | Исходная строка или целочисленный код символа |
| args[1] | Integer | Нет | Количество символов в результирующей строке при преобразовании числа в строку. По умолчанию 1 |

**Возвращает**:
- При строковом аргументе — `Integer` — код первого символа (диапазон 0-255)
- При числовом аргументе — `String` — строка из указанного количества символов

```EME-L
' Код символа '
Code = is_char("A")
' Code = 65 '

' Символ по коду '
Symbol = is_char(65)
' Symbol = "A" '

' Несколько символов по коду '
Symbols = is_char(65, 3)
' Symbols = "AAA" '

' Проверка обратимости '
Result = is_char(is_char("X"))
' Result = "X" '
```

### is_trim_token — удаление токенов из строки

удаляет из строки указанное количество токенов, разделённых заданным символом. При отрицательном количестве удаление выполняется с конца строки.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка |
| args[1] | String | Да | Символ-разделитель токенов |
| args[2] | Integer | Да | Количество удаляемых токенов. Если < 0 — удаление с конца строки |

**Возвращает**: `String` — строка после удаления указанных токенов.

```EME-L
' Удаление первых двух токенов '
Result = is_trim_token("a;b;c;d", ";", 2)
' Result = "c;d" '

' Удаление последнего токена '
Result = is_trim_token("a;b;c;d", ";", -1)
' Result = "a;b;c" '
```

### is_format — форматирование строки

формирует строку по заданному формату, аналогично функции `printf`. Поддерживаются аргументы типа строка, целое число и вещественное число.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Форматная строка в спецификации printf |
| args[1] | String/Integer/Real | Нет | Значение для подстановки |
| args[2] | String/Integer/Real | Нет | Значение для подстановки |
| ... | ... | Нет | Дополнительные значения |

**Возвращает**: `String` — отформатированная строка.

```EME-L
' Форматирование числа '
Result = is_format("Value: %d", 42)
' Result = "Value: 42" '

' Несколько аргументов '
Result = is_format("Name: %s, Age: %d", "Ivan", 30)
' Result = "Name: Ivan, Age: 30" '

' Вещественное число с точностью '
Result = is_format("Pi = %.2f", 3.14159)
' Result = "Pi = 3.14" '
```
