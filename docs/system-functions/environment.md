# Окружение и информация о системе

> Synonyms for search: окружение, система, is_computer_name, is_domain_name, is_domain_dc, is_domain_username, is_server, is_http_server, is_brain_computer, is_workstation_mode, is_workstation_type, is_system_family, is_db_driver_name, is_get_tick_count, is_keycode, is_key_press, имя компьютера, домен, рабочая станция, сервер, драйвер БД

### is_computer_name — сетевое имя компьютера

Возвращает сетевое имя локального компьютера. Вызов GetComputerName.

**Параметры**: нет.

**Возвращает**: `String` — имя компьютера.

```EME-L
name = is_computer_name()
```

---

### is_domain_name — имя домена

Возвращает имя домена текущего пользователя операционной системы. Вызов GetCurrentUserDomainInfo.

**Параметры**: нет.

**Возвращает**: `String` — имя домена. В случае ошибки возвращает пустое значение.

```EME-L
domain = is_domain_name()
```

---

dc = is_domain_dc()

### is_server — проверка сервера БД

Проверяет, выполняется ли текущий код на сервере БД. Сравнение get_workstation_mode() == SERVER.

**Параметры**: нет.

**Возвращает**: `Boolean` — TRUE, если код выполняется на сервере, иначе FALSE.

```EME-L
' Проверить, выполняется ли код на сервере БД (TRUE — сервер) '
result = is_server()
```

---

### is_http_server — проверка HTTP-сервера

Проверяет, выполняется ли текущий код на HTTP-сервере. Если указан параметр transactionPort = TRUE, дополнительно проверяет, что запрос пришёл на транзакционный порт HTTP-сервера.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| transactionPort | Boolean | Нет | TRUE — проверить транзакционный порт (по умолчанию FALSE) |

**Возвращает**: `Boolean` — TRUE, если условие выполнено, иначе FALSE.

```EME-L
' Проверить, выполняется ли код на HTTP-сервере (TRUE — HTTP-сервер) '
result = is_http_server()

' Проверить, выполняется ли код на транзакционном порту HTTP-сервера (TRUE — транзакционный порт) '
result = is_http_server(TRUE)
```

---

### is_brain_computer — мозговой компьютер

Проверяет, выполняется ли текущий код на мозговом компьютере (brain computer). Проверка флага командной строки CLF_BC.

**Параметры**: нет.

**Возвращает**: `Boolean` — TRUE, если код выполняется на мозговом компьютере, иначе FALSE.

```EME-L
' Проверить, выполняется ли код на мозговом компьютере (TRUE — мозговой компьютер) '
result = is_brain_computer()
```

---

### is_workstation_mode — режим рабочей станции

Возвращает режим рабочей станции в виде строки.

**Параметры**: нет.

**Возвращает**: `String` — режим рабочей станции: `"MONO"` — монопольный режим, `"WORKSTATION"` — рабочая станция, `"SERVER"` — сервер, `"CANAL_BRAIN"` — мозговой компьютер канала, `"TERMINAL"` — терминал, `"TERM_BRAIN"` — мозговой компьютер терминала.

```EME-L
mode = is_workstation_mode()
```

---

### is_workstation_type — тип рабочей станции

Возвращает тип рабочей станции по наличию данных виртуальной машины.

**Параметры**: нет.

**Возвращает**: `String` — тип рабочей станции: `"UNDEFINED_WS"` — не определено, `"FULLVM_WS"` — полная виртуальная машина, `"NETVM_WS"` — сетевая виртуальная машина.

```EME-L
wtype = is_workstation_type()
```

---

### is_system_family — семейство ОС

Возвращает семейство текущей операционной системы.

**Параметры**: нет.

**Возвращает**: `String` — `"LINUX"` для Linux, `"WINDOWS"` для Windows.

```EME-L
' Проверить семейство ОС (TRUE — выполнение на Linux) '
result = (is_system_family() = "LINUX")
```

### is_get_tick_count — счётчик миллисекунд

Возвращает количество миллисекунд, прошедших с момента запуска системы. Вызов GetTickCount.

### is_keycode — нажатые клавиши

Возвращает комбинацию нажатых в данный момент клавиш в виде текстовой строки. Вызов GetKeyState.

**Параметры**: нет.

**Возвращает**: `String` — строка с комбинацией клавиш (например `"Ctrl+Alt+Shift+Z"`).

```EME-L
keys = is_keycode()
```

---

### is_key_press — проверка нажатия клавиши

Проверяет, нажата ли в данный момент указанная виртуальная клавиша. Вызов GetAsyncKeyState.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| vkey | Integer | Да | Код виртуальной клавиши (например `0x10` — Shift, `0x11` — Ctrl) |

**Возвращает**: `Boolean` — TRUE, если клавиша нажата, иначе FALSE.

```EME-L
' Проверить, нажата ли клавиша A (TRUE — нажата) '
result = is_key_press(0x41)
```
