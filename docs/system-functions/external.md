# Внешние программы, версии и буфер обмена

### is_shell_execute — запуск внешней программы

Запускает внешнее приложение, открывает документ или протокол через ShellExecute/ShellExecuteEx. Если указан таймаут, дожидается завершения процесса и возвращает его exit code. В режиме службы (NT Service) запуск заблокирован.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Действие (строка): "open", "explore", "print" и т.д. |
| args[1] | String | Да | Рабочий каталог (строка) |
| args[2] | String | Да | Имя приложения, документа, файла или протокола (строка) |
| args[3] | String | Нет | Параметры командной строки (строка) |
| args[4] | Integer | Нет | Таймаут ожидания завершения в миллисекундах (целое число). 0 — ждать бесконечно. При отсутствии параметра возвращает результат ShellExecute |

**Возвращает**: без args[4] — `Integer` — результат ShellExecute (> 32 при успехе). С args[4] — `Integer` — exit code завершённого процесса. Отрицательные коды при ошибках: -1 — ошибка ShellExecuteEx, -2 — ошибка ожидания, -3 — таймаут, -4 — некорректное завершение.

```EME-L
is_shell_execute("open", "C:\\", "notepad.exe")
is_shell_execute("open", "", "document.pdf", "", 10000)
```

### is_shell_execute_link — запуск по ярлыку

Запускает программу по указанному ярлыку (.lnk) через ShellExecute. Не дожидается завершения запущенного процесса. В режиме службы (NT Service) запуск заблокирован.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Полный путь к файлу ярлыка (например, "d:\\EME\\mono.lnk") |

**Возвращает**: `Boolean` — TRUE при успехе.

```EME-L
is_shell_execute_link("d:\\EME\\app.lnk")
```

## Версии

### is_check_version — проверка версии ядра

проверяет, что текущая версия ядра и метапроекта больше или равна указанной.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Нет | Ядро — major |
| args[1] | Integer | Нет | Ядро — minor |
| args[2] | Integer | Нет | Метапроект — major |
| args[3] | Integer | Нет | Метапроект — minor |

**Возвращает**: `Boolean` — TRUE если текущая версия >= указанной.

```EME-L
' Проверить версию ядра (TRUE — версия ядра >= 30.5) '
result = is_check_version(30, 5)

' Проверить версию ядра и метапроекта (TRUE — ядро >= 30.5 и метапроект >= 20.0) '
result = is_check_version(30, 5, 20, 0)
```

---

### is_type_version — тип версии (лицензия)

возвращает тип лицензии программы.

**Параметры**: нет.

**Возвращает**: `Integer` — 0 (без ключа), 1 (полная версия), 2 (стандартная).

```EME-L
vtype = is_type_version()
```

---

### is_license_param — параметр лицензии

возвращает значение параметра лицензии по имени.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя параметра лицензии |

**Возвращает**: `String` — значение параметра или `Empty` если не найден.

```EME-L
val = is_license_param("MaxUsers")
```

## Буфер обмена

### is_get_clipboard_data_type — тип данных буфера

возвращает тип данных в буфере обмена.

**Параметры**: нет.

**Возвращает**: `Integer` — -1 (пусто/неизвестно), 2 (CF_BITMAP), 15 (CF_HDROP).

```EME-L
ctype = is_get_clipboard_data_type()
```

---

### is_get_clipboard_as_bitmap — снимок экрана из буфера

получает содержимое буфера обмена как bitmap. Поддерживается только снимок экрана.

**Параметры**: нет.

**Возвращает**: `Empty` если буфер пуст или данные неподдерживаемы. Образ BMP-файла если в буфере снимок экрана.

```EME-L
bmp = is_get_clipboard_as_bitmap()
```

---

### is_put_clipboard_data_as_text — текст в буфер обмена

записывает текстовую строку в буфер обмена в формате Unicode.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Строка для записи |

**Возвращает**: `Boolean` — TRUE при успехе.

```EME-L
is_put_clipboard_data_as_text("Скопированный текст")
```

---

### is_put_clipboard_data_as_filelist — файл в буфер обмена

записывает ссылку на файл в буфер обмена.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Путь к файлу |
| args[1] | Boolean | Нет | TRUE — копировать (по умолчанию), FALSE — вырезать |

**Возвращает**: `Boolean` — TRUE при успехе.

```EME-L
is_put_clipboard_data_as_filelist("C:\\file.txt", TRUE)
```
