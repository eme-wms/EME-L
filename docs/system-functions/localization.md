# Язык и локализация

> Synonyms for search: язык, локализация, is_set_language, is_get_language, is_set_translate_regim, is_get_translate_regim, ML, tr, is_transliterate, is_decimal_separator, is_locale_info, перевод, транслитерация, локаль

### is_set_language — установка языка

устанавливает код языка локализации для текущего потока. Если код не указан, устанавливается язык по умолчанию для интерфейса.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Код языка (например, "RUS", "ENG"). При отсутствии — язык по умолчанию для интерфейса |

**Возвращает**: `Integer` — 1 при успешной установке, 0 если функциональность недоступна.

```EME-L
is_set_language("EN")

' Язык по умолчанию '
is_set_language()
```

---

### is_get_language — текущий язык

возвращает код языка локализации, установленный для текущего потока.

**Параметры**: нет.

**Возвращает**: `String` — код языка. Если функциональность недоступна, возвращает "0".

```EME-L
lang = is_get_language()
```

---

### is_set_translate_regim — режим перевода

устанавливает режим перевода для открытия отчётов с диалогами.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Режим перевода: 0 — отключить перевод, 1 — только считывать тексты, 2 — переводить тексты |

**Возвращает**: `Integer` — 0.

```EME-L
is_set_translate_regim(1)
```

---

### is_get_translate_regim — текущий режим перевода

возвращает текущий режим перевода для открытия отчётов с диалогами.

**Параметры**: нет.

**Возвращает**: `Integer` — режим перевода: 0 — отключен, 1 — считывать тексты, 2 — переводить тексты.

```EME-L
regim = is_get_translate_regim()
```

---

### ML — заглушка локализации

заглушка для функции локализации. Всегда возвращает пустое значение.

**Параметры**: нет.

**Возвращает**: пустое значение.

---

### tr — перевод строки

возвращает перевод строки на текущий язык из записи Локализация. Если перевод отсутствует, возвращает исходную строку (или её транслитерацию). Перевод зависит от контекста.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка для перевода |
| args[1] | String | Нет | Контекст перевода. Для EME-L-кода формируется автоматически. Пример: "EMEL:ProcessTemplates" |

**Возвращает**: `String` — переведённая строка. Если перевод не найден — исходная строка.

```EME-L
translated = tr("Hello")
translated = tr("Hello", "context")
```

---

### is_transliterate — транслитерация

транслитерирует строку: заменяет русские буквы и символы кодировки Windows-1251 на подходящие по произношению английские буквы и сочетания. Длина результирующей строки может увеличиться.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка |

**Возвращает**: `String` — транслитерированная строка.

```EME-L
' Privet '
result = is_transliterate("Привет")
```

---

### is_decimal_separator — десятичный разделитель

возвращает текущий десятичный разделитель из локальных настроек системы. Вызов GetLocaleInfo с LOCALE_SDECIMAL.

**Параметры**: нет.

**Возвращает**: `String` — десятичный разделитель (обычно "." или ",").

```EME-L
sep = is_decimal_separator()
```

---

### is_locale_info — информация о локали

возвращает локальную установку системы. Аналог WinAPI GetLocaleInfo.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Идентификатор локали: "LOCALE_SYSTEM_DEFAULT" (системная) или "LOCALE_USER_DEFAULT" (пользователя) |
| args[1] | String | Да | Тип запрашиваемой установки: "LOCALE_SDECIMAL" (десятичный разделитель) или "LOCALE_SLIST" (разделитель списка). Другие значения не поддерживаются. |

**Возвращает**: `String` — значение локальной установки.

```EME-L
sep = is_locale_info("LOCALE_USER_DEFAULT", "LOCALE_SDECIMAL")
```
