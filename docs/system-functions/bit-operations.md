# Битовые операции

в языке EME-L битовые операции реализованы группой is-функций `Bit`. в системе EME.WMS они используются для манипуляции флагами, проверки прав доступа, формирования масок и низкоуровневой обработки данных. Все функции поддерживают 32-битные целые числа; некоторые также принимают массивы байтов.

---

### is_bit — проверка установленного бита

в языке EME-L функция `is_bit` проверяет состояние указанного бита в битовом поле. Поддерживает целые числа (32-битный `long`, `VT_I4`) и массивы байтов (`VT_ARRAY|VT_UI1`). Для целых чисел проверяется бит в 32-битном значении; для массивов байтов — бит в соответствующем байте массива.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / Array | Да | Битовое поле для проверки. Целое число (4 байта, VT_I4) или массив байтов (VT_ARRAY\|VT_UI1) |
| args[1] | Integer | Да | Номер проверяемого бита, начиная с младшего бита 0 |

**Возвращает**: `Boolean` — `TRUE`, если указанный бит установлен (равен 1); `FALSE`, если бит сброшен (равен 0).

```EME-L
' Проверка бита 3 в флагах '
Result = is_bit(Flags, 3)

' Проверка бита 5 в массиве байтов '
Result = is_bit(ByteArray, 5)
```

---

### is_clear_bit — сброс бита

в системе EME.WMS функция `is_clear_bit` сбрасывает указанный бит в битовом поле в значение 0. Для целых чисел возвращает модифицированное число; для массивов байтов — модифицированный массив. Тип возвращаемого значения соответствует типу входного параметра.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / Array | Да | Битовое поле для модификации. Целое число (VT_I4) или массив байтов (VT_ARRAY\|VT_UI1) |
| args[1] | Integer | Да | Номер сбрасываемого бита, начиная с 0 |

**Возвращает**: `Integer` или `Array` — модифицированное битовое поле со сброшенным указанным битом. Тип соответствует типу входного параметра.

```EME-L
' Снять бит 5, результат присваиваем обратно в ту же переменную '
Flags = is_clear_bit(Flags, 5)

' Снять бит 2 в массиве, результат присваиваем обратно '
ByteArray = is_clear_bit(ByteArray, 2)
```

**Возвращает**: `Integer` или `Array` — модифицированное битовое поле с установленным указанным битом. Тип соответствует типу входного параметра.


### is_reverse_bit — инверсия бита

в EME БД функция `is_reverse_bit` инвертирует значение указанного бита в битовом поле: 0 становится 1, 1 становится 0. Входное значение безусловно преобразуется в массив байтов перед операцией, поэтому при передаче целого числа возвращается массив байтов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / Array | Да | Битовое поле для модификации. Преобразуется в массив байтов (VT_ARRAY\|VT_UI1) |
| args[1] | Integer | Да | Номер инвертируемого бита, начиная с 0 |

**Возвращает**: `Array` — модифицированное битовое поле в виде массива байтов с инвертированным указанным битом.

```EME-L
' Инвертировать бит 3, результат присваиваем обратно '
Flags = is_reverse_bit(Flags, 3)
```

### is_and — побитовое И

в языке EME-L функция `is_and` выполняет побитовую операцию И (AND) над всеми переданными аргументами. Поддерживает произвольное количество параметров. Каждый параметр может быть целым числом, 8-байтовым числом (при условии значения в диапазоне 0..0xFFFFFFFF) или массивом байтов. Операнды приводятся к 4-байтовым целым числам перед выполнением операции.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[N] | Integer / Array | Нет | Операнды для побитового умножения. Целое число, 8-байтовое число в диапазоне 0..0xFFFFFFFF или массив байтов |

**Вызов без аргументов**: возвращает -1 (все 32 бита установлены в 1).

**Возвращает**: `Integer` — результат побитовой операции И всех переданных операндов.

```EME-L
' Побитовое И двух флагов '
Result = is_and(Flags1, Flags2)

' Побитовое И трех операндов '
Result = is_and(0xFF, 0x0F, 0xF0)

' Без аргументов - возвращает -1 '
Result = is_and()
```

### is_or — побитовое ИЛИ

в системе EME.WMS функция `is_or` выполняет побитовую операцию ИЛИ (OR) над всеми переданными аргументами. Поддерживает произвольное количество параметров. Каждый параметр может быть целым числом, 8-байтовым числом (при условии значения в диапазоне 0..0xFFFFFFFF) или массивом байтов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[N] | Integer / Array | Нет | Операнды для побитового сложения. Целое число, 8-байтовое число в диапазоне 0..0xFFFFFFFF или массив байтов |

**Вызов без аргументов**: возвращает 0 (все 32 бита сброшены в 0).

**Возвращает**: `Integer` — результат побитовой операции ИЛИ всех переданных операндов.

```EME-L
' Объединение двух флагов '
Result = is_or(Flags1, Flags2)

' Объединение нескольких масок '
Result = is_or(0x01, 0x02, 0x04)

' Без аргументов - возвращает 0 '
Result = is_or()
```

### is_xor — побитовое исключающее ИЛИ

в языке EME-L функция `is_xor` выполняет побитовую операцию исключающее ИЛИ (XOR) между двумя аргументами. Первый аргумент может быть целым числом, 8-байтовым числом (при условии значения в диапазоне 0..0xFFFFFFFF) или массивом байтов. Второй аргумент преобразуется в целое число.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / Array | Да | Первый операнд. Целое число, 8-байтовое число (0..0xFFFFFFFF) или массив байтов |
| args[1] | Integer | Да | Второй операнд (целое число) |

**Возвращает**: `Integer` — результат побитовой операции исключающее ИЛИ двух операндов.

```EME-L
' XOR двух флагов '
Result = is_xor(Flags1, Flags2)

' Переключение битов по маске '
Result = is_xor(Value, 0xFF)
```

### is_not — побитовое НЕ

в EME БД функция `is_not` выполняет побитовую операцию НЕ (NOT) — инвертирует все биты аргумента. Аргумент может быть целым числом, 8-байтовым числом (при условии значения в диапазоне 0..0xFFFFFFFF) или массивом байтов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / Array | Да | Операнд для побитовой инверсии. Целое число, 8-байтовое число (0..0xFFFFFFFF) или массив байтов |

**Возвращает**: `Integer` — результат побитовой инверсии всех битов операнда.

```EME-L
' Инвертировать все биты '
Result = is_not(Flags)

' Инвертировать 0x00FF - результат 0xFF00 '
Result = is_not(0x00FF)
```

### is_mask — формирование битовой маски

в языке EME-L функция `is_mask` формирует битовую маску, устанавливая в 1 биты с указанными номерами. Принимает произвольное количество аргументов — номеров битов, которые необходимо установить. Нумерация битов начинается с 0 (младший бит).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[N] | Integer | Нет | Номера битов для установки в маске, начиная с 0 |

**Вызов без аргументов**: вызывает ошибку исполнения (требуется минимум один аргумент).

**Возвращает**: `Integer` — сформированная битовая маска с установленными битами по указанным позициям.

```EME-L
' Маска с битами 1, 3 и 5: 0b101010 = 0x2A '
Mask = is_mask(1, 3, 5)

' Маска с одним битом 4: 0b10000 = 0x10 '
Mask = is_mask(4)
```

### is_shift_left — побитовый сдвиг влево

в системе EME.WMS функция `is_shift_left` выполняет побитовый сдвиг значения влево на заданное количество бит. Операнды преобразуются в беззнаковые 32-битные целые числа перед выполнением сдвига, что предотвращает размножение знакового бита.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Исходное значение для сдвига. Преобразуется в беззнаковое 32-битное целое число |
| args[1] | Integer | Да | Количество бит для сдвига влево |

**Возвращает**: `Integer` — результат побитового сдвига влево. При сдвиге на 32 и более бит результат зависит от архитектуры процессора.

```EME-L
' Сдвиг влево на 2 бита: 0b0001 - 0b0100 '
Result = is_shift_left(1, 2)

' Сдвиг байта влево '
Result = is_shift_left(0x01, 8)
```

---

### is_shift_right — побитовый сдвиг вправо

в языке EME-L функция `is_shift_right` выполняет побитовый сдвиг значения вправо на заданное количество бит. Операнды преобразуются в беззнаковые 32-битные целые числа перед выполнением сдвига, что предотвращает размножение знакового бита при сдвиге вправо.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Исходное значение для сдвига. Преобразуется в беззнаковое 32-битное целое число |
| args[1] | Integer | Да | Количество бит для сдвига вправо |

**Возвращает**: `Integer` — результат побитового сдвига вправо. При сдвиге на 32 и более бит результат зависит от архитектуры процессора.

```EME-L
' Сдвиг вправо на 2 бита: 0b0100 - 0b0001 '
Result = is_shift_right(4, 2)

' Логический сдвиг без знакового расширения '
Result = is_shift_right(0x80000000, 1)
```
