# Печать и PDF

> Synonyms for search: печать, PDF, is_find_printer, is_join_pdf, is_create_blank_pdf, is_getnoofpages_pdf, is_get_pdfimages, принтер, объединение PDF, извлечение изображений


### is_find_printer — поиск принтера по имени

выполняет поиск принтера в системе по части его имени. Возвращает полное имя найденного принтера или пустую строку, если принтер не найден.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Фрагмент имени принтера для поиска |

**Возвращает**: `String` — полное имя найденного принтера. Пустая строка если принтер не найден.

```EME-L
printer = is_find_printer("Zebra")
' printer = "ZDesigner ZT411-300dpi ZPL" (пример) '

' Проверить, что принтер HP не найден '
result = (is_find_printer("HP") = "")
```

### is_join_pdf — объединение PDF-файлов

объединяет несколько PDF-файлов в один выходной документ. в языке EME-L для объединения используется внутренняя реализация на базе Ghostscript. Входные файлы передаются в виде массива строк.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя выходного PDF-файла |
| args[1] | Array | Да | Массив строк с путями к входным PDF-файлам |

**Возвращает**: `Integer` — 1 при успехе, 0 при ошибке объединения, 9 если в массиве менее двух элементов.

```EME-L
Dim files = New CIArray()
files.Add("C:\Docs\page1.pdf")
files.Add("C:\Docs\page2.pdf")
files.Add("C:\Docs\page3.pdf")

result = is_join_pdf("C:\Docs\combined.pdf", files)
' result = 1 при успехе '
```

### is_create_blank_pdf — создание пустого PDF

создаёт пустой PDF-документ с указанным именем. При необходимости использует существующий файл в качестве образца для формирования структуры документа.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя создаваемого PDF-файла |
| args[1] | String | Нет | Имя файла-образца для копирования структуры документа. По умолчанию пустая строка |

**Возвращает**: `Integer` — 1 при успешном создании, 0 при ошибке.

```EME-L
result = is_create_blank_pdf("C:\Docs\blank.pdf")
' result = 1 при успехе '

' Создать PDF с файлом-образцом '
result = is_create_blank_pdf("C:\Docs\new.pdf", "C:\Docs\template.pdf")
```

### is_getnoofpages_pdf — количество страниц PDF

возвращает общее количество страниц в указанном PDF-документе.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя PDF-файла для анализа |

**Возвращает**: `Integer` — количество страниц в документе.

```EME-L
pages = is_getnoofpages_pdf("C:\Docs\report.pdf")
' pages = 12 (пример) '
```

### is_get_pdfimages — извлечение страниц PDF в изображения

извлекает страницы из PDF-файла и сохраняет их в виде изображений в формате JPEG в указанный контейнер данных. в системе EME.WMS поддерживаемые типы контейнеров: CIArchive, CICollection, CIArray и CIDataStorage. При использовании CIDataStorage сохраняется только одна страница.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя исходного PDF-файла |
| args[1] | Reference | Да | Контейнер для сохранения изображений: CIArchive (Store), CICollection (элементы по номеру страницы), CIArray (объекты CIDataStorage), CIDataStorage (одна страница) |
| args[2] | Integer | Нет | Номер начальной страницы. 0 — с первой (по умолчанию) |
| args[3] | Integer | Нет | Номер последней страницы. 0 — до последней (по умолчанию) |
| args[4] | Integer | Нет | Разрешение в DPI. По умолчанию 300 |
| args[5] | Integer | Нет | Коэффициент сглаживания: 1 — минимальное, 2 — среднее, 4 — максимальное (по умолчанию) |

**Возвращает**: `Integer` — количество извлечённых страниц при успехе. Отрицательное значение при ошибке Ghostscript.

```EME-L
' Извлечь все страницы в CIArchive '
Dim archive = New CIArchive()
count = is_get_pdfimages("C:\Docs\report.pdf", archive)
' count = количество извлечённых страниц '

' Извлечь страницы 1-3 с разрешением 150 DPI '
count = is_get_pdfimages("C:\Docs\report.pdf", archive, 0, 2, 150)
```
