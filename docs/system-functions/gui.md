# Управление органами диалогов (GUI)

> Synonyms for search: GUI, органы диалогов, is_read_only, is_selected, is_edit_data, is_sum, is_focus, is_modified, is_tree_line, is_tree_rec, is_recursion_ref, is_get_key_full_text, is_show_ctrl, фокус, дерево, орган управления


### is_read_only — проверка флага «только чтение»

GUI-функция. Проверяет, установлен ли для указанного органа управления флаг READ_ONLY (только для чтения). Если имя органа не указано, проверяется текущий орган управления в контексте редактирования.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Имя органа управления для проверки. Если не задано, используется текущий орган управления |

**Возвращает**: `Boolean` — `TRUE` если флаг READ_ONLY установлен, `FALSE` если не установлен.

```EME-L
' Проверить флаг READ_ONLY органа EditField1 (TRUE — только чтение) '
result = is_read_only("EditField1")

' Проверить флаг READ_ONLY текущего органа (TRUE — только чтение) '
result = is_read_only()
```

### is_selected — проверка выбора радиокнопки/флажка

GUI-функция. Проверяет, выбрана ли указанная радиокнопка или установлен ли указанный флажок в группе органов управления.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя группы радиокнопок или флажков |
| args[1] | Integer | Да | Номер радиокнопки или флажка для проверки |
| args[2] | Integer | Нет | Уровень иерархии диалога. 0 — текущий диалог (по умолчанию). Для веб-контекста допустимы только значения -1 и 0 |

**Возвращает**: `Boolean` — `TRUE` если радиокнопка выбрана или флажок установлен, `FALSE` если нет.

```EME-L
' Проверить выбор радиокнопки в группе (TRUE — выбрана вторая) '
result = is_selected("RadioGroup", 2)
```

### is_edit_data — содержимое органа управления

GUI-функция. Возвращает содержимое указанного органа управления или колонки таблицы с учётом его атрибутов и привязки к данным. Поддерживает чтение через реляционные пути и привязанные органы управления.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Имя органа управления или колонки таблицы. Если не задано, используется текущий орган управления |
| args[1] | Integer | Нет | Уровень иерархии объекта/диалога. 0 — текущий объект (по умолчанию) |
| args[2] | String | Нет | Реляционный путь для разыменования ссылочного поля. Если задан непустой путь, выполняется переход по ссылкам БД |

**Возвращает**: тип зависит от атрибута поля — строка, целое число, вещественное число, дата, время, `Boolean` или ссылка.

```EME-L
' Чтение значения текущего органа '
value = is_edit_data()

' Чтение значения колонки таблицы '
name = is_edit_data("NameColumn")

' Чтение с реляционным путём '
refName = is_edit_data("Field1", 0, "Name")
```

### is_sum — сумма по колонке/ячейке

GUI-функция. Вычисляет сумму значений по указанной колонке таблицы или по указанной ячейке отчёта с накоплением по всем строкам/элементам отчёта. в контексте таблицы суммирует значения колонки. в контексте отчёта суммирует данные по всем экземплярам ячейки вертикального банда.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя ячейки отчёта или колонки таблицы |
| args[1] | Boolean | Нет | Признак округления перед суммированием. `TRUE` — округлять в соответствии с точностью поля (по умолчанию). `FALSE` — суммировать без округления |

**Возвращает**: `Float` или `Integer` (в зависимости от атрибута поля) — сумму значений. Если данных нет, возвращает 0.

```EME-L
total = is_sum("AmountColumn")
' total = сумма по всем строкам колонки AmountColumn '

' Без округления '
total = is_sum("AmountColumn", FALSE)
```

### is_tree_line — номер строки узла дерева

GUI-функция. Возвращает номер строки (DB line) текущего или указанного узла дерева. Поддерживает навигацию по родительским узлам на заданное количество уровней вверх.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Софтовое имя органа управления «Дерево». Если не задано, используется дерево, встроенное в текущий диалог |
| args[1] | Integer | Нет | Уровень иерархии диалога с деревом. 0 — текущий диалог (по умолчанию) |
| args[2] | Integer | Нет | Смещение по иерархии узлов относительно текущего выбранного. Поддерживаются только отрицательные значения (родительские уровни). 0 — текущий узел (по умолчанию) |
| args[3] | Boolean | Нет | Игнорировать отсутствие родительского узла. `TRUE` — вернуть `NULL_REF` при отсутствии родителя. `FALSE` — вызвать ошибку (по умолчанию) |

**Возвращает**: `Integer` — номер строки элемента дерева. `NULL_REF` — если ничего не выбрано или при args[3]=`TRUE` отсутствует родительский узел.

```EME-L
' Строка текущего узла дерева '
line = is_tree_line()

' Строка родительского узла (на 1 уровень вверх) '
parentLine = is_tree_line(, , -1)

' Строка родителя с обработкой отсутствия '
parentLine = is_tree_line("MyTree", 0, -1, TRUE)
' parentLine = NULL_REF если родителя нет '
```

### is_recursion_ref — ссылка на родительский узел дерева

GUI-функция. Возвращает ссылку (DB line) на родительский узел текущего выбранного элемента в диалоге с управляющим органом «Дерево». в языке EME-L предназначена для использования при добавлении новой строки (NEW_LINE_MARK), когда требуется определить родителя для рекурсивной вставки.

**Параметры**: нет.

**Возвращает**: `Integer` — ссылка на родительский узел дерева. `NULL_REF` — если дерево отсутствует, ничего не выбрано, или у текущего узла нет родителя.

```EME-L
parentRef = is_recursion_ref()
' Получить ссылку на родительский узел дерева (TRUE — есть родитель для рекурсивной вставки) '
hasParent = (parentRef <> NULL_REF)
```

### is_get_key_full_text — текстовое представление горячей клавиши

GUI-функция. Преобразует числовой код горячей клавиши, полученный методом `CONTROL_KEYDOWN_NOTIFY_EX::GetKeyFullCode`, в текстовое представление (аналог метода GetKeyFullText).

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Числовой код горячей клавиши |

**Возвращает**: `String` — текстовое представление комбинации клавиш. Пустая строка — если код не распознан или функциональность недоступна.

```EME-L
text = is_get_key_full_text(keyCode)
' text = "Ctrl+Shift+F" (пример) '
```

### is_show_ctrl — видимость органа управления

GUI-функция. Изменяет видимость указанного органа управления в диалоге.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя органа управления в диалоге |
| args[1] | Boolean | Да | Признак видимости. `TRUE` — показать, `FALSE` — скрыть |

**Возвращает**: пустое значение.

```EME-L
' Скрыть орган управления '
is_show_ctrl("ExtraField", FALSE)

' Показать орган управления '
is_show_ctrl("ExtraField", TRUE)
```
