# Цвета

> Synonyms for search: цвета, is_RGB, is_RGBA, is_blendRGB, is_sys_color, is_legend_color, COLORREF, палитра, RGB, RGBA, системный цвет

### is_RGB — цвет из RGB

формирует цветовое значение типа COLORREF из трёх компонент RGB. Вызов макроса RGB.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Интенсивность красного канала (0-255) |
| args[1] | Integer | Да | Интенсивность зелёного канала (0-255) |
| args[2] | Integer | Да | Интенсивность синего канала (0-255) |

**Возвращает**: `Integer` — цветовое значение COLORREF.

```EME-L
color = is_RGB(255, 128, 0)
```

### is_RGBA — цвет из RGBA

формирует цветовое значение RGBA с альфа-каналом (прозрачностью). Поддерживает два режима вызова. Вызов qRgba.

**Режим 1** — отдельные каналы (4 аргумента):

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Интенсивность красного канала (0-255) |
| args[1] | Integer | Да | Интенсивность зелёного канала (0-255) |
| args[2] | Integer | Да | Интенсивность синего канала (0-255) |
| args[3] | Integer | Нет | Альфа-канал (0-255, по умолчанию 0x00 — полностью прозрачный) |

**Режим 2** — из готового RGB-цвета (2 аргумента):

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Готовый RGB-цвет (COLORREF) |
| args[1] | Integer | Нет | Альфа-канал (0-255, по умолчанию 0x00 — полностью прозрачный) |

**Возвращает**: `Integer` — цветовое значение RGBA.

```EME-L
' Полупрозрачный красный из каналов '
color = is_RGBA(255, 0, 0, 128)
' Из готового RGB с прозрачностью '
color = is_RGBA(rgb_color, 200)
```

### is_blendRGB — смешивание цветов

вычисляет промежуточный цвет между двумя заданными цветами по линейной интерполяции с указанным коэффициентом.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Первый цвет (COLORREF) |
| args[1] | Integer | Да | Второй цвет (COLORREF) |
| args[2] | Real | Нет | Коэффициент смешивания (0.0-1.0, по умолчанию 0.5). 0.0 — полностью второй цвет, 1.0 — полностью первый цвет. |

**Возвращает**: `Integer` — смешанный цвет COLORREF.

```EME-L
' 70% первый, 30% второй '
mixed = is_blendRGB(color1, color2, 0.7)
' 50/50 '
mixed = is_blendRGB(color1, color2)
' Полностью второй цвет '
mixed = is_blendRGB(color1, color2, 0.0)
' Полностью первый цвет '
mixed = is_blendRGB(color1, color2, 1.0)
```

### is_sys_color — системный цвет Windows

возвращает текущий системный цвет указанного элемента Windows. Вызов GetSysColor.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Название отображаемого элемента Windows. Поддерживаемые значения: "3DDKSHADOW", "3DFACE", "BTNFACE", "3DHILIGHT", "3DLIGHT", "3DSHADOW", "ACTIVEBORDER", "ACTIVECAPTION", "APPWORKSPACE", "BACKGROUND", "BTNTEXT", "CAPTIONTEXT", "GRAYTEXT", "HIGHLIGHT", "HIGHLIGHTTEXT", "INACTIVEBORDER", "INACTIVECAPTION", "INACTIVECAPTIONTEXT", "INFOBK", "INFOTEXT", "MENU", "MENUTEXT", "SCROLLBAR", "WINDOW", "WINDOWFRAME", "WINDOWTEXT", "GRADIENTACTIVECAPTION", "GRADIENTINACTIVECAPTION", "HOTLIGHT". |

**Возвращает**: `Integer` — текущий системный цвет элемента (COLORREF).

```EME-L
color = is_sys_color("WINDOW")
color = is_sys_color("HIGHLIGHTTEXT")
color = is_sys_color("GRADIENTACTIVECAPTION")
```

### is_legend_color — цвет из палитры легенды

возвращает цвет из встроенной палитры легенды по индексу (0-21).

| Параметр | Тип | Обязательный | Описание |

| args[0] | Integer | Да | Номер цвета в палитре (0-21). См. таблицу палитры ниже. |

**Возвращает**: `Integer` — цветовое значение из палитры (COLORREF). При выходе индекса за диапазон возвращает серый цвет RGB(192,192,192).

| 10 | синий | Синий |
| 11 | васильковый синий | Васильковый синий |
| 12 | циановый | Циановый |
