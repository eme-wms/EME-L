# Утилиты и вспомогательные функции

> Synonyms for search: утилиты, is_exists_ext_par, is_extract_picture_from_DB, is_dbref, is_dbref_record, is_dbref_field, is_dbref_line, is_price_in_full, is_bcp_solution_active, is_bcp_solution_sos_turn_on, is_codec, is_keyWGH, is_auto_change_statuses, is_auto_create_moving_in_pickbyline, is_planned_creations_orders, is_future, SHA256, dbref, кодирование, автоматизация

в языке EME-L вспомогательные функции группы WMS обеспечивают работу со строками и датами, формирование ссылок dbref, кодирование и хеширование данных, управление планировщиком и таймерами, а также доступ к системным параметрам BCP Solution. в системе EME.WMS эти функции используются для служебных операций, автоматизации процессов и интеграции с внешними системами.

## Работа со строками и датами

### is_column — значение column из TS_PARAM

возвращает значение `column` из структуры `TS_PARAM`.

**Параметры**: нет.

**Возвращает**: `Integer` — текущее значение `column`.

```EME-L
' Получить текущее значение column '
col = is_column()
' col = номер текущей колонки '
```

---

### is_convert_string_to_date — конвертация строки в дату

конвертирует строку формата `YYYYMMDD` в значение типа дата.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Строка даты в формате `YYYYMMDD` |

**Возвращает**: `Date` — дата, полученная из строки.

```EME-L
' Конвертировать строку в дату '
d = is_convert_string_to_date("20250115")
' d = 15.01.2025 — дата '
```

---

### is_insert_delimeters_in_string — вставка разделителей в строку

вставляет разделители в строку на указанных позициях.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Исходная строка |
| args[1] | String | Да | Разделитель для вставки |
| args[2] | Integer | Да | Первая позиция вставки разделителя |
| args[3+] | Integer | Нет | Последующие позиции в порядке возрастания |

**Возвращает**: `String` — строка с вставленными разделителями.

```EME-L
' Вставить дефисы в строку даты '
result = is_insert_delimeters_in_string("20250115", "-", 4, 6)
' result = "2025-01-15" '

' Форматировать телефонный номер '
phone = is_insert_delimeters_in_string("78001234567", " ", 1, 4, 7, 9, 11)
' phone = "7 800 123 45 67" '
```

### is_dbref — создание и проверка ссылки dbref

создаёт ссылку `dbref` или проверяет строку на соответствие формату `dbref`.

**При вызове с 3 аргументами** — создаёт ссылку:

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer / String | Да | Номер записи или имя записи |
| args[1] | Integer / String | Да | Номер поля или имя поля |
| args[2] | Integer | Да | Номер строки |

**При вызове с 1 аргументом** — проверяет ссылку:

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Строка для проверки |

**Возвращает**: при 3 аргументах — `String`, строка ссылки `dbref`. При 1 аргументе — `Boolean`: `TRUE`, если строка является ссылкой `dbref`, `FALSE` — в противном случае.

```EME-L
' Создать ссылку dbref '
ref = is_dbref("DOC_RECORD", "DocField", 1)
' ref = "dbref://DOC_RECORD/DocField/1" '

' Проверить строку на соответствие формату dbref '
isValid = is_dbref("dbref://DOC_RECORD/Field/1")
' isValid = TRUE '
```

## Планировщик и таймеры

### is_save_ware_timer_param — сохранение параметров таймера

сохраняет параметры таймера события «Проверка стоковых статусов».

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Периодичность: `0` — ежедневно; `1` — еженедельно; `2` — ежемесячно; `3` — выключен |
| args[1] | Time | Да | Время запуска |
| args[2] | Integer | Да | День недели для еженедельного запуска: `1` — понедельник, …, `7` — воскресенье |
| args[3] | Integer | Да | Число месяца для ежемесячного запуска, диапазон `1–31` |

**Возвращает**: пустое значение.

```EME-L
' Настроить ежедневный запуск в 02:00 '
is_save_ware_timer_param(0, Time(2, 0, 0), 1, 1)

' Настроить еженедельный запуск по понедельникам в 03:00 '
is_save_ware_timer_param(1, Time(3, 0, 0), 1, 1)

' Выключить таймер '
is_save_ware_timer_param(3, Time(0, 0, 0), 1, 1)
```

### is_price_in_full — сумма прописью

возвращает сумму прописью в рублях и копейках.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Float | Да | Сумма для преобразования |
| args[1] | Integer | Нет | Формат дробной части: `1` — прописью, `0` или не задан — цифрами |

**Возвращает**: `String` — сумма прописью.

```EME-L
' Получить сумму прописью '
text = is_price_in_full(1234.56)
' text = "Одна тысяча двести тридцать четыре рубля 56 копеек" '

' С дробной частью прописью '
text = is_price_in_full(1234.56, 1)
' text = "Одна тысяча двести тридцать четыре рубля пятьдесят шесть копеек" '
```

## Кодирование и безопасность

### is_codec — XOR-кодирование строки

выполняет XOR-кодирование или декодирование строки.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Входная строка для кодирования или декодирования |

**Возвращает**: `String` — последние 3 символа результата XOR-кодирования. При входной строке длиной 4 символа к строке добавляются 7 случайных цифр перед кодированием.

```EME-L
' Закодировать строку '
encoded = is_codec("test")
' encoded = результат XOR-кодирования '
```

### is_SHA256_as_base64_string — хеш SHA256 в Base64

вычисляет хеш-функцию SHA256 для переданных данных и возвращает результат в кодировке base64.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Reference | Да | Объект `CDataStorage` с данными для хеширования |

**Возвращает**: `String` — значение хеш-функции SHA256 в виде строки base64.

```EME-L
' Вычислить SHA256 для данных '
storage = Object("CDataStorage")
' ... заполнение storage данными ... '
hash = is_SHA256_as_base64_string(storage)
' hash = "dGVzdEhhc2hWYWx1ZTEyMzQ1Njc4OQ==" '
```

## Тестирование и время

### is_future — управление внутренним временем

управляет внутренним временем системы при тестировании в монопольном режиме.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Код команды: `0` — установить начальную дату тестирования; `1` — сдвинуть время вперёд на указанный интервал |
| args[1] | Date / Integer | Нет | Параметр команды: при команде `0` — начальная дата; при команде `1` — интервал в секундах |

**Возвращает**: пустое значение.

```EME-L
' Установить начальную дату тестирования '
is_future(0, Date(2025, 1, 1))

' Сдвинуть время вперёд на 3600 секунд (1 час) '
is_future(1, 3600)
```
