## Печать, экспорт и GUI

### is_enter_value_dialog — диалог ввода значения

GUI-функция. Выводит модальный диалог для ввода значения (целое число, вещественное число, строка или дата). Тип вводимых данных определяется типом стартового значения, переданного в параметре arg1.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Текст заголовка диалогового окна |
| args[1] | Variant | Да | Стартовое значение, определяющее тип вводимых данных: VT_I4 — целое число, VT_R8 — вещественное число, VT_BSTR — строка, VT_DATE — дата |
| args[2] | Integer | Нет | Минимальное значение для числового ввода |
| args[3] | Integer | Нет | Максимальное значение для числового ввода |
| args[4] | Variant | Нет | Значение, возвращаемое при нажатии кнопки Отмена |

**Возвращает**: тип соответствует типу args[1] — введённое пользователем значение. При отмене возвращает args[4], если задан, иначе пустое значение.

```EME-L
' Запросить целое число через диалог '
result = is_enter_value_dialog("Введите количество", 10)
' result = введённое пользователем целое число '

' Запросить строку с заголовком '
name = is_enter_value_dialog("Введите имя", "")
' name = введённая строка, или пустая строка при отмене '

' Запросить дату '
d = is_enter_value_dialog("Выберите дату", #01.06.2025#)
' d = введённая дата, или NULL_DATE при отмене '

' Числовой ввод с ограничениями и значением при отмене '
count = is_enter_value_dialog("Количество", 1, 1, 100, 0)
' count = введённое число от 1 до 100, или 0 при отмене '
```

### is_run_free_browser — запуск непривязанного браузера

GUI-функция. Запускает непривязанный браузер для выбора строки из набора данных. Открывает модальное окно браузера с указанными колонками и значениями.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Наименование браузера |
| args[1] | Array | Да | Массив наименований колонок |
| args[2] | Array | Да | Массив массивов значений |
| args[3] | Array | Нет | Массив для номеров выделенных строк, выходной параметр. Если задан, браузер поддерживает множественное выделение |

**Возвращает**: `Integer` — номер выбранной строки.

```EME-L
' Подготовить колонки и данные для браузера '
Dim cols = New CIArray()
cols.Add("Код")
cols.Add("Наименование")

Dim rows = New CIArray()
Dim row1 = New CIArray()
row1.Add("001")
row1.Add("Товар А")
rows.Add(row1)

Dim row2 = New CIArray()
row2.Add("002")
row2.Add("Товар Б")
rows.Add(row2)

' Запустить браузер для выбора одной строки '
selected = is_run_free_browser("Выбор товара", cols, rows)
' selected = номер выбранной строки (0 или 1) '

' Запустить браузер с множественным выделением '
Dim selectedLines = New CIArray()
selected = is_run_free_browser("Выбор товаров", cols, rows, selectedLines)
' selectedLines = массив номеров выделенных строк '
```

### is_run_free_browser_by_str — запуск браузера из строк

GUI-функция. Запускает непривязанный браузер для выбора строки из набора данных, переданного в виде строк. Открывает модальное окно браузера с колонками и значениями, разобранными из строк.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Наименование браузера |
| args[1] | String | Да | Наименования колонок, разделённые запятой |
| args[2] | String | Да | Значения, разделённые запятой в рамках колонки и точкой с запятой в рамках строки |
| args[3] | Array | Нет | Массив для номеров выделенных строк, выходной параметр. Если задан, браузер поддерживает множественное выделение |

**Возвращает**: `Integer` — номер выбранной строки.

```EME-L
' Запустить браузер из строковых данных '
selected = is_run_free_browser_by_str(
    "Выбор клиента",
    "Код,Наименование,Город",
    "001,ООО Альфа,Москва;002,ООО Бета,СПб"
)
' selected = номер выбранной строки (0 или 1) '
```

### is_print_docs — печать документов из запроса

Печатает документы, хранящиеся в базе данных, по данным запроса. Для каждой строки запроса извлекает документ из указанного поля и отправляет на печать.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Reference | Да | Объект запроса |
| args[1] | String or Integer | Да | Имя или номер поля запроса, содержащего ссылку на запись документов |
| args[2] | String, Integer or empty | Нет | Имя или номер поля записи с печатаемым документом. Если пустое, ищет первое поле переменной длины с неопределённым атрибутом |
| args[3] | Boolean or Integer | Нет | Режим двусторонней печати: 1 или TRUE — двусторонняя, 0 или FALSE — односторонняя |
| args[4] | Integer | Нет | Количество копий. По умолчанию 1 |
| args[5] | Array | Нет | Массив параметров для функции OnEditDocument |

**Возвращает**: пустое значение.

```EME-L
' Печать документов из запроса '
Dim q = Object("dsDB", "DOC_RECORD")
q.SetFilter("Status", "=", 1)
q.Read()

' Напечатать документы по ссылкам из поля DocRef '
is_print_docs(q, "DocRef")

' Печать с двусторонней печатью и 2 копиями '
is_print_docs(q, "DocRef", "", TRUE, 2)
```

### is_print_report_indirect — печать отчёта напрямую

Выполняет печать отчёта напрямую по указанным параметрам, без открытия диалога предварительного просмотра.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя отчёта |
| args[1] | Integer | Да | Номер строки записи |
| args[2] | Integer | Да | Количество копий отчёта |
| args[3] | Reference | Нет | Дополнительный параметр Lparam |

**Возвращает**: пустое значение.

```EME-L
' Напечатать отчёт Invoice по текущей строке '
is_print_report_indirect("Invoice", currentLine, 1)

' Напечатать 3 копии отчёта с дополнительным параметром '
is_print_report_indirect("Invoice", currentLine, 3, extraParams)
```

### is_export_pdf_docs — экспорт документов в PDF

Экспортирует документы из базы данных в файл PDF по данным запроса. Для каждой строки запроса извлекает документ из указанного поля и добавляет в выходной PDF-файл.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Путь к выходному файлу PDF |
| args[1] | Reference | Да | Объект запроса |
| args[2] | String or Integer | Да | Имя или номер поля запроса со ссылкой на документы |
| args[3] | String, Integer or empty | Нет | Имя или номер поля записи с документом. Если пустое, ищет первое поле переменной длины с неопределённым атрибутом |
| args[4] | Array | Да | Массив для записи ошибок экспорта |

**Возвращает**: `Integer` — 1 если экспорт выполнен успешно, 0 при ошибке.

```EME-L
' Экспорт документов из запроса в PDF '
Dim q = Object("dsDB", "DOC_RECORD")
q.SetFilter("Status", "=", 1)
q.Read()

Dim errors = New CIArray()
result = is_export_pdf_docs("C:\\Docs\\export.pdf", q, "DocRef", "", errors)
' result = 1 при успехе, 0 при ошибке '
' errors = массив сообщений об ошибках '
```

### is_save_image_from_field_to_pdf — сохранение изображения в PDF

Сохраняет изображение из указанного поля базы данных в файл PDF.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String or Integer | Да | Номер или имя записи |
| args[1] | String or Integer | Да | Номер или имя поля |
| args[2] | Integer | Да | Номер строки |
| args[3] | String | Да | Путь к выходному файлу PDF |
| args[4] | Array | Да | Массив для записи ошибок |

**Возвращает**: `Integer` — 1 если сохранение выполнено успешно, 0 при ошибке.

```EME-L
' Сохранить изображение из поля в PDF '
Dim errors = New CIArray()
result = is_save_image_from_field_to_pdf(
    "IMAGES_RECORD",
    "ImageData",
    1,
    "C:\\Docs\\image.pdf",
    errors
)
' result = 1 при успехе, 0 при ошибке '
```

### is_save_image_from_field_to_pdf2 — сохранение массива изображений в PDF

Сохраняет изображения из массива полей базы данных в файл PDF. Позволяет объединить несколько изображений в один PDF-документ.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Array | Да | Массив номеров или имён записей |
| args[1] | Array | Да | Массив номеров или имён полей |
| args[2] | Array | Да | Массив номеров строк |
| args[3] | String | Да | Путь к выходному файлу PDF |
| args[4] | Array | Да | Массив для записи ошибок |

**Возвращает**: `Integer` — 1 если сохранение выполнено успешно, 0 при ошибке.

```EME-L
' Сохранить несколько изображений в один PDF '
Dim recs = New CIArray()
recs.Add("IMAGES_RECORD")
recs.Add("IMAGES_RECORD")

Dim fields = New CIArray()
fields.Add("ImageData1")
fields.Add("ImageData2")

Dim lines = New CIArray()
lines.Add(1)
lines.Add(2)

Dim errors = New CIArray()
result = is_save_image_from_field_to_pdf2(
    recs, fields, lines,
    "C:\\Docs\\combined.pdf",
    errors
)
' result = 1 при успехе, 0 при ошибке '
```

### is_extract_picture_from_DB — извлечение изображения из БД в файл

Сохраняет изображение из поля базы данных в файл на диск. Автоматически определяет формат изображения и добавляет корректное расширение к имени файла.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя файла для сохранения |
| args[1] | String or Integer | Да | Номер или имя записи |
| args[2] | String or Integer | Да | Номер или имя поля |
| args[3] | Integer | Да | Номер строки |

**Возвращает**: `String` — имя сохранённого файла с корректным расширением. При ошибке возвращает пустую строку.

```EME-L
' Извлечь изображение из базы данных в файл '
fileName = is_extract_picture_from_DB(
    "C:\\Temp\\image",
    "IMAGES_RECORD",
    "ImageData",
    1
)
' fileName = "C:\\Temp\\image.png" (расширение определено автоматически) '

' Проверить результат извлечения '
If (fileName <> "")
    ' Изображение успешно сохранено '
    is_message("Готово", "Файл сохранён: " + fileName)
End If
```
