# Филиалы и разделы

> Synonyms for search: филиалы, разделы, is_center, is_root_center, is_node_center, is_filial, is_system_filial, is_parent_filial_ref, is_parent_filial_code, is_filial_no, is_filial_ref, is_division_no, is_division_ref, is_change_division_no, центр, филиал, раздел БД

### is_center — центральный филиал

проверяет, является ли текущий филиал центром. Вызов DBS_IsCenterFilial.

**Параметры**: нет.

**Возвращает**: `Boolean` — TRUE, если текущий филиал — центр, иначе FALSE.

```EME-L
' Проверить, является ли текущий филиал центральным (TRUE — центральный филиал) '
result = is_center()
```

---

проверяет, является ли текущий филиал главным (корневым) центром в многоуровневой иерархии филиалов. Вызов IsRootCenter.

**Параметры**: нет.

**Возвращает**: `Boolean` — TRUE, если текущий филиал — главный центр, иначе FALSE.

```EME-L
' Проверить, является ли текущий филиал главным (корневым) центром (TRUE — главный центр) '
result = is_root_center()
```

### is_change_division_no — изменение номера раздела

изменяет номер текущего раздела базы данных. Можно вызывать только в монопольном режиме или на сервере. При ошибке вызывает исключение.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | Integer | Да | Новый номер раздела БД |

**Возвращает**: пустое значение.

```EME-L
is_change_division_no(5)
```
