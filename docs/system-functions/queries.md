# Работа с запросами — синонимы: is_message_report, is_stop_execute_query, is_query, is_query_data, is_query_result, is_query_count, is_sql_count, is_trace_query, is_expimp_query_save_line, is_expimp_query_compare_ref_fld, is_expimp_query_get_refs_keys, is_qtext, is_record_description

Функции для работы с запросами в системе EME.WMS: получение данных из запросов,
управление выполнением, тестирование и вспомогательные операции.

### is_message_report — сообщение в диалог прогресс-бара

Выводит сообщение в диалог прогресс-бара SQL-запроса с указанным текстом и режимом обработки.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Текст сообщения. По умолчанию "ОШИБКА!" |
| args[1] | Integer | Нет | Режим: 1=продолжить, 2=остановить, 3=catastroflog, 4=всегда продолжать, 5=завершить запрос. По умолчанию 1 |
| args[2] | Integer | Нет | Флаг catastroflog (любое значение добавляет запись в лог) |

**Возвращает**: `Empty` — пустое значение.

```EME-L
' Вывести сообщение с возможностью продолжить '
is_message_report("Внимание: дубликат найден", 1)

' Записать ошибку в catastroflog и остановить '
is_message_report("Критическая ошибка", 2, 1)
```

Завершает формирование SQL-запроса с диалогом прогресс-бара.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Текст сообщения об ошибке. По умолчанию пустая строка |

**Возвращает**: `Empty` — пустое значение.

```EME-L
' Остановить формирование запроса '
is_stop_execute_query("Превышен лимит строк")
```

### is_query — значение поля запроса

Возвращает значение поля запроса или указатель на объект запроса.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Нет | Имя источника данных или запроса. Пусто = текущий запрос |
| args[1] | String | Нет | Имя поля запроса |
| args[2..N] | String | Нет | Реляционный путь (поля-ссылки) |

**Возвращает**: `Variant` — значение поля, указатель на запрос или пустое значение.

```EME-L
' Получить значение поля текущего запроса '
name = is_query("", "Наименование")

' Получить значение через реляционный путь '
inn = is_query("", "Контрагент", "ИНН")

' Получить указатель на запрос '
q = is_query("МойЗапрос")
```

### is_query_data — значение поля по переменной диалога

Возвращает значение поля запроса по имени переменной диалога или имени запроса.
Предназначена для использования в отчётах.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя переменной диалога или запроса |
| args[1] | Integer | Да | Номер строки. Пусто = текущая строка |
| args[2] | String | Да | Имя поля |
| args[3..N] | String | Нет | Реляционный путь |

**Возвращает**: `Variant` — значение поля или пустое значение.

```EME-L
' Прочитать поле из запроса в переменной диалога '
val = is_query_data("QUERY_VAR", 0, "Сумма")
```

### is_query_result — результат запроса (итоговые значения)

Выполняет запрос и возвращает значение поля в строке 0 результата.
Предназначена для получения итоговых сумм. Должна вызываться в контексте CONST.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String | Да | Имя запроса |
| args[1] | String | Да | Имя поля |

**Возвращает**: `Variant` — значение поля в строке 0 или пустое значение.

```EME-L
' Получить итоговую сумму из запроса '
total = is_query_result("ИтогиПоСкладу", "ОбщаяСумма")
```

### is_trace_query — тестирование запроса

Тестирует запрос в режиме просмотра QUERY_BROWSER.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| — | — | — | Параметров нет |

**Возвращает**: `Boolean` — TRUE если запрос оттестирован, иначе FALSE.

```EME-L
' Проверить, отображается ли запрос в браузере '
result = is_trace_query()
If (result)
    ' Запрос оттестирован '
End If
```

### is_expimp_query_compare_ref_fld — сравнить ссылки в запросе

Сравнивает ссылки в поле запроса экспорта-импорта с БД или другой строкой по ключам.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | CIQuery | Да | Объект запроса |
| args[1] | Integer | Да | Номер строки запроса |
| args[2] | String | Да | Имя ссылочного поля |
| args[3] | Integer | Да | Номер строки для сравнения |
| args[4] | Integer | Да | Режим: 0 = с БД, ненулевое = со строкой запроса |

**Возвращает**: `Boolean` — TRUE при совпадении ключей.

```EME-L
' Сравнить ссылки с базой данных '
match = is_expimp_query_compare_ref_fld(q, 0, "СсылкаНаТовар", 5, 0)

' Сравнить ссылки двух строк запроса '
match = is_expimp_query_compare_ref_fld(q, 0, "СсылкаНаТовар", 1, 1)
```

Заполняет массив ключами из ссылочного поля запроса экспорта-импорта.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | CIQuery | Да | Объект запроса |
| args[1] | Integer | Да | Номер строки |
| args[2] | String | Да | Имя ссылочного поля |
| args[3] | CIArray | Да | Объект массива для заполнения |

**Возвращает**: `Empty` — пустое значение.

```EME-L
' Заполнить массив ключами ссылок '
arr = Object("Array")
is_expimp_query_get_refs_keys(q, 0, "Ссылки", arr)
```

### is_qtext — текст запроса

Возвращает текст запроса по имени или по ссылке.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| args[0] | String/Integer | Да | Имя запроса (строка) или ссылка (целое число) |

**Возвращает**: `String` — текст запроса или пустое значение.

```EME-L
' Получить текст запроса по имени '
text = is_qtext("SalesQuery")

' Получить текст по ссылке '
text = is_qtext(12345)
```
