### IrrBillboard — класс билборда движка Irrlicht (язык EME-L)

Класс `IrrBillboard` в языке EME-L — это класс билборда (billboard) движка Irrlicht. Представляет плоский прямоугольный спрайт, который автоматически поворачивается лицом к камере, сохраняя неизменной видимую площадь независимо от угла обзора. Наследует все методы базового класса `IrrNode` — управление положением, поворотом, масштабом, видимостью, именем, материалом и текстурой узла — и расширяет их возможностью задавать собственные ширину и высоту через метод `SetSize`.

Класс `IrrBillboard` в системе EME.WMS используется в прикладных скриптах EME-L для отображения двумерных изображений в трёхмерной сцене: маркеров, индикаторов состояния, меток оборудования, частиц и декоративных элементов. Размеры билборда масштабируются глобальным масштабом устройства `IrrDevice`, а не масштабом родительской ноды — это обеспечивает единообразный визуальный размер билборда относительно камеры при любом расположении в сцене.

### Создание объекта IrrBillboard

В блоке регистрации `BEGIN_IMPLEMENT(CIIrrBillboard)` в C++ исходнике нет зарегистрированных конструкторов с аргументами (`IMPLEMENT_CTOR` отсутствует), поэтому объект `IrrBillboard` не создаётся через `Object("IrrBillboard", ...)` напрямую. Канонический способ получить билборд в языке EME-L — вызвать метод `AddBillboard()` у родительского узла `IrrNode`. Этот метод создаёт дочерний узел-билборд и возвращает ссылку на новый объект `IrrBillboard`.

```EME-L
'Создать устройство Irrlicht и корневой узел сцены'
objDevice = Object("IrrDevice");
objRoot   = objDevice.GetRootNode();

'Добавить дочерний билборд к корневому узлу'
objBoard = objRoot.AddBillboard();
```

При создании билборда в методе `CreateBillboard` в системе EME.WMS устанавливаются значения по умолчанию: идентификатор узла Irrlicht `0`, тип узла `Billboard`, масштаб `(1, 1, 1)`, имя `""` (пустая строка). Ширина и высота при создании равны `0` — их необходимо задать методом `SetSize`.

При загрузке билборда из дерева свойств PropertyTree (`LoadNode`) в языке EME-L используются ключи размеров: `BillboardWidth` (ширина, по умолчанию `0.0`) и `BillboardHeight` (высота, по умолчанию `0.0`).

### Метод управления размером билборда IrrBillboard

Метод `SetSize` в языке EME-L устанавливает ширину и высоту билборда в единицах сцены. Переданные значения умножаются на глобальный масштаб устройства `IrrDevice.m_dGlobalScale` перед вызовом Irrlicht.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetSize(arg0, arg1)` | arg0: Real (ширина), arg1: Real (высота) | Empty | Устанавливает ширину и высоту билборда в единицах сцены с учётом глобального масштаба устройства. Если текущий узел не является билбордом, выводится сообщение об ошибке. |

Метод `SetSize` в системе EME.WMS вызывает `setSize` Irrlicht с порядком аргументов высота, ширина, ширина (`height, width, width`) — это особенность Irrlicht `IBillboardSceneNode::setSize`, где второй и третий параметры задают верхнюю и нижнюю ширину соответственно. Для прикладного скрипта EME-L это прозрачно: достаточно передать желаемую ширину и высоту, а преобразование выполняется внутри метода.

### Примеры использования IrrBillboard в языке EME-L

Создание билборда с текстурой над зоной ворот в трёхмерной сцене:

```EME-L
'Получить корневой узел сцены IrrDevice'
objRoot = objDevice.GetRootNode();

'Создать дочерний билборд'
objBoard = objRoot.AddBillboard();

'Задать размеры: ширина 1.0, высота 0.2 единицы сцены'
objBoard.SetSize(1.0, 0.2);

'Сместить над зоной ворот по оси Y'
objBoard.SetPosition(0.0, GateHeight + 0.3, 0.0);

'Включить прозрачность по альфа-каналу и наложить текстуру'
objBoard.SetMaterialType("trans_alphach");
objBoard.SetMaterialTexture(BillboardImage);
objBoard.SetLighting(FALSE);
```

В системе EME.WMS типичный сценарий с билбордом: маркер над стеллажом, табличка зоны, индикатор состояния товара. Для полупрозрачных наложений используется тип материала `trans_alphach` (прозрачность по альфа-каналу), освещение обычно отключается (`SetLighting(FALSE)`), так как билборд несёт готовое изображение текстуры.
