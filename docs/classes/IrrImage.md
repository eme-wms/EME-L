### IrrImage — класс двумерного изображения движка Irrlicht (язык EME-L)

Класс `IrrImage` в языке EME-L — это класс двумерного изображения движка Irrlicht. Представляет растровое изображение, хранящееся в формате ARGB32 (8 бит на канал: альфа, красный, зелёный, синий), и предоставляет средства для его создания, рисования примитивов и текста, загрузки из файла, сохранения, копирования, поворота, а также наложения одного изображения на другое.

Основные возможности класса `IrrImage` в системе EME.WMS: создание изображения заданного размера; заливка фона и прямоугольников цветом RGBA; рисование прямоугольников, эллипсов, текста и других изображений; настройка пера, кисти, шрифта, сглаживания примитивов и текста; получение размеров изображения и габаритов строки текста; загрузка и сохранение изображений из файла или в файл с указанием формата; копирование части изображения в другой объект `IrrImage` и поворот изображения; наложение одного изображения на другое в заданной области прямоугольника `Rect`.

Класс `IrrImage` в языке EME-L используется в прикладных скриптах для подготовки и обработки двумерной графики, текстур и элементов интерфейса перед отображением в сцене Irrlicht. Внутренне реализован поверх `QImage` и `QPainter` из библиотеки Qt.

### Создание объекта IrrImage в языке EME-L

Класс `IrrImage` имеет два конструктора, зарегистрированных в блоке `BEGIN_IMPLEMENT(CIIrrImage)` — по одному на каждое число аргументов. Объекты создаются через `Object("IrrImage", ...)`.

```EME-L
'Конструктор без аргументов — изображение 32x32 пикселя'
objImage = Object("IrrImage");
```

```EME-L
'Конструктор с указанием ширины и высоты в пикселях'
objImage = Object("IrrImage", 1000, 1000);
```

Конструктор без аргументов (0 параметров) создаёт изображение размером 32×32 пикселя. Конструктор с двумя аргументами (ширина, высота) создаёт изображение заданного размера. В обоих случаях формат пикселей — ARGB32 (32 бита на пиксель, `QImage::Format_ARGB32`), что соответствует формату текстур Irrlicht `ECF_A8R8G8B8`.

### Методы цвета и заливки IrrImage

Методы заливки в языке EME-L принимают цвет как целое число в формате RGBA. Для формирования значения цвета используйте системную функцию `is_RGBA(r, g, b, a)`, где каждый компонент — число от 0 до 255.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Fill(arg0)` | arg0: Integer (RGBA) | Empty | Заполняет всё изображение заданным цветом. |
| `FillRect(arg0, arg1, arg2, arg3, arg4)` | arg0: Integer (left), arg1: Integer (top), arg2: Integer (width), arg3: Integer (height), arg4: Integer (RGBA) | Empty | Заполняет прямоугольник цветом. Координаты `left` и `top` задают верхний левый угол; `width` и `height` — размеры прямоугольника в пикселях. |

### Методы пера, кисти и шрифта IrrImage

Перо (pen) в языке EME-L задаёт цвет линий и контуров фигур. Кисть (brush) задаёт цвет заливки фигур. Шрифт (font) устанавливает гарнитуру, размер и вес для рисования текста методом `DrawText`.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetPen(arg0)` | arg0: Integer (RGBA) | Empty | Задаёт цвет пера для последующих операций рисования линий и контуров. |
| `SetBrush(arg0)` | arg0: Integer (RGBA) | Empty | Задаёт цвет кисти для последующих операций заливки фигур. |
| `SetFont(arg0, arg1)` | arg0: String (имя шрифта), arg1: Integer (размер в пунктах) | Empty | Устанавливает текущий шрифт для рисования текста. |
| `SetFont(arg0, arg1, arg2)` | arg0: String (имя), arg1: Integer (размер, пт), arg2: Integer (вес 0–99) | Empty | То же с указанием веса шрифта по шкале QFont: 50 — нормальный, 75 — жирный. |
| `SetAntialiasing(arg0)` | arg0: Boolean | Empty | Включает (`TRUE`) или отключает (`FALSE`) сглаживание примитивов при рисовании. |
| `SetTextAntialiasing(arg0)` | arg0: Boolean | Empty | Включает (`TRUE`) или отключает (`FALSE`) сглаживание текста. |

Вес шрифта в системе EME.WMS соответствует шкале Qt QFont: 0 — минимальный, 50 — `Normal` (нормальный), 75 — `Bold` (жирный), 99 — максимальный. Значения за пределами 0–99 не предусмотрены.

### Методы рисования примитивов и текста IrrImage

Методы рисования в языке EME-L используют текущие перо и кисть, установленные методами `SetPen` и `SetBrush`. Текст рисуется шрифтом, заданным методом `SetFont`, с выравниванием, управляемым строкой флагов.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `DrawRect(arg0, arg1, arg2, arg3)` | arg0: Integer (left), arg1: Integer (top), arg2: Integer (width), arg3: Integer (height) | Empty | Рисует контур прямоугольника текущим пером. |
| `DrawEllipse(arg0, arg1, arg2, arg3)` | arg0: Integer (left), arg1: Integer (top), arg2: Integer (width), arg3: Integer (height) | Empty | Рисует эллипс, вписанный в ограничивающий прямоугольник, текущим пером. |
| `DrawText(arg0, arg1, arg2, arg3, arg4, arg5)` | arg0: Integer (left), arg1: Integer (top), arg2: Integer (width), arg3: Integer (height), arg4: String (флаги выравнивания), arg5: String (текст) | Empty | Рисует текст `arg5` в прямоугольнике с выравниванием `arg4`. |
| `GetTextHeight(arg0)` | arg0: String (текст) | Integer | Высота строки текста в пикселях при текущем шрифте. |
| `GetTextWidth(arg0)` | arg0: String (текст) | Integer | Ширина строки текста в пикселях при текущем шрифте. |

Флаги выравнивания в `DrawText` в языке EME-L передаются строкой через точку с запятой. Допустимые значения: `"Left"` — по левому краю, `"Right"` — по правому краю, `"HCenter"` — по центру горизонтали; `"Top"` — по верхнему краю, `"Bottom"` — по нижнему краю, `"VCenter"` — по центру вертикали. По умолчанию применяется выравнивание `"HCenter;VCenter"`; если строка флагов пуста, также используется выравнивание по центру. Неизвестное имя флага вызывает ошибку выполнения.

Методы `GetTextHeight` и `GetTextWidth` в системе EME.WMS возвращают габариты текста в пикселях при текущем шрифте. Эти методы позволяют динамически рассчитать размер прямоугольника для `DrawText` до фактической отрисовки.

### Методы загрузки и сохранения IrrImage

Методы загрузки и сохранения в языке EME-L работают с файлами изображений на диске и через файловую систему EME (запись «Файловая система» или «Файловые ресурсы»). Формат изображения может определяться автоматически по расширению или задаваться явно строкой.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `LoadFile(arg0)` | arg0: String (имя файла) | Boolean | Загружает изображение из файла. `TRUE` — загрузка выполнена, `FALSE` — ошибка. |
| `LoadFile(arg0, arg1)` | arg0: String (имя файла), arg1: String (формат) | Boolean | Загружает изображение из файла в указанном формате. |
| `SaveFile(arg0)` | arg0: String (имя файла) | Boolean | Сохраняет изображение в файл. `TRUE` — сохранение выполнено, `FALSE` — ошибка. |
| `SaveFile(arg0, arg1)` | arg0: String (имя файла), arg1: String (формат) | Boolean | Сохраняет изображение в файл в указанном формате. |

Метод `LoadFile` в системе EME.WMS сначала пытается прочитать файл через файловую систему EME (запись «Файловая система» или «Файловые ресурсы»); если файл не найден в базе, используется прямой путь на диске. При успешной загрузке содержимое изображения заменяется новым.

|-------|-----------|------------|----------|
| `GetWidth()` | — | Integer | Ширина изображения в пикселях. |

### Методы операций с изображениями IrrImage

Эти методы в языке EME-L принимают ссылки на другие объекты — `IrrImage` и `Rect`. Для создания прямоугольника `Rect` используйте `Object("Rect", left, top, right, bottom)`, где `left`/`top` — координаты левого верхнего угла, `right`/`bottom` — правого нижнего.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `CopyImage(arg0, arg1)` | arg0: IrrImage (по ссылке), arg1: Rect (по ссылке) | Empty | Копирует часть текущего изображения в объект `arg0`. Прямоугольник `arg1` задаёт копируемую область текущего изображения. |
| `RotateImage(arg0, arg1)` | arg0: IrrImage (по ссылке), arg1: Real (угол, градусы) | Empty | Поворачивает текущее изображение на `arg1` градусов и помещает результат в `arg0`. |
| `DrawImage(arg0, arg1)` | arg0: IrrImage (по ссылке), arg1: Rect (по ссылке) | Empty | Рисует изображение `arg0` на текущем изображении в области `arg1`. |

Метод `RotateImage` в системе EME.WMS поворачивает изображение по часовой стрелке на указанный угол в градусах. Результат помещается в целевой объект `arg0`, перезаписывая его предыдущее содержимое. Положительный угол означает поворот по часовой стрелке.

Метод `CopyImage` в языке EME-L копирует пиксели из прямоугольной области текущего изображения в целевой объект `arg0`. Размер целевого изображения автоматически устанавливается равным размеру копируемой области.

Метод `DrawImage` накладывает изображение `arg0` на текущее изображение, масштабируя его до размеров прямоугольника `arg1`.

### Примеры использования IrrImage в языке EME-L

Загрузка изображения, рисование и сохранение результата:

```EME-L
'Загрузить исходное изображение из файла'
objImage = Object("IrrImage");
objImage.LoadFile("D:\\1.png");

'Задать чёрное перо и нарисовать контур прямоугольника'
objImage.SetPen(is_RGBA(0, 0, 0, 255));
objImage.DrawRect(10, 10, 100, 20);

'Сохранить результат в новый файл'
objImage.SaveFile("D:\\result.png");
```

Создание изображения заданного размера с заливкой, текстом и поворотом:

```EME-L
'Создать изображение 200x200 с жёлтой заливкой'
objImg = Object("IrrImage", 200, 200);
objImg.Fill(is_RGBA(0xFD, 0xD8, 0x2F, 0xFF));

'Настроить шрифт и нарисовать текст по центру'
objImg.SetFont("Arial", 16);
objImg.SetPen(is_RGBA(0, 0, 0, 255));
objImg.DrawText(0, 0, 200, 200, "HCenter;VCenter", "Hello!");

'Повернуть на 90 градусов в новый объект IrrImage'
objRotated = Object("IrrImage");
objImg.RotateImage(objRotated, 90);

objRotated.SaveFile("D:\\rotated.png");
```

Копирование части изображения и наложение на другое:

```EME-L
'Загрузить источник и создать пустой холст'
objSource = Object("IrrImage");
objSource.LoadFile("3D\\rack.png");
objCanvas = Object("IrrImage", 1000, 1000);

'Скопировать фрагмент (60,25)-(110,202) из источника'
objFragment = Object("IrrImage");
objSource.CopyImage(objFragment, Object("Rect", 60, 25, 110, 202));

'Нарисовать фрагмент на холсте'
objCanvas.DrawImage(objFragment, Object("Rect", 0, 0, 50, 177));

objCanvas.SaveFile("3D\\composed.png");
```

Измерение габаритов текста при текущем шрифте:

```EME-L
objImage = Object("IrrImage");
objImage.SetFont("Arial", 24);
Height = objImage.GetTextHeight("Text");
Width  = objImage.GetTextWidth("Text");
```
