# Класс Array

> Синонимы для поиска: CIArray Array класс EME-L, динамический массив EME-L, массив произвольных типов EME-L, контейнер данных EME-L, GetAt SetAt SetAtGrow GetSize SetSize SetSizeGrow IsEmpty Add InsertAt Remove RemoveAt RemoveAll RemoveLast GetLast Find LowerBound BinarySearch Lookup Sort SmartSort SetFirstLine SetNextLine IsValidLine GetLine SetLine Get Put DeleteLine GetContent Copy AddArray Compare AddDataStorage GetRefArray PutRefArray, итерация Loop по массиву, сортировка массива EME-L, массив ссылок EME-L, ArrayOfLong ArrayOfDouble Copy, CDataStorage в массиве, базовый контейнер EME.WMS

Класс `Array` в языке EME-L — динамический массив произвольных типов данных. Класс Array в системе EME предназначен для хранения и манипулирования упорядоченными наборами значений различных типов, поддерживает операции добавления, удаления, вставки, поиска и сортировки элементов, а также доступ по индексу. Индексация элементов массива в языке EME-L начинается с 0.

Класс Array в языке EME-L предоставляет методы для работы с текущей «строкой» (текущим индексом), что позволяет использовать массив в качестве источника данных в циклах `Loop`. Поддерживается копирование данных из других массивов (Array, ArrayOfLong, ArrayOfDouble), сериализация, отладочный вывод и специальные операции с массивами ссылок. Класс Array — основной контейнер для организации коллекций данных в прикладных скриптах EME-L и системе EME.WMS.

## Создание объекта класса Array

Конструктор класса Array в языке EME-L имеет две перегрузки — без аргументов (размер 0) и с заданным начальным размером.

```EME-L
'Пустой массив — размер 0, заполняется через Add или SetSize'
arr = Object("Array");

'Массив из 10 элементов, заполненных Empty-значениями'
arr = Object("Array", 10);
```

В обеих перегрузках класса Array в языке EME-L внутренний курсор «текущей строки» инициализируется значением -1: до вызова `SetFirstLine` или `SetLine` методы `GetLine()` и `IsValidLine()` возвращают -1 и FALSE соответственно. Это важно для корректного запуска цикла `Loop (arr)`.

## Управление размером класса Array

Методы класса Array в языке EME-L для управления размером динамического массива.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetSize()` | — | Integer | Текущее количество элементов в массиве. |
| `SetSize(arg0)` | arg0: Integer | Empty | Новый размер массива. При расширении новые элементы заполняются Empty; при сокращении лишние элементы удаляются с конца. |
| `SetSize(arg0, arg1)` | arg0: Integer (размер), arg1: Integer (буфер) | Empty | То же с заданным размером внутреннего буфера — массив будет расширяться до `arg1` без реаллокации. |
| `IsEmpty()` | — | Boolean | TRUE — массив пустой. |

> Не рекомендуется увеличивать размер массива класса Array параллельно с его заполнением — это приводит к перевыделению памяти. Предпочитайте `Add` или задавайте размер через `SetSize` один раз перед заполнением.

## Доступ по индексу в классе Array

Методы доступа класса Array в языке EME-L по индексу элемента.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetAt(arg0)` | arg0: Integer | Любой тип | Элемент по индексу. Индекс должен быть в пределах 0..`GetSize()`-1; при выходе за пределы или отрицательном индексе выбрасывает исключение. |
| `SetAt(arg0, arg1)` | arg0: Integer, arg1: любой тип | Empty | Присваивает значение элементу по индексу. Контракт индекса тот же, что у `GetAt`. |
| `SetAtGrow(arg0, arg1)` | arg0: Integer, arg1: любой тип | Empty | Присваивает значение по индексу; если индекс превышает текущий размер, массив автоматически расширяется, а промежуточные элементы заполняются Empty. |
| `GetLast()` | — | Любой тип или Empty | Последний элемент массива. Если массив пустой — Empty. |

## Добавление и удаление в классе Array

Методы класса Array в языке EME-L для добавления и удаления элементов динамического массива.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Add(arg0)` | arg0: любой тип | Integer | Добавляет элемент в конец. Возвращает индекс добавленного элемента и сдвигает на него внутренний курсор. |
| `InsertAt(arg0, arg1)` | arg0: Integer, arg1: любой тип | Empty | Вставляет элемент по индексу. При выходе за текущий размер расширяется аналогично `SetAtGrow`. |
| `Remove()` | — | Любой тип | Удаляет последний элемент и возвращает его. Выбрасывает исключение, если массив пустой. |
| `RemoveAt(arg0)` | arg0: Integer | Любой тип | Удаляет элемент по индексу (0..`GetSize()`-1) и возвращает его. Выбрасывает исключение при выходе за пределы или если массив пустой. |
| `RemoveLast()` | — | Любой тип или Empty | Удаляет последний элемент и возвращает его. Если массив пустой — Empty (без исключения). |
| `RemoveAll()` | — | Empty | Удаляет все элементы и очищает внутренний пул объектов. |

Если `Add` класса Array получает объект, созданный внутри метода, при выходе из метода этот объект будет удалён. Для хранения объектов между вызовами в языке EME-L используйте классы `ArrayOfObjects` или `DataStorage`.

## Поиск в классе Array

Методы поиска класса Array в языке EME-L работают по прямому сравнению элементов. `LowerBound` и `BinarySearch` требуют предварительно отсортированного массива — на неотсортированном поведение непредсказуемо.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Find(arg0)` | arg0: любой тип | Integer | Индекс первого вхождения. Если элемент не найден — -1. |
| `LowerBound(arg0)` | arg0: любой тип | Integer | Индекс первого элемента, не меньшего `arg0`, в отсортированном массиве. Если совпадения нет — -1. |
| `BinarySearch(arg0)` | arg0: любой тип | Boolean | TRUE — элемент найден бинарным поиском в отсортированном массиве. |
| `Lookup(arg0)` | arg0: Integer | Любой тип или Empty | Элемент по индексу. При выходе за пределы (включая отрицательный индекс) возвращает Empty. |
| `Lookup(arg0, arg1)` | arg0: Integer, arg1: любой тип | Любой тип | Элемент по индексу. При выходе за пределы возвращает значение по умолчанию `arg1`. |

## Сортировка класса Array

Тип сортировки класса Array в языке EME-L определяется по типу первого элемента массива: строковая (`VT_BSTR`), вещественная (`VT_R4`/`VT_R8`) или целочисленная (прочие типы).

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Sort()` | — | Empty | Сортировка по возрастанию. |
| `Sort(arg0)` | arg0: Boolean | Empty | TRUE — по убыванию, FALSE — по возрастанию (по умолчанию). |
| `SmartSort(arg0, arg1[, arg2...])` | arg0: объект (обычно `Object()`), arg1: String (имя метода-предиката), arg2…: дополнительные параметры | Empty | Сортировка пользовательским предикатом. Минимум 2 аргумента. Метод-предикат принимает два сравниваемых элемента и возвращает результат сравнения (Boolean или Integer). Выбрасывает исключение, если в массиве есть ссылка на удалённый объект. |

## Итерация (цикл Loop) по классу Array

Методы класса Array в языке EME-L для работы с текущей «строкой» позволяют использовать массив как источник данных в цикле `Loop`. Внутренний курсор инициализируется значением -1.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetFirstLine()` | — | Empty | Устанавливает курсор на индекс 0. |
| `SetNextLine()` | — | Empty | Сдвигает курсор на +1. |
| `IsValidLine()` | — | Boolean | TRUE — курсор в пределах массива. До `SetFirstLine` — FALSE. |
| `IsValidLine(arg0)` | arg0: Integer | Boolean | TRUE — элемент с указанным индексом существует. |
| `GetLine()` | — | Integer | Текущий индекс. До `SetFirstLine` — -1. |
| `SetLine(arg0)` | arg0: Integer | Empty | Устанавливает курсор на указанный индекс. |
| `Get()` | — | Любой тип | Значение элемента текущей строки. Выбрасывает исключение, если массив пустой или курсор вне пределов. |
| `Put(arg0)` | arg0: любой тип | Empty | Присваивает значение текущему элементу. Выбрасывает исключение при пустом массиве или выходе курсора за пределы. |
| `DeleteLine()` | — | Empty | Удаляет элемент текущей строки. Выбрасывает исключение при пустом массиве или выходе курсора за пределы. |

Для перебора элементов массива класса Array в языке EME-L предпочтителен цикл `Loop (arr)` — он автоматически вызывает `SetFirstLine`, `IsValidLine` и `SetNextLine`. Ручной цикл `For (i=0; i<arr.GetSize(); i=i+1)` с обращением через `GetAt(i)` допустим, но многословен. Не используйте `Object("Array")` внутри циклов — создавайте массив до итерации.

## Преобразование и отладка класса Array

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetContent()` | — | String | Строковое представление содержимого для отладчика EME-L. Элементы разделяются строкой `"; "` (точка с запятой + пробел). Использовать только для отладки. |

## Копирование и сравнение класса Array

Методы класса Array в языке EME-L для копирования и сравнения массивов.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Copy(arg0)` | arg0: Array / ArrayOfLong / ArrayOfDouble (по ссылке) | Empty | Копирует содержимое источника в текущий массив, перезаписывая его. Выбрасывает исключение, если аргумент не поддерживаемый класс массива. |
| `AddArray(arg0)` | arg0: Array (по ссылке) | Empty | Добавляет все элементы другого `Array` в конец текущего. |
| `Compare(arg0)` | arg0: Array (по ссылке) | Boolean | Поэлементное сравнение: размеры, типы и значения. TRUE — массивы равны. Сравнивает типы Empty, Integer, Real, String, Boolean, DateTime, байтовый массив, объект, Null. |
| `AddDataStorage()` | — | CDataStorage | Создаёт новый объект `CDataStorage`, добавляет его в конец массива и возвращает. |
| `GetRefArray()` | — | Array | Содержимое массива в виде массива ссылок. Если массив пустой — массив из четырёх пустых ссылок. Выбрасывает исключение, если в массиве есть элемент, отличный от Integer, Real или Empty. |
| `PutRefArray(arg0)` | arg0: Array | Empty | Заполняет текущий массив данными из массива ссылок, перезаписывая его. |

## Примеры работы с классом Array

Инициализация массива-члена класса в конструкторе — типовой паттерн EME-L:

```EME-L
'Массив пунктов меню'
m_MenuItems = Object("Array");

'Массив имён методов-обработчиков'
m_Handlers = Object("Array");
```

Сбор значений из запроса Query в массив через `Add` в языке EME-L:

```EME-L
oldGoods = Object("Array");
'Запись товарных единиц с давностью отгрузки больше порога'
For (i = 0; i < qGoods.GetNoOfLines(); i = i + 1)
    If ((is_dos_date() - qGoods.GetData("Дата отгрузки", i)) > threshold)
        oldGoods.Add(qGoods.GetData("@Товарная единица", i));
    End If
End For
```

Перебор массива класса Array через `Loop` в языке EME-L:

```EME-L
arrReports = Object("Array");
arrReports.Add("ТТН");
arrReports.Add("Накладная");

Loop (arrReports)
    csName = arrReports.Get();
    'обработка csName...'
End Loop
```

Безопасный доступ по индексу через `Lookup` класса Array:

```EME-L
arr = Object("Array", 5);
'При выходе за пределы — Empty, без исключения'
val = arr.Lookup(10);
'При выходе за пределы — явно заданное значение по умолчанию'
val = arr.Lookup(10, "нет данных");
```
