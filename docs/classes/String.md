# Класс String

> Синонимы для поиска: CIString String класс EME-L, строковые операции EME-L, GetLength GetAt SetAt AsString AsReal AsInteger AsDateTime SetString Compare CompareNoCase Add Mid Left Right MakeUpper MakeLower MakeCapitalLetter MakeReverse TrimLeft TrimRight Find FindNoCase ReverseFind ReverseFindSubStr FindChar FindOneOf Format Replace AnsiToOem OemToAnsi ExtractUpto ExtractUptoString ExtractUptoEx RemoveSymbol GetContent IsEmpty ISO88595ToAnsi AnsiToISO88595 UTF8ToAnsi AnsiToUTF8 UTF16ToAnsi AnsiToUTF16 IsNumericOnly Split, конкатенация строк EME-L, подстрока EME-L, кодировка EME-L, CDataStorage EME-L

Класс `String` в языке EME-L — класс для работы со строками текста. В языке EME-L класс String предоставляет полный набор операций для создания, модификации, поиска, форматирования и преобразования строковых данных. Поддерживает извлечение подстрок, конкатенацию, замену фрагментов, изменение регистра, обрезку пробельных символов и преобразование между различными кодировками (ANSI, OEM, UTF-8, UTF-16, ISO-8859-5).

Объект `String` в системе EME создаётся либо как пустая строка, либо с инициализацией начальным значением. Методы класса модифицируют внутреннее состояние объекта (например, MakeUpper, TrimLeft, Replace) и возвращают изменённое строковое значение, либо возвращают результат операции без изменения состояния (например, Mid, Left, GetLength).

Класс поддерживает преобразование строк в числовые типы (AsInteger, AsReal, AsDateTime) и обратно, а также разбиение строки на массив подстрок по заданному разделителю. Используется в прикладных скриптах EME-L для обработки и манипулирования текстовыми данными.

## Создание объекта класса String

Класс `String` в языке EME-L имеет два конструктора — один на каждое число аргументов.

```EME-L
'Конструктор без аргументов — пустая строка'
csString = Object("String");
```

```EME-L
'Конструктор с начальным значением — любое значение приводится к строке'
csString = Object("String", "Готово");
```

Конструктор с одним аргументом в языке EME-L принимает любой тип, кроме Object. Переданное значение приводится к строковому представлению. Если строка не задана в конструкторе, её можно задать позже методом SetString.

## Базовые операции

Методы базовых операций класса String в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetLength()` | — | Integer | Количество символов в строке. |
| `IsEmpty()` | — | Boolean | TRUE — строка пуста (длина 0), иначе FALSE. |
| `IsNumericOnly()` | — | Boolean | TRUE — строка состоит только из цифр 0–9 (коды ASCII 48–57). Не поддерживает плавающую точку, отрицательные числа, пробелы и любые не-цифровые символы. Пустая строка возвращает TRUE. |
| `Add(arg0)` | arg0: String | String | Конкатенация: добавляет строку в конец текущей. Модифицирует объект и возвращает результирующую строку. |
| `Compare(arg0)` | arg0: String | Boolean | TRUE — строки идентичны с учётом регистра, иначе FALSE. |
| `CompareNoCase(arg0)` | arg0: String | Integer | Лексикографическое сравнение без учёта регистра. 0 — равны, отрицательное — текущая меньше, положительное — текущая больше. |
| `GetAt(arg0)` | arg0: Integer | String | Символ по индексу (с 0). Выбрасывает исключение при выходе за пределы. |
| `SetAt(arg0, arg1)` | arg0: Integer, arg1: String | Empty | Заменяет символ по индексу. Используется первый символ arg1. Выбрасывает исключение при выходе за пределы. |

## Преобразование типов

Методы преобразования строки в различные типы данных в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AsString()` | — | String | Текущее содержимое объекта в виде строки. |
| `AsInteger()` | — | Integer | Целочисленное представление строки. Если в левой части корректное число — оно извлекается. Если цифр нет — 0. Для чисел с плавающей точкой возвращается целая часть. |
| `AsReal()` | — | Real | Числовое представление строки. Десятичный разделитель — точка или запятая. Если строка не содержит цифр или не удаётся распознать число — 0.0. |
| `AsDateTime()` | — | DateTime | Значение даты/времени. Формат должен соответствовать региональным настройкам ОС. При невозможности распознать — пустое значение даты (0). |

## Извлечение подстрок

Методы извлечения подстрок класса String в языке EME-L возвращают результат без изменения состояния объекта.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Mid(arg0)` | arg0: Integer (индекс) | String | Подстрока от индекса до конца. Если индекс за пределами — пустая строка. |
| `Mid(arg0, arg1)` | arg0: Integer (индекс), arg1: Integer (длина) | String | Подстрока заданной длины от индекса. |
| `Left(arg0)` | arg0: Integer | String | Первые N символов. Если N превышает длину — исходная строка. |
| `Right(arg0)` | arg0: Integer | String | Последние N символов. Если N превышает длину — исходная строка. |

## Поиск

Методы поиска в строке в языке EME-L. Все методы возвращают индекс (с 0) или -1, если ничего не найдено.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Find(arg0)` | arg0: String (подстрока) | Integer | Индекс первого вхождения подстроки с начала строки. Если не найдено — -1. |
| `Find(arg0, arg1)` | arg0: String, arg1: Integer (индекс) | Integer | То же с поиском начиная с указанной позиции. Если не найдено — -1. |
| `FindNoCase(arg0)` | arg0: String | Integer | Индекс первого вхождения без учёта регистра (с начала). Если не найдено — -1. |
| `FindNoCase(arg0, arg1)` | arg0: String, arg1: Integer | Integer | То же начиная с указанной позиции. Если не найдено — -1. |
| `ReverseFind(arg0)` | arg0: String (символ) | Integer | Индекс последнего вхождения символа (используется первый символ arg0). Если не найдено — -1. |
| `ReverseFindSubStr(arg0)` | arg0: String | Integer | Индекс начала последнего вхождения подстроки. Если не найдено — -1. |
| `FindChar(arg0, arg1)` | arg0: String (символ), arg1: Integer (индекс) | Integer | Индекс первого вхождения символа начиная с заданной позиции. Если не найдено — -1. |
| `FindOneOf(arg0)` | arg0: String (набор) | Integer | Индекс первого символа из набора. Если ни один не найден — -1. |

## Регистр и модификация

Методы модификации класса String в языке EME-L изменяют содержимое объекта и возвращают изменённую строку.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `MakeUpper()` | — | String | Все символы в верхнем регистре. |
| `MakeLower()` | — | String | Все символы в нижнем регистре. |
| `MakeCapitalLetter()` | — | String | Первый символ в верхнем регистре, остальные — в нижнем. |
| `MakeReverse()` | — | String | Символы в обратном порядке. |
| `TrimLeft()` | — | String | Удаляет пробельные символы (пробел, табуляция, перевод строки, возврат каретки) слева. |
| `TrimLeft(arg0)` | arg0: String (символы) | String | Удаляет слева любые символы из набора. |
| `TrimRight()` | — | String | Удаляет пробельные символы справа. |
| `TrimRight(arg0)` | arg0: String (символы) | String | Удаляет справа любые символы из набора. |
| `Replace(arg0, arg1)` | arg0: String (что), arg1: String (на что) | Integer | Заменяет все вхождения подстроки. Возвращает количество замен. Модифицирует объект. |
| `RemoveSymbol(arg0)` | arg0: String (символ) | String | Удаляет изолированные вхождения указанного символа целиком, а из серий подряд идущих одинаковых символов — каждый второй (через один). Используется первый символ arg0. Если строка или arg0 пустая — возвращается исходная строка без изменений. Модифицирует объект. |
| `Format(arg0[, arg1[, ...]])` | arg0: String (шаблон), arg1..N: любые | Empty | Форматирует строку по шаблону со спецификаторами (%s, %d, %f и т. д.). Результат сохраняется в объекте. Поддерживает произвольное число аргументов. Внутри делегирует в is_format. |
| `SetString(arg0)` | arg0: String или CDataStorage (по ссылке) | Empty | Устанавливает новое значение строки. Если arg0 — строка, она становится новым содержимым. Если arg0 — объект CDataStorage, переданный по ссылке, строка заполняется бинарным содержимым хранилища (приводится к ANSI). |

## Извлечение с удалением

Методы извлекают часть строки из объекта String в языке EME-L и удаляют её из объекта. Если разделитель отсутствует, возвращается вся строка, а объект становится пустым.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `ExtractUpto(arg0)` | arg0: String (символ) | String | Часть от начала до указанного символа включительно. |
| `ExtractUptoString(arg0)` | arg0: String (подстрока) | String | Часть от начала до подстроки включительно. |
| `ExtractUptoEx(arg0, arg1)` | arg0: String (разделитель), arg1: String (экран) | String | Часть от начала до первого неэкранированного разделителя. |

## Разбиение

Метод разбиения строки на массив подстрок в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Split(arg0, arg1)` | arg0: Array (по ссылке), arg1: String (разделитель) | Empty | Разбивает строку на подстроки и добавляет их в массив. Разделитель не может быть пустым. Выбрасывает исключение, если arg0 не Array или разделитель пуст. |

## Кодировки

Методы преобразования кодировок класса String в языке EME-L модифицируют содержимое объекта.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AnsiToOem()` | — | Empty | ANSI (Windows-1251) → DOS (CP866). |
| `OemToAnsi()` | — | Empty | DOS (CP866) → ANSI (Windows-1251). |
| `ISO88595ToAnsi()` | — | String | ISO-8859-5 → ANSI (Windows-1251). |
| `AnsiToISO88595()` | — | String | ANSI (Windows-1251) → ISO-8859-5. |
| `UTF8ToAnsi()` | — | Boolean | UTF-8 → ANSI (Windows-1251). TRUE — успешно, FALSE — ошибка конвертации. |
| `AnsiToUTF8()` | — | Boolean | ANSI (Windows-1251) → UTF-8. Всегда возвращает TRUE. |
| `AnsiToUTF16(arg0)` | arg0: CDataStorage (по ссылке) | Empty | ANSI → UTF-16, результат записывается в CDataStorage. Исходный объект не изменяется. Выбрасывает исключение, если arg0 не CDataStorage. |
| `UTF16ToAnsi(arg0)` | arg0: CDataStorage (по ссылке) | Empty | UTF-16 из CDataStorage → ANSI, результат помещается в текущий объект. Выбрасывает исключение, если arg0 не CDataStorage. |

## Отладка

Метод GetContent класса String в языке EME-L используется только для отладочных целей — возвращает содержимое строки с экранированными управляющими символами.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetContent()` | — | String | Содержимое строки с экранированными управляющими символами (\n, \r, \t). |

## Примеры

Форматирование строки с подстановкой значений и конкатенация в языке EME-L:

```EME-L
strTemp = Object("String");
strTemp.Format("ASN %s. Печать этикеток для позиций: %d", RegNo, CountLines);
Result = strTemp.AsString();
```

Поиск подстроки и извлечение части строки (пример из `LabelPrint.ParseVetBarcode`):

```EME-L
MainVetBarcode = Object("String", ctrlVeterinaryBarcode.Get());
Pos01 = MainVetBarcode.Find("(01)");
Pos20 = MainVetBarcode.Find("(20)");
Pos10 = MainVetBarcode.Find("(10)");
If(PRDStr != "")
    String = Object("String", PRDStr);
    Year = is_type(String.Mid(0,2),1) + 2000;
    Month = is_type(String.Mid(2,2),1);
    Day = is_type(String.Mid(4,2),1);
    PRDDate = is_date(Day, Month, Year);
End If
```

Создание пустого объекта String для накопления результата (пример из `IncomeAttachAcceptance.OnCommand`):

```EME-L
CheckMessage     = Object("String");
FilledByRule     = Object("String");
FilledByAlgoritm = Object("String");
```

Разбиение строки на массив подстрок по разделителю в языке EME-L:

```EME-L
arrRows = Object("Array");
csSource = Object("String", "строка1=|=строка2=|=строка3");
csSource.Split(arrRows, "=|=");
'arrRows содержит три элемента: строка1, строка2, строка3'
```
