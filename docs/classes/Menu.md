# Класс Menu

> Синонимы для поиска: CIMenu Menu класс EME-L, контекстное меню EME-L, popup menu, CreatePopupMenu AppendMenu TrackPopupMenu TrackPopupMenuWithMousePos SetDefaultItem CheckMenuItem GetItemsQty GetItemName, всплывающее меню, пункт меню, подменю, разделитель, MENUBARBREAK MENUBREAK SEPARATOR STRING POPUP ENABLED DISABLED GRAYED CHECKED, TPM_LEFTALIGN TPM_CENTERALIGN TPM_RIGHTALIGN TPM_LEFTBUTTON TPM_RIGHTBUTTON TPM_RETURNCMD, MF_CHECKED MF_UNCHECKED MF_BYPOSITION MF_BYCOMMAND

Класс `Menu` в языке EME-L — класс для создания и отображения контекстных меню в графическом интерфейсе приложения. В языке EME-L класс Menu предоставляет инструментарий для построения всплывающего меню: создание пустого меню, добавление текстовых пунктов и подменю, разделителей и маркеров разрыва колонок, управление состоянием пунктов (доступность, недоступность, отметка галочкой), установку пункта по умолчанию, получение количества пунктов и их текстов, а также отображение меню в заданных координатах или в позиции курсора мыши.

Основной сценарий использования в языке EME-L: создание объекта Menu, формирование структуры меню через AppendMenu, установка пункта по умолчанию при необходимости через SetDefaultItem, вызов TrackPopupMenu или TrackPopupMenuWithMousePos для отображения меню пользователю и обработка возвращенного идентификатора выбранного пункта.

Идентификаторы пунктов, передаваемые в AppendMenu, должны быть неотрицательными целыми значениями. Внутренне к идентификатору прибавляется системное смещение команд меню (START_MENU_COMMAND), поэтому возвращаемые TrackPopupMenu значения соответствуют исходным идентификаторам, заданным в AppendMenu. Рекомендуемый диапазон — 0..100, но в реальных скриптах EME-L используются значения вплоть до 999998; значение -1 трактуется как «пункт без команды» (используется для декоративных элементов календарей и плейсхолдеров).

Используется в прикладных скриптах EME-L для организации контекстного взаимодействия с пользователем через всплывающие меню.

## Создание объекта класса Menu

Класс `Menu` в языке EME-L имеет два конструктора — один на каждое число аргументов.

```EME-L
'Конструктор без аргументов — создаёт инструментарий без самого меню в языке EME-L'
objMenu = Object("Menu");
'Для использования необходимо вызвать CreatePopupMenu'
objMenu.CreatePopupMenu();
```

```EME-L
'Конструктор с флагом создания пустого меню сразу при создании объекта в языке EME-L'
objMenu = Object("Menu", TRUE);
```

Параметр `создать_меню` — boolean, по умолчанию FALSE. При TRUE конструктор сразу создаёт пустое всплывающее меню вызовом CreatePopupMenu, эквивалентным последовательности Object("Menu") + CreatePopupMenu().

## Создание и наполнение меню

Методы класса `Menu` в языке EME-L для создания и наполнения контекстного меню:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `CreatePopupMenu()` | — | Empty | Создаёт пустое всплывающее меню (popup menu), к которому затем можно добавлять пункты и подменю. |

## Добавление пунктов

Перегрузки `AppendMenu` разделяются по числу аргументов: с одним аргументом добавляются разделители и маркеры разрыва колонок, с тремя и четырьмя — текстовые пункты и подменю.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AppendMenu(arg0)` | arg0: String (тип) | Empty | Добавляет разделительный элемент или маркер разрыва колонки. Допустимые значения arg0: `"SEPARATOR"` — горизонтальная разделительная линия; `"MENUBREAK"` — новая колонка без разделительной линии; `"MENUBARBREAK"` — новая колонка, отделенная вертикальной линией. |
| `AppendMenu(arg0, arg1, arg2)` | arg0: String (тип), arg1: Integer или Menu (идентификатор/подменю), arg2: String (текст) | Empty | Добавляет текстовый пункт или вложенное подменю. Для arg0 = `"STRING"`: arg1 — числовой идентификатор пункта (целое число); рекомендуемый диапазон 0..100, но в реальных скриптах EME-L используются значения вплоть до 999998 (значение -1 — декоративный элемент без команды). arg2 — отображаемый текст. Для arg0 = `"POPUP"`: arg1 — объект Menu (вложенное подменю), arg2 — заголовок подменю. |
| `AppendMenu(arg0, arg1, arg2, arg3)` | + arg3: String (состояние) | Empty | То же с указанием состояния доступности. arg3: `"ENABLED"` — пункт доступен (по умолчанию); `"DISABLED"` — недоступен; `"GRAYED"` — недоступен и отображается серым; `"CHECKED"` — доступен и отмечен галочкой. |

## Управление состоянием пунктов

Методы класса `Menu` в языке EME-L для управления состоянием пунктов меню:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetDefaultItem(arg0)` | arg0: Integer (идентификатор) | Integer | Задаёт пункт, выделяемый полужирным шрифтом. К arg0 внутренне прибавляется системное смещение команд меню. Возвращает идентификатор выделенного пункта или -1. |
| `SetDefaultItem(arg0, arg1)` | arg0: Integer, arg1: Boolean (по_позиции) | Integer | То же с выбором способа поиска пункта. При arg1 = TRUE значение arg0 трактуется как позиция (нумерация с 0) без смещения; при FALSE (по умолчанию) — как идентификатор со смещением. |
| `SetDefaultItem(arg0, arg1, arg2)` | + arg2: Boolean (режим_по_умолчанию) | Integer | То же с режимом выбора по умолчанию. При arg2 = TRUE нажатие Enter в открытом меню возвращает идентификатор пункта по умолчанию. |
| `CheckMenuItem(arg0, arg1)` | arg0: Integer (идентификатор), arg1: Integer (флаги) | Integer | Добавляет или удаляет флажок возле пункта. arg1 — комбинация флагов: MF_CHECKED (0x0008) — установить галочку; MF_UNCHECKED (0x0000) — снять; MF_BYPOSITION (0x0400) — искать пункт по позиции; MF_BYCOMMAND (0x0000) — по идентификатору. Возвращает предыдущее состояние: MF_CHECKED (8), MF_UNCHECKED (0) или -1. |

## Отображение меню

Методы класса `Menu` в языке EME-L для отображения контекстного меню пользователю:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `TrackPopupMenu(arg0, arg1)` | arg0: Integer (x), arg1: Integer (y) | Integer | Отображает меню в заданных экранных координатах. Возвращает идентификатор выбранного пункта или -1, если ничего не выбрано. |
| `TrackPopupMenu(arg0, arg1, arg2)` | + arg2: Integer (флаги) | Integer | То же с флагами позиционирования и поведения. По умолчанию TPM_LEFTALIGN | TPM_RIGHTBUTTON | TPM_RETURNCMD. |
| `TrackPopupMenuWithMousePos()` | — | Integer | Отображает меню в текущей позиции курсора мыши. Возвращает идентификатор выбранного пункта или -1. |

### Флаги TrackPopupMenu (arg2)

Константы для композиции значения arg2 в языке EME-L:

| Флаг | Hex | Назначение |
|------|-----|-----------|
| `TPM_LEFTALIGN` | — | Выровнять меню по левому краю относительно x. |
| `TPM_CENTERALIGN` | — | Выровнять по центру относительно x. |
| `TPM_RIGHTALIGN` | — | Выровнять по правому краю относительно x. |
| `TPM_LEFTBUTTON` | — | Отслеживать только нажатия левой кнопки мыши. |
| `TPM_RIGHTBUTTON` | — | Отслеживать нажатия левой и правой кнопок мыши. |
| `TPM_RETURNCMD` | — | Возвращать идентификатор выбранного пункта вместо отправки команды окну. |

### Флаги CheckMenuItem (arg1)

Константы для композиции значения arg1 в языке EME-L:

| Флаг | Hex | Назначение |
|------|-----|-----------|
| `MF_CHECKED` | 0x0008 | Установить галочку. |
| `MF_UNCHECKED` | 0x0000 | Снять галочку. |
| `MF_BYPOSITION` | 0x0400 | Искать пункт по позиции. |
| `MF_BYCOMMAND` | 0x0000 | Искать пункт по командному идентификатору. |

### Константы типов и состояний пунктов

Значения, передаваемые строкой в AppendMenu в языке EME-L:

**Тип пункта (arg0)**: `"STRING"` (текстовый пункт), `"POPUP"` (вложенное подменю — объект Menu), `"SEPARATOR"` (горизонтальная линия), `"MENUBREAK"` (новая колонка без линии), `"MENUBARBREAK"` (новая колонка с вертикальной линией).

**Состояние (arg3)**: `"ENABLED"` (доступен, по умолчанию), `"DISABLED"` (недоступен), `"GRAYED"` (недоступен, серым цветом), `"CHECKED"` (доступен и отмечен галочкой).

|-------|-----------|------------|----------|

## Примеры

Создание простого меню у текущей позиции курсора в языке EME-L:

```EME-L
'Построить меню из трёх пунктов и показать его пользователю в языке EME-L'
Menu = Object("Menu", TRUE);
Menu.AppendMenu("STRING", 0, tr("Создать"));
Menu.AppendMenu("STRING", 1, tr("Изменить"));
Menu.AppendMenu("SEPARATOR");
Menu.AppendMenu("STRING", 100, tr("Выход"));
Action = Menu.TrackPopupMenuWithMousePos();
'Action содержит идентификатор выбранного пункта или -1'
```

Меню с вложенным подменю в языке EME-L:

```EME-L
'Главное меню с вложенным подменю "Печать" в языке EME-L'
Menu = Object("Menu", TRUE);
Submenu = Object("Menu", TRUE);
Submenu.AppendMenu("STRING", 1, tr("Печать А4"));
Submenu.AppendMenu("STRING", 2, tr("Печать А5"));
Menu.AppendMenu("POPUP", Submenu, tr("Печать"));
Menu.AppendMenu("STRING", 0, tr("Закрыть"));
Action = Menu.TrackPopupMenuWithMousePos();
```

Меню с пунктом по умолчанию и условной доступностью в языке EME-L (по образцу ARXIncoming):

```EME-L
'Пункт 3 по умолчанию (полужирный); Enter в открытом меню вернёт его идентификатор'
menu = Object("Menu", TRUE);
menu.AppendMenu("STRING", 3, tr("Отгрузить"), "ENABLED");
menu.AppendMenu("STRING", 4, tr("Переместить"), "ENABLED");
menu.SetDefaultItem(3,, TRUE);
action = menu.TrackPopupMenu(rect.GetLeft(), rect.GetTop());
If (action < 0)
    'Ничего не выбрано в языке EME-L'
    Return;
End If
'CheckMenuItem возвращает предыдущее состояние: MF_CHECKED (8), MF_UNCHECKED (0), -1'
If (menu.CheckMenuItem(action, 0) == -1)
    'Пункт не существует или недоступен — подобрать замену'
    If (menu.CheckMenuItem(4, 0) != -1)
        action = 4;
    End If
End If
```

Календарь в виде меню с разбиением на колонки через MENUBREAK в языке EME-L (по образцу BBDControl):

```EME-L
'Календарь на месяц: семь колонок (дни недели), MENUBREAK начинает новую колонку'
smenu = Object("Menu", TRUE);
smenu.AppendMenu("STRING", -1, tr("ПН"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("ВТ"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("СР"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("ЧТ"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("ПТ"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("СБ"), "DISABLED");
smenu.AppendMenu("STRING", -1, tr("ВС"), "DISABLED");
smenu.AppendMenu("MENUBREAK");
For (j = 1; j <= is_days_in_month(month, year); j = j + 1)
    smenu.AppendMenu("STRING", month * 100 + j, j);
    If (is_fdatetime(is_date(j, month, year), "%w") == "0")
        'Воскресенье — начать новую колонку в языке EME-L'
        smenu.AppendMenu("MENUBREAK");
    End If
End For
Day = smenu.TrackPopupMenuWithMousePos();
'Day содержит month*100+day или -1'
```
