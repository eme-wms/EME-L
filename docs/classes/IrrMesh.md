### IrrMesh — класс полигональной сетки движка Irrlicht (язык EME-L)

Класс `IrrMesh` в языке EME-L — это класс полигональной сетки движка Irrlicht. Представляет трёхмерный объект-сетку, загружаемый из mesh-файла или создаваемый программно через объект `IrrMeshCreator`. В системе EME.WMS используется для отображения статических 3D-моделей (складских стеллажей, напольной разметки, оборудования) в 3D-органе управления.

Класс `IrrMesh` наследует базовые возможности узла сцены Irrlicht и позволяет изменять используемую сетку методом `SetMesh`, управлять цветом и прозрачностью вершин, задавать зеркальный цвет материала первого буфера, а также добавлять и удалять объёмные тени.

Класс доступен только при сборке ядра EME-L с включённой поддержкой Irrlicht (макрос компиляции `IRRLICHT_IS_AVAILABLE`). Объект создаётся через `Object("IrrMesh")` и является узлом сцены Irrlicht: для отрисовки он должен быть добавлен как дочерний к корневому узлу `IrrDevice` или другому узлу `IrrNode`.

C++ класс: `CIIrrMesh`. Регистрация: `REGISTRY_EMEL_CLASS(IrrMesh, …)` в `p_System.cpp` под `#ifdef IRRLICHT_IS_AVAILABLE`. Блок методов: `BEGIN_IMPLEMENT(CIIrrMesh)` в `i_IrrMesh.cpp`.

### Создание объекта IrrMesh

В блоке регистрации `BEGIN_IMPLEMENT(CIIrrMesh)` макросы `IMPLEMENT_CTOR` отсутствуют — класс `IrrMesh` в языке EME-L использует единственный конструктор по умолчанию без аргументов. Источник полигональной сетки (mesh-файл или объект `IrrMeshCreator`) задается после создания методом `SetMesh`.

```EME-L
'Создать узел полигональной сетки Irrlicht без предустановленной сетки'
objMesh = Object("IrrMesh");
```

После создания объект не имеет связанной геометрии — для отображения в сцене Irrlicht необходимо вызвать метод `SetMesh`, передав путь к mesh-файлу (строка) или объект `IrrMeshCreator`. Внутренний конструктор C++ также принимает `IrrMeshBuffer`, однако через макрос `SetMesh` этот тип объекта в текущей реализации не проводится.

### Метод SetMesh — установка полигональной сетки

Метод `SetMesh` в языке EME-L загружает или формирует новую полигональную сетку и устанавливает её для текущего узла `IrrMesh`. Возвращает текущий объект `IrrMesh` (реализация `IMPLEMENT_MTYPED`, возвращаемый тип `CIIrrMesh`), что позволяет использовать цепочку вызовов.

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| arg0 | String или IrrMeshCreator (ссылка) | Да | Источник новой сетки. Строка — путь к mesh-файлу, загружаемому менеджером сцены Irrlicht. Ссылка — объект `IrrMeshCreator`, описывающий создаваемую полигональную сетку. |

**Возвращает**: `IrrMesh` — текущий объект `IrrMesh` (ссылка) для цепочки вызовов.

При передаче строки путь передаётся в `ISceneManager::getMesh` движка Irrlicht. При передаче объекта `IrrMeshCreator` вызывается его метод `CreateMesh`, формирующий полигональную сетку из описания. В обоих случаях результат устанавливается через `IMeshSceneNode::setMesh`.

### Цвет и прозрачность вершин IrrMesh

Методы управления цветом вершин полигональной сетки в языке EME-L. Цвет задаётся как 32-битное целое число в формате RGBA `0xAARRGGBB`, где каждая компонента (альфа, красный, зелёный, синий) принимает значения 0...255.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetVertexColors(arg0)` | arg0: Integer (RGBA, `0xAARRGGBB`) | Empty | Окрашивает все вершины текущей полигональной сетки заданным RGBA-цветом. Вызывает `IMeshManipulator::setVertexColors`. |
| `SetVertexColorAlpha(arg0)` | arg0: Integer (0...255) | Empty | Задаёт прозрачность (альфа-канал) всех вершин текущей полигональной сетки. 0 — полностью прозрачные вершины, 255 — полностью непрозрачные. |
| `SetSpecularColor(arg0)` | arg0: Integer (RGBA, `0xAARRGGBB`) | Empty | Задаёт зеркальный цвет материала первого буфера (`getMaterial(0)`) текущей полигональной сетки. |

Метод `SetVertexColors` полностью заменяет цвет вершин, сохраняя все четыре компонента RGBA. Метод `SetVertexColorAlpha` изменяет только альфа-канал, не затрагивая RGB-компоненты существующего цвета вершин.

### Зеркальный цвет материала IrrMesh

Метод `SetSpecularColor` в языке EME-L задаёт зеркальный (бликовый) цвет материала первого буфера полигональной сетки. Цвет передаётся как 32-битное целое в формате RGBA `0xAARRGGBB` (компоненты 0...255). Зеркальный цвет влияет на отображение бликов освещения на поверхности меша в сцене Irrlicht.

### Тени IrrMesh

Узел объёмной тени (shadow volume scene node) в языке EME-L может быть добавлен к полигональной сетке `IrrMesh` для расчёта динамических теней в сцене Irrlicht. Управление тенями осуществляется парой методов `AddShadow` и `RemoveShadow`.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AddShadow()` | — | Empty | Добавляет к текущей полигональной сетке узел объёмной тени, если тень ещё не была добавлена. Повторный вызов игнорируется. |
| `RemoveShadow()` | — | Empty | Удаляет узел объёмной тени, если тень была добавлена ранее. Безопасен при отсутствии тени. |

Метод `AddShadow` создаёт внутренний узел тени через `IMeshSceneNode::addShadowVolumeSceneNode` и сохраняет ссылку на него во внутреннем поле `m_Shadow`. Метод `RemoveShadow` вызывает `remove()` на сохранённом узле и сбрасывает ссылку. На каждый узел `IrrMesh` поддерживается не более одного узла тени.

### Примеры использования IrrMesh

Создание узла с сеткой из файла и настройка материала в скрипте EME-L:

```EME-L
'Создать узел сетки и загрузить 3D-модель из файла'
objMesh = Object("IrrMesh");
objMesh.SetMesh("3D\\rack.md2");

'Перекрасить вершины в белый цвет и задать зеркальный блик'
objMesh.SetVertexColors(0xFFFFFFFF);
objMesh.SetSpecularColor(0xFF333333);

'Добавить объёмную тень'
objMesh.AddShadow();
```

Программное построение сетки через `IrrMeshCreator` с последующей установкой в узел `IrrMesh`:

```EME-L
'Описать сетку стеллажа через IrrMeshCreator'
meshCreator = Object("IrrMeshCreator");
meshCreator.SetMaterial("3D\\rack.png", 500, 1000);
PlaneNode = meshCreator.AddPlane(-1);
meshCreator.SetImageRect(PlaneNode, 0, 0, 499, 999);

'Создать узел IrrMesh и установить в него построенную сетку'
objMesh = Object("IrrMesh");
objMesh.SetMesh(meshCreator);
objMesh.SetVertexColorAlpha(255);
```
