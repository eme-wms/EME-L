### IrrLight — класс источника света движка Irrlicht (язык EME-L)

Класс `IrrLight` в языке EME-L — это класс источника света движка Irrlicht. Представляет узел сцены Irrlicht типа Light, обеспечивающий освещение трёхмерной сцены. Позволяет настраивать рассеянный (Diffuse), окружающий (Ambient) и зеркальный (Specular) цвета света, радиус действия источника и тип: точечный (Point), прожекторный (Spot) или направленный (Directional).

Каждый цвет в классе `IrrLight` в языке EME-L передаётся как целочисленный RGBA — 32-битное значение `0xAARRGGBB`, упакованное из четырёх байтовых компонент: альфа (старший байт), красный, зелёный, синий (младший байт). Диапазон: `0x00000000` — `0xFFFFFFFF`. Радиус источника света задаётся в единицах сцены и умножается на глобальный масштаб (global scale) перед применением.

Класс `IrrLight` в системе EME.WMS наследует все методы базового класса `IrrNode` — управление положением, поворотом, масштабом, видимостью, именем, материалом и текстурой узла. Используется в EME-L-скриптах для настройки освещения при визуализации трёхмерных сцен.

### Создание объекта IrrLight

В блоке регистрации `BEGIN_IMPLEMENT(CIIrrLight)` в C++ исходнике нет зарегистрированных конструкторов с аргументами (`IMPLEMENT_CTOR` отсутствует), поэтому объект `IrrLight` не создаётся через `Object("IrrLight", ...)` напрямую. Канонический способ получить источник света в языке EME-L — вызвать метод `AddLight()` у родительского узла `IrrNode`. Этот метод создаёт дочерний узел-источник света и возвращает ссылку на новый объект `IrrLight`.

```EME-L
'Создать устройство Irrlicht и корневой узел сцены'
objDevice = Object("IrrDevice");
objRoot   = objDevice.GetRootNode();

'Добавить дочерний источник света к корневому узлу'
objLight = objRoot.AddLight();
```

При создании узла `IrrLight` методом `AddLight()` в системе EME.WMS узел инициализируется значениями по умолчанию: масштаб `(1, 1, 1)`, пустое имя, идентификатор `0`.

При инициализации источника света через `LoadNode` (загрузка из дерева свойств PropertyTree) в системе EME.WMS применяются значения по умолчанию, если соответствующие ключи отсутствуют в дереве. Эти же значения служат ориентиром для типичной конфигурации источника света в языке EME-L:

| Параметр источника света | Ключ в PropertyTree | Значение по умолчанию |
|--------------------------|---------------------|-----------------------|
| Рассеянный цвет (DiffuseColor) | `a`, `r`, `g`, `b` внутри поддерева `DiffuseColor` | `255, 255, 255, 255` (непрозрачный белый) |
| Радиус источника света | `LightRadius` | `100.0` |

### Методы цвета источника света IrrLight

Методы управления цветами источника света в языке EME-L. Все три цвета передаются как целочисленный RGBA, упакованный по схеме `0xAARRGGBB`. Значение формируется из четырёх байтовых компонент: альфа (старший байт), красный, зелёный, синий (младший байт). Диапазон: `0x00000000` — `0xFFFFFFFF`. Все методы доступны только для узлов типа Light; при вызове для узла другого типа генерируется ошибка выполнения.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetDiffuseColor(arg0)` | arg0: Integer (RGBA) | Empty | Устанавливает рассеянный (Diffuse) цвет — основной цвет света, падающего на поверхности. |
| `SetAmbientColor(arg0)` | arg0: Integer (RGBA) | Empty | Устанавливает окружающий (Ambient) цвет — равномерную фоновую засветку сцены. |
| `SetSpecularColor(arg0)` | arg0: Integer (RGBA) | Empty | Устанавливает зеркальный (Specular) цвет — цвет бликов на глянцевых поверхностях. |

### Методы радиуса и типа источника света IrrLight

Методы управления радиусом действия и типом источника света в языке EME-L. Радиус определяет, на каком расстоянии от источника свет затухает до нуля. Тип определяет характер излучения: точечный, прожекторный или направленный.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetLightRadius(arg0)` | arg0: Real (радиус) | Empty | Устанавливает радиус действия источника света в единицах сцены. Значение умножается на глобальный масштаб (global scale) перед установкой. |
| `SetLightType(arg0)` | arg0: String (тип) | Empty | Устанавливает тип источника света. Допустимые значения: `"Point"` — точечный (излучает во все стороны), `"Spot"` — прожекторный (направленный конус), `"Directional"` — направленный (параллельные лучи, имитация солнца). При неизвестном значении генерируется ошибка выполнения. |

### Примеры использования IrrLight в языке EME-L

Создание источника света и настройка цветов:

```EME-L
'Получить корневой узел сцены'
objRoot = objDevice.GetRootNode();

'Добавить точечный источник света'
objLight = objRoot.AddLight();

'Рассеянный свет — тёплый белый (непрозрачный, 0xFFFFFFFF)'
objLight.SetDiffuseColor(0xFFFFFFFF);

'Окружающий свет — тёмно-серый (0xFF404040)'
objLight.SetAmbientColor(0xFF404040);

'Зеркальные блики — чистый белый (0xFFFFFFFF)'
objLight.SetSpecularColor(0xFFFFFFFF);
```

Настройка радиуса и типа:

```EME-L
'Радиус действия — 50 единиц сцены'
objLight.SetLightRadius(50.0);

'Направленный источник (имитация солнца)'
objLight.SetLightType("Directional");
```
