# Класс HTTP

> Синонимы для поиска: CIHTTP HTTP класс EME-L, HTTP клиент EME-L, HTTP запрос EME-L, HTTPS EME-L, REST API EME-L, Open SetRequestHeader Send GetStatus GetResponseText GetResponseContentType GetRequest SetTimeouts, Basic Auth EME-L, Boost.Beast EME-L

Класс `HTTP` в языке EME-L — клиент для выполнения HTTP- и HTTPS-запросов к удалённым серверам. В системе EME.WMS класс HTTP предоставляет методы для формирования и отправки запросов с поддержкой методов `PUT`, `GET`, `POST`, `PATCH` и `DELETE`. Поддерживает базовую HTTP-аутентификацию (Basic Auth) с передачей имени пользователя и пароля, а также автоматическое кодирование учётных данных в заголовок `Authorization`.

Класс `HTTP` в языке EME-L позволяет устанавливать произвольные HTTP-заголовки, отправлять запросы с телом и без тела, а также получать код состояния ответа, тело ответа, тип содержимого и полный текст сформированного запроса.

Использует библиотеки Boost.Asio и Boost.Beast для сетевого взаимодействия по протоколам HTTP/1.1 и HTTPS (SSL/TLS). Применяется для интеграции с внешними веб-сервисами, REST API и другими HTTP-совместимыми системами из скриптов EME-L.

Объект класса `HTTP` в языке EME-L создаётся через `Object()` и не привязан к контексту диалога или браузера — это самостоятельный сетевой клиент. Перед отправкой запроса необходимо вызвать метод `Open` для указания HTTP-метода, URL-адреса и (опционально) данных аутентификации. После выполнения запроса (`Send`) результат доступен через методы `GetStatus`, `GetResponseText`, `GetResponseContentType`.

## Создание объекта класса HTTP

Класс `HTTP` в языке EME-L имеет один конструктор — без аргументов.

```EME-L
'Создание объекта HTTP-клиента'
objHTTP = Object("HTTP");
```

Конструктор инициализирует внутреннее состояние объекта. Перед использованием в языке EME-L необходимо вызвать метод `Open` для указания метода, URL и параметров аутентификации.

## Управление таймаутами класса HTTP в языке EME-L

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetTimeouts(arg0, arg1, arg2, arg3)` | arg0: Integer (таймаут разрешения DNS), arg1: Integer (таймаут соединения), arg2: Integer (таймаут отправки), arg3: Integer (таймаут получения) | Empty | Устанавливает таймауты для различных этапов выполнения HTTP-запроса в языке EME-L (в миллисекундах). **Примечание:** в текущей реализации метод не выполняет действий — заглушка для будущего использования. |

## Инициализация запроса класса HTTP в языке EME-L

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Open(arg0, arg1)` | arg0: String (HTTP-метод), arg1: String (URL) | Empty | Инициализирует HTTP-запрос с указанием метода и URL-адреса. |
| `Open(arg0, arg1, arg2)` | + arg2: String (имя пользователя) | Empty | То же с именем пользователя для Basic Auth. |
| `Open(arg0, arg1, arg2, arg3)` | + arg3: String (пароль) | Empty | То же с именем пользователя и паролем для Basic Auth. |

Поддерживаемые HTTP-методы в языке EME-L: `"PUT"`, `"GET"`, `"POST"`, `"PATCH"`, `"DELETE"`. При указании неподдерживаемого метода генерируется ошибка.

URL должен включать схему (`http://` или `https://`), хост и путь. Порт определяется по схеме: 80 для HTTP, 443 для HTTPS; если порт указан в URL явно и отличается от стандартного, он передаётся в заголовке `Host`. Параметры запроса (query string) автоматически включаются в целевой путь.

Учётные данные для Basic Auth кодируются в Base64 и устанавливаются в заголовок `Authorization` в формате `Basic <base64(user:password)>`. В языке EME-L допускается указание имени пользователя без пароля.

## Заголовки запроса класса HTTP в языке EME-L

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetRequestHeader(arg0, arg1)` | arg0: String (имя заголовка), arg1: String (значение) | Empty | Устанавливает произвольный HTTP-заголовок для текущего запроса в языке EME-L. Например `"Content-Type"` или `"Accept"`. |

## Отправка запроса класса HTTP в языке EME-L

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Send()` | — | String | Отправляет запрос без тела. Устанавливает `Content-Length` в 0. При успехе возвращает пустую строку; при сетевой ошибке — текст исключения с описанием ошибки. |
| `Send(arg0)` | arg0: String (тело запроса) | String | Отправляет запрос с телом (например JSON или XML). Устанавливает `Content-Length` равным длине тела. При успехе возвращает пустую строку; при сетевой ошибке — текст исключения. |

Метод `Send` в языке EME-L synchronously выполняет запрос через TCP (для HTTP) или SSL/TLS (для HTTPS). Для HTTPS используется SSL-соединение через Boost.Asio, для HTTP — обычное TCP-соединение. При сетевой ошибке код ответа устанавливается в неопределённое значение, а текст исключения доступен как возвращаемое значение метода `Send`.

## Получение результата класса HTTP в языке EME-L

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetStatus()` | — | Integer | Числовой код состояния HTTP-ответа, полученного после последнего вызова `Send` в языке EME-L (например, 200 — успешно, 404 — не найдено, 500 — ошибка сервера). При сетевой ошибке возвращает неопределённое значение. |
| `GetResponseText()` | — | String | Тело HTTP-ответа в виде строки. Если запрос ещё не выполнялся или ответ пуст, возвращает пустую строку. |
| `GetResponseContentType()` | — | String | Значение заголовка `Content-Type` из HTTP-ответа (например, `"application/json"` или `"text/html"`). Если заголовок отсутствует — пустая строка. |
| `GetRequest()` | — | String | Полный текст сформированного HTTP-запроса в языке EME-L, включая метод, заголовки и целевой путь. |

## Примеры использования класса HTTP в языке EME-L

GET-запрос без аутентификации в языке EME-L:

```EME-L
'Получить данные от REST API'
objHTTP = Object("HTTP");
objHTTP.Open("GET", "https://api.example.com/v1/products");
objHTTP.Send();
Status = objHTTP.GetStatus();
If (Status = 200)
    Response = objHTTP.GetResponseText();
End If
```

POST-запрос с JSON-телом и заголовком в языке EME-L:

```EME-L
'Отправить JSON на сервер'
objHTTP = Object("HTTP");
objHTTP.Open("POST", "https://api.example.com/v1/orders");
objHTTP.SetRequestHeader("Content-Type", "application/json");
objHTTP.Send(JsonString);
Status = objHTTP.GetStatus();
If (Status != 200)
    ErrorText = objHTTP.GetResponseText();
End If
```

POST-запрос с Basic Auth в языке EME-L:

```EME-L
'Запрос к серверу с аутентификацией'
objHTTP = Object("HTTP");
objHTTP.Open("POST", "http://192.168.3.19/api/event", "Admin", "password");
objHTTP.SetRequestHeader("Content-Type", "application/json");
objHTTP.Send(BodyText);
Status = objHTTP.GetStatus();
Response = objHTTP.GetResponseText();
```
