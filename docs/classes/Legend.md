# Класс Legend

> Синонимы для поиска: CILegend Legend класс EME-L, графическая легенда EME-L, цветовая маркер легенда EME-L, SetCaption AddInfo RemoveAll SelectColor GetWidth GetHeight Draw FindColor FindByTag GetColor GetSize SaveLegend LoadLegend, IrrImage PropertyTree EME-L, отрисовка легенды EME-L, визуализация EME-L

Класс `Legend` в языке EME-L — класс для создания и отображения графических легенд с цветовыми маркерами и текстовыми подписями. В языке EME-L класс Legend представляет собой обертку над ядерным классом `CLegendElement` и предоставляет методы для формирования легенды из набора пунктов, каждый из которых характеризуется RGB-цветом, текстовой меткой и опциональным числовым тегом.

Основные возможности в языке EME-L: добавление пунктов с цветом, текстом и тегом; удаление всех пунктов; установка заголовка; выделение пункта по цвету (жирным шрифтом); вычисление размеров с учетом шрифта, отступов и маркеров; отрисовка на объекте IrrImage; поиск по цвету или тегу; получение цвета по индексу и количества пунктов; сохранение и загрузка в дерево свойств PropertyTree.

При отрисовке в системе EME.WMS легенда формирует прямоугольную область с заголовком (если задан), цветовыми квадратами-индикаторами и соответствующими текстовыми подписями. Размеры вычисляются автоматически на основе метрик шрифта Arial. Используется в графических отчетах и визуализациях для пояснения цветовой кодировки данных.

## Создание объекта класса Legend

Класс `Legend` в языке EME-L имеет один конструктор — без аргументов.

```EME-L
'Создает объект легенды с параметрами по умолчанию'
Legend = Object("Legend");
```

Параметры по умолчанию, устанавливаемые в конструкторе в языке EME-L:

| Параметр | Значение | Описание |
|----------|----------|----------|
| Высота текста | 8 | Размер шрифта в пунктах |
| Размер маркера | 20 x 10 | Ширина и высота цветового квадрата в пикселях |
| Отступы | 3 x 3 | Внутренние отступы легенды в пикселях |
| Ограничители | 3 x 3 | Расстояние между элементами в пикселях |
| Фоновый цвет | RGB(255, 255, 230) | Цвет фона с прозрачностью 225 |

## Настройка и управление пунктами

В языке EME-L методы класса Legend для управления содержимым легенды:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetCaption(заголовок)` | заголовок: String | Empty | Задает текстовый заголовок легенды. При изменении сбрасывается кэш размера изображения. |
| `AddInfo(цвет, текст)` | цвет: Integer, текст: String | Boolean | Добавляет пункт с цветом и текстом. Возвращает TRUE (-1), если пункт добавлен; FALSE (0), если пункт с таким сочетанием уже существует. Сбрасывает кэш размера. |
| `AddInfo(цвет, текст, тег)` | + тег: Integer | Boolean | Добавляет пункт с цветом, текстом и числовым тегом. Возвращает TRUE (-1) при успехе, FALSE (0) при дубликате. Сбрасывает кэш размера. |
| `RemoveAll()` | — | Empty | Удаляет все пункты легенды, включая заголовок и цветовые маркеры. Сбрасывает кэш размера. |
| `SelectColor(цвет)` | цвет: Integer | Empty | Выделяет пункт с указанным цветом. При отрисовке выделенный пункт отображается жирным шрифтом. Сбрасывает кэш размера. |

## Размеры и отрисовка

В языке EME-L методы класса Legend для вычисления размеров и отрисовки:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetWidth(изображение)` | изображение: IrrImage (по ссылке) | Integer | Ширина легенды в пикселях с учетом текущего состава пунктов, заголовка, шрифта, отступов и размеров маркеров. Размер вычисляется на основе метрик шрифта Arial и кэшируется до следующего изменения состава. |
| `GetHeight(изображение)` | изображение: IrrImage (по ссылке) | Integer | Высота легенды в пикселях. Вычисляется аналогично GetWidth. |
| `Draw(изображение, x, y)` | изображение: IrrImage (по ссылке), x: Integer, y: Integer | Empty | Выполняет отрисовку легенды на указанном изображении в заданных координатах. Рисует фоновый прямоугольник с рамкой, заголовок (если задан) жирным шрифтом, цветовые маркеры с текстовыми подписями. Выделенный пункт отображается жирным. |

## Поиск и доступ к данным

В языке EME-L методы класса Legend для поиска и чтения данных:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `FindColor(цвет)` | цвет: Integer | Integer | Ищет пункт по RGB-цвету. Возвращает индекс (>= 0) или -1, если не найден. |
| `FindByTag(тег)` | тег: Integer | Integer | Ищет пункт по числовому тегу. Возвращает индекс (>= 0) или -1, если не найден. |
| `GetColor(индекс)` | индекс: Integer | Integer | Возвращает RGB-цвет пункта по индексу (от 0 до GetSize()-1). |
| `GetSize()` | — | Integer | Количество пунктов в текущей легенде. |

## Сериализация

В языке EME-L методы класса Legend для сохранения и загрузки:

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SaveLegend(деревоСвойств)` | деревоСвойств: PropertyTree (по ссылке) | Empty | Сохраняет состав легенды (заголовок и список пунктов с цветом, текстом и тегом) в указанное дерево свойств. Каждый пункт сохраняется как дочерний узел с полями color, text и tag. |
| `LoadLegend(деревоСвойств)` | деревоСвойств: PropertyTree (по ссылке) | Empty | Загружает состав легенды из указанного дерева свойств. Перед загрузкой очищает текущий состав. Сбрасывает кэш размера. |

## Примеры

### Создание легенды и отрисовка на изображении

В языке EME-L пример формирования легенды и отрисовки на IrrImage:

```EME-L
'#DEV_INITIALS 29.06.2025 EME company'
'Формирование легенды цветовой кодировки ячеек'
Legend = Object("Legend");
Legend.SetCaption(tr("Тип ячеек:"));
Legend.AddInfo(is_RGB(255, 0, 0), tr("Красный"));
Legend.AddInfo(is_RGB(0, 255, 0), tr("Зеленый"));
Legend.AddInfo(is_RGB(0, 0, 155), tr("Синий"));
Legend.SelectColor(is_RGB(255, 0, 0));

'Вычисление размеров и отрисовка'
Image = Object("IrrImage", Width, Height);
Image.Fill(is_RGBA(255, 255, 255, 0));
Legend.Draw(Image, 0, 0);
```

### Сохранение легенды в дерево свойств

В языке EME-L пример сохранения состава легенды для передачи или хранения:

```EME-L
'#DEV_INITIALS 29.06.2025 EME company'
'Сохранение состава легенды для передачи или хранения'
Legend = Object("Legend");
Legend.SetCaption(tr("Цвет"));
Legend.AddInfo(is_RGB(255, 0, 0), tr("Красный"));
Legend.AddInfo(is_RGB(0, 255, 0), tr("Зеленый"));
Legend.AddInfo(is_RGB(0, 0, 255), tr("Синий"));

PropertyTree = Object("PropertyTree");
Legend.SaveLegend(PropertyTree);
JSON = PropertyTree.GetJSON();
```
