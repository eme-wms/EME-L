### IrrNode — базовый класс узла сцены движка Irrlicht (язык EME-L)

Класс `IrrNode` в языке EME-L — базовый класс узла сцены движка Irrlicht. Представляет узел трёхмерной сцены Irrlicht и предоставляет операции для управления иерархией, положением, поворотом, масштабом, видимостью, именем, пользовательскими свойствами, материалом и текстурой узла. Является родительским классом для специализированных узлов: IrrCamera, IrrLight, IrrMesh, IrrAnimatedMesh, IrrOctree, IrrBillboard, IrrDummy, IrrSkyBox и IrrCube.

Класс `IrrNode` в системе EME.WMS — точка построения иерархии трёхмерной сцены. Через его методы скрипт EME-L добавляет дочерние узлы специализированных типов (кубы, сетки, камеры, источники света, билборды, пустышки, скайбоксы), подключает аниматор движения по сплайну, сохраняет и загружает параметры узла в дерево свойств PropertyTree, получает ограничивающую рамку и точку пересечения с лучом.

### Создание объекта IrrNode

В блоке `BEGIN_IMPLEMENT(CIIrrNode)` в C++ исходнике нет зарегистрированных конструкторов с аргументами (`IMPLEMENT_CTOR` отсутствует), поэтому объект `IrrNode` не создаётся через `Object("IrrNode", ...)` напрямую. Канонический способ получить узел сцены в языке EME-L — обратиться к корневому узлу сцены через `GetRootNode()` устройства IrrDevice либо создать дочерний узел одного из специализированных типов методом `Add*` (AddCube, AddMesh, AddCamera и др.).

```EME-L
'Создать устройство Irrlicht и получить корневой узел сцены'
objDevice = Object("IrrDevice");
objRoot   = objDevice.GetRootNode();

'Задать имя корневого узла для поиска по имени'
objRoot.SetName("Root");
```

При создании узла в системе EME.WMS применяются значения по умолчанию, заданные в конструкторе `CIIrrNode::Ctor()`: видимость — TRUE, освещение — TRUE, нормализация нормалей — FALSE, отсечение задних граней — TRUE, тип материала — solid, реакция на мышь (clickable) — FALSE, масштаб текстуры по U и V — 0 (не применяется).

### Методы иерархии и поиска IrrNode

Управление родительскими и дочерними узлами, поиск узлов по имени, проверка валидности и удаление узла из сцены в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetParentNode(arg0)` | arg0: IrrNode (ссылка) | Empty | Устанавливает родительский узел сцены. Положение и масштаб текущего узла пересчитываются с учётом нового родителя. |
| `FindNodeByName(arg0)` | arg0: String | IrrNode | Ищет в поддереве текущего узла дочерний узел Irrlicht по имени и возвращает обёртку IrrNode. |
| `AddNode(arg0)` | arg0: PropertyTree (ссылка) | Empty | Создаёт дочерний узел указанного типа из дерева свойств и загружает в него параметры. Поддерживаемые типы: Cube, Mesh, Billboard, Dummy, SkyBox, Light, Camera. При неизвестном типе генерируется ошибка. |
| `Remove()` | — | Empty | Удаляет текущий узел вместе со всеми дочерними узлами, удаляет связанные спрайты, исключает узел из реестра имён устройства и освобождает память. |
| `IsValidNode()` | — | Boolean | TRUE — объект связан с внутренним узлом сцены Irrlicht; FALSE — узел не инициализирован. |

### Методы положения, поворота и масштаба IrrNode

Положение и масштаб узла в языке EME-L задаются в локальной системе координат относительно родительского узла и умножаются на глобальный масштаб устройства IrrDevice перед установкой в Irrlicht. Поворот задаётся в градусах и не зависит от глобального масштаба.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetPosition(arg0, arg1, arg2)` | arg0: Real (X), arg1: Real (Y), arg2: Real (Z) | Empty | Задаёт локальное положение узла. Координаты умножаются на глобальный масштаб. |
| `GetPosition(arg0)` | arg0: Array (по ссылке, 3 элемента) | Empty | Возвращает локальное положение узла. Координаты X, Y, Z записываются в элементы массива с индексами 0, 1, 2. |
| `SetRotation(arg0, arg1, arg2)` | arg0: Real (X°), arg1: Real (Y°), arg2: Real (Z°) | Empty | Задаёт углы поворота узла в градусах по осям X, Y, Z. |
| `GetRotation(arg0)` | arg0: Array (по ссылке, 3 элемента) | Empty | Возвращает углы поворота узла в градусах. Записывает X, Y, Z в элементы массива 0, 1, 2. |
| `SetScale(arg0, arg1, arg2)` | arg0: Real (X), arg1: Real (Y), arg2: Real (Z) | Empty | Задаёт локальный масштаб узла. Значения умножаются на глобальный масштаб. |

Методы `GetPosition` и `GetRotation` в языке EME-L не возвращают значение напрямую — результат записывается в переданный массив Array минимум из трёх элементов. После вызова координаты читаются из элементов массива с индексами 0 (X), 1 (Y), 2 (Z). Возвращаемое значение метода — пустое (Empty).

### Примеры использования IrrNode в языке EME-L

Построение сцены с корневым узлом, кубом и камерой:

```EME-L
'Создать устройство Irrlicht и получить корневой узел'
objDevice = Object("IrrDevice");
objRoot   = objDevice.GetRootNode();

'Добавить дочерний куб с длиной ребра 10.0'
objCube = objRoot.AddCube(10.0);

'Назначить текстуру из файла'
objCube.SetMaterialTexture("D:\\Data\\box.png");

'Добавить камеру и сделать её активной'
objCam = objRoot.AddCamera();
```

Поиск дочернего узла по имени и чтение его положения:

```EME-L
'Найти узел по имени в поддереве корневого узла IrrNode'
objFound = objRoot.FindNodeByName("Box01");

'Массив для приёма координат X, Y, Z'
arrPos = is_array(3);

If (objFound.IsValidNode())
    objFound.GetPosition(arrPos);
    PosX = arrPos[0];
    PosY = arrPos[1];
    PosZ = arrPos[2];
End If
```
