### Random — класс генератора псевдослучайных значений в языке EME-L

Класс `Random` (C++ `CIRandom`) в языке EME-L — генератор псевдослучайных значений различных типов. Предназначен для создания случайных целых и вещественных чисел, дат, а также строковых данных заданного семантического типа: слов, буквенно-цифровых комбинаций, числовых последовательностей, адресов, e-mail, интернет-адресов, имен, фамилий, отчеств, полных ФИО, названий компаний и номеров телефонов.

В системе EME.WMS класс `Random` используется в прикладных скриптах EME-L для генерации тестовых и прикладных данных, заполнения полей диалогов и создания уникальных идентификаторов. Объект класса `Random` создаётся через `Object()` и не привязан к контексту диалога или браузера — это самостоятельный класс.

Класс `Random` доступен только в версиях ядра EME.WMS 23.17 и выше. Регистрация класса в C++ защищена условием компиляции `LAST_VERSION >= MAKE_VERSION(23, 17)`.

### Создание объекта класса Random в языке EME-L

Класс `Random` имеет один конструктор — без аргументов.

```EME-L
'Создаёт генератор случайных значений с начальной инициализацией'
random = Object("Random");
```

Конструктор без аргументов в языке EME-L создаёт и инициализирует внутренний генератор псевдослучайных чисел, устанавливая количество генерируемых слов в строке равным 1.

### Генерация строк универсальным методом GetString в EME-L

Метод `GetString` класса `Random` в языке EME-L — универсальный генератор случайных строк. Первые два параметра поддерживают двойную семантику: целое число интерпретируется как длина строки, а строка — как тип формируемых данных. Если длина не задана ни одним из целочисленных параметров, используется диапазон 4–25 символов.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetString()` | — | String | Случайное слово на русском языке, длина 4–25 символов. |
| `GetString(arg0)` | arg0: Integer | String | Слово фиксированной длины на русском языке. |
| `GetString(arg0)` | arg0: String | String | Строка указанного типа, длина по умолчанию 4–25 символов. |
| `GetString(arg0, arg1)` | arg0: Integer, arg1: Integer | String | Строка заданной длины (min–max) на русском языке, тип «Слова». |
| `GetString(arg0, arg1, arg2)` | arg0: Integer, arg1: Integer, arg2: String | String | Строка заданной длины и типа на русском языке. |
| `GetString(arg0, arg1, arg2, arg3)` | + arg3: String (язык) | String | Строка заданной длины, типа и языка. |

Допустимые значения типа строки в методе `GetString` класса `Random`:

| Значение | Что генерируется |
|----------|------------------|
| `"Слова"` | Последовательность букв (по умолчанию) |
| `"Буквы+Цифры"` | Буквенно-цифровая комбинация |
| `"Буквы"` | Только буквы |
| `"Цифры"` | Только цифры |
| `"E-mail"` | Адрес электронной почты |
| `"WWW"` | Интернет-адрес |
| `"Адрес"` | Почтовый адрес |
| `"ФИО"` | Фамилия, имя и отчество |
| `"Фамилия"` | Фамилия |
| `"Имя"` | Имя |
| `"Отчество"` | Отчество |
| `"Фирма"` | Название компании |
| `"Телефон"` | Номер телефона |
| `"Double '%.3f'"` | Вещественное число с форматированием |
| `"Int '%d'"` | Целое число с форматированием |

Допустимые значения языка строковых данных в методе `GetString` класса `Random`:

| Значение | Язык |
|----------|------|
| `"rus"` | Русский (по умолчанию) |
| `"eng"` | Английский |

Язык строки в методе `GetString` применяется только на время данного вызова, после чего внутренний генератор возвращается к русскому языку.

### Генерация чисел и дат в классе Random EME-L

Методы `GetDouble`, `GetInt` и `GetDate` класса `Random` в языке EME-L формируют случайные числовые значения и даты. Если минимальная граница превышает максимальную, числовые методы возвращаются к диапазонам по умолчанию, а метод `GetDate` возвращает минимальную дату.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetDouble()` | — | Real | Случайное вещественное число в диапазоне [0.0, 1.0]. |
| `GetDouble(arg0)` | arg0: Real | Real | Случайное число в диапазоне [0.0, arg0]. |
| `GetDouble(arg0, arg1)` | arg0: Real (min), arg1: Real (max) | Real | Случайное число в диапазоне [min, max]. Если min > max, используется [0.0, 1.0]. |
| `GetInt()` | — | Integer | Случайное целое число в диапазоне [0, 255]. |
| `GetInt(arg0)` | arg0: Integer | Integer | Случайное число в диапазоне [0, arg0]. |
| `GetInt(arg0, arg1)` | arg0: Integer (min), arg1: Integer (max) | Integer | Случайное число в диапазоне [min, max]. Если min > max, используется [0, 255]. |
| `GetDate(arg0, arg1)` | arg0: DateTime (min), arg1: DateTime (max) | DateTime | Случайная дата в заданном диапазоне. Если min > max, возвращается min. |

### Примеры использования класса Random в языке EME-L

Генерация штрих-кода PDF417 для алкогольной марки в системе EME.WMS (из `TerminalTester.txt`):

```EME-L
'Создаём генератор один раз в конструкторе класса'
Random = Object("Random");

'Цифровая часть кода фиксированной длины 20 символов'
numericPart = Random.GetNumericString(20);

'Буквенно-цифровая часть фиксированной длины 129 на английском языке'
alphaPart = is_upper(Random.GetString(129, 129, "Буквы+Цифры", "eng"));

'Сборка кода марки и запись в PDF417'
StampsCode = "1" + numericPart + alphaPart;
r_Stamps.PutPDF417(StampsCode);
```

Подбор уникального случайного кода для карточки сотрудника в системе EME.WMS (из `Staff.txt`):

```EME-L
randoml = Object("Random");
dsStaff = Object("dsDB", "Staff");
value = 0;
While (true)
    'Случайный код в диапазоне 1000–99999'
    value = randoml.GetInt(1000, 99999);
    dsStaff.SetSkipMode();
    dsStaff.GetCodeFld().MustBeEQ(value);
    dsStaff.MustBeValid();
    dsStaff.SetFirstLine();
    If (~dsStaff.IsValidLine())
        Break;
    End If
End Loop
```

Генерация разнообразных строковых и числовых данных в EME-L:

```EME-L
random = Object("Random");

'Случайное ФИО на русском языке'
fullName = random.GetFullnameString();

'Случайный e-mail длиной 12 символов'
email = random.GetEmailString(12);

'Случайное вещественное число от 10.0 до 100.0'
price = random.GetDouble(10.0, 100.0);

'Случайная дата в диапазоне 2024–2025'
minDate = is_date(1, 1, 2024);
maxDate = is_date(31, 12, 2025);
randDate = random.GetDate(minDate, maxDate);
```
