### IrrCamera — класс камеры движка Irrlicht (язык EME-L)

Класс `IrrCamera` в языке EME-L — это класс камеры движка Irrlicht. Представляет узел сцены Irrlicht, отвечающий за формирование вида трёхмерной сцены. Позволяет задавать точку зрения (target), угол поля зрения, ближнюю и дальнюю плоскости отсечения, соотношение сторон, перспективную и ортографическую проекции, проекционную матрицу 4×4, а также ограничивать перемещение камеры куполом и подключать аниматор управления мышью и клавиатурой.

Класс `IrrCamera` в системе EME.WMS наследует все методы базового класса `IrrNode` — управление положением, поворотом, масштабом, видимостью, именем, материалом и текстурой узла. Используется в EME-L-скриптах для настройки камеры при визуализации трёхмерных сцен.

### Создание объекта IrrCamera

В блоке регистрации `BEGIN_IMPLEMENT(CIIrrCamera)` в C++ исходнике нет зарегистрированных конструкторов с аргументами (`IMPLEMENT_CTOR` отсутствует), поэтому объект `IrrCamera` не создаётся через `Object("IrrCamera", ...)` напрямую. Канонический способ получить камеру в языке EME-L — вызвать метод `AddCamera()` у родительского узла `IrrNode`. Этот метод создаёт дочерний узел-камеру, делает её активной камерой устройства `IrrDevice` и возвращает ссылку на новый объект `IrrCamera`.

```EME-L
'Создать устройство Irrlicht и корневой узел сцены'
objDevice = Object("IrrDevice");
objRoot   = objDevice.GetRootNode();

'Добавить дочернюю камеру к корневому узлу'
objCam = objRoot.AddCamera();
```

При инициализации камеры через `LoadNode` (загрузка из дерева свойств PropertyTree) в системе EME.WMS применяются значения по умолчанию, если соответствующие ключи отсутствуют в дереве. Эти же значения служат ориентиром для типичной конфигурации камеры в языке EME-L:

| Параметр камеры | Ключ в PropertyTree | Значение по умолчанию |
|-----------------|---------------------|-----------------------|
| Ближняя плоскость отсечения | `NearValue` | `1.0` |
| Дальняя плоскость отсечения | `FarValue` | `600.0` |
| Угол поля зрения (FOV) | `FOV` | `0.5` |
| Радиус купола камеры | `CameraDomeRadius` | `100.0` |
| Высота пола купола | `CameraDomeFloor` | `1.5` |

### Методы точки зрения (target) камеры IrrCamera

Методы управления точкой зрения камеры в языке EME-L. Координаты точки зрения (target) — мировые координаты, в которых камера «смотрит». Передаваемые значения координат умножаются на глобальный масштаб узла (global scale) перед установкой в Irrlicht.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetCameraTarget(arg0, arg1, arg2)` | arg0: Real (X), arg1: Real (Y), arg2: Real (Z) | Empty | Задаёт точку зрения камеры в мировых координатах. Координаты умножаются на глобальный масштаб узла перед установкой. |
| `GetCameraTarget(arg0)` | arg0: Array (по ссылке, 3 элемента) | Empty | Возвращает текущую точку зрения камеры в мировых координатах с учётом глобального масштаба. Координаты X, Y, Z записываются в элементы массива с индексами 0, 1, 2. |

Метод `GetCameraTarget` в языке EME-L ожидает, что в `arg0` передан объект `Array` минимум из трёх элементов. Результат записывается по индексам: 0 → X, 1 → Y, 2 → Z. Возвращаемое значение метода — пустое (`Empty`); координаты читаются из массива после вызова.

### Методы плоскостей отсечения и поля зрения IrrCamera

Плоскости отсечения (near/far clipping planes) и угол поля зрения (FOV) определяют видимую область камеры в системе EME.WMS. Значения расстояний до плоскостей отсечения в геттерах делятся на глобальный масштаб узла, а в сеттерах — умножаются на него перед установкой.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetNearValue()` | — | Real | Расстояние от камеры до ближней плоскости отсечения с учётом глобального масштаба узла. |
| `SetNearValue(arg0)` | arg0: Real | Empty | Задаёт расстояние до ближней плоскости отсечения. Значение умножается на глобальный масштаб узла перед установкой. |
| `GetFarValue()` | — | Real | Расстояние от камеры до дальней плоскости отсечения с учётом глобального масштаба узла. |
| `SetFarValue(arg0)` | arg0: Real | Empty | Задаёт расстояние до дальней плоскости отсечения. Значение умножается на глобальный масштаб узла перед установкой. |
| `GetFOV()` | — | Real | Текущий угол поля зрения камеры (field of view) в радианах. |
| `SetFOV(arg0)` | arg0: Real | Empty | Задаёт угол поля зрения камеры в радианах. |

Угол поля зрения в языке EME-L задаётся в радианах, не в градусах. Для перевода градусов в радианы используйте формулу `радианы = градусы * π / 180` (например, 60° ≈ 1.047 рад).

### Методы соотношения сторон IrrCamera

Соотношение сторон (aspect ratio) отсекающей рамки камеры в языке EME-L. Определяет пропорцию ширины кадра к высоте и влияет на дисторсию перспективной проекции.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetAspectRatio()` | — | Real | Текущее соотношение сторон (aspect ratio) отсекающей рамки камеры. |
| `SetAspectRatio(arg0)` | arg0: Real | Empty | Задаёт соотношение сторон (ширина к высоте) отсекающей рамки камеры. |

### Метод купола IrrCamera (ограничение перемещения камеры)

Купол (dome) в системе EME.WMS ограничивает перемещение камеры сферической зоной. Радиус купола и высота пола (фундамента) умножаются на глобальный масштаб узла перед установкой в движок Irrlicht.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetCameraDome(arg0, arg1)` | arg0: Real (радиус), arg1: Real (высота пола) | Empty | Устанавливает купол, ограничивающий перемещение камеры. Радиус и высота пола умножаются на глобальный масштаб узла перед установкой. |

### Методы проекции IrrCamera

Камера в языке EME-L поддерживает перспективную и ортографическую проекции, а также прямой доступ к проекционной матрице 4×4 (16 элементов). Проекционная матрица передаётся через объект `Array`.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetPerspectiveProjection(arg0, arg1, arg2, arg3)` | arg0: Real (FOV, рад), arg1: Real (aspect), arg2: Real (near), arg3: Real (far) | Empty | Устанавливает перспективную проекцию камеры. |
| `SetOrthogonalProjection(arg0, arg1, arg2, arg3)` | arg0: Real (ширина рамки), arg1: Real (высота рамки), arg2: Real (near), arg3: Real (far) | Empty | Устанавливает ортографическую проекцию камеры. |
| `GetProjectionMatrix(arg0)` | arg0: Array (по ссылке, 16 элементов) | Empty | Записывает текущую проекционную матрицу 4×4 (16 элементов) в переданный массив. |
| `SetProjectionMatrix(arg0, arg1)` | arg0: Array (по ссылке, 16 элементов), arg1: Boolean | Empty | Задаёт проекционную матрицу 4×4 и тип проекции: `arg1 = TRUE` — ортографическая, `FALSE` — перспективная. |

Методы `GetProjectionMatrix` и `SetProjectionMatrix` в языке EME-L работают с плоской записью матрицы 4×4: 16 вещественных чисел в массиве с индексами от 0 до 15. Порядок элементов соответствует строке за строкой матрицы Irrlicht `core::matrix4`.

### Методы выравнивания и аниматора IrrCamera

Признак выравнивания (align) по высоте позиции и цели камеры в языке EME-L синхронизирует высоту позиции камеры с высотой её цели. Аниматор (animator) подключает интерактивное управление камерой мышью и клавиатурой через Irrlicht.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AlignTargetAndPosition(arg0)` | arg0: Boolean | Empty | Включает (`TRUE`) или отключает (`FALSE`) выравнивание по высоте позиции и цели камеры. |
| `GetTargetAndPositionAligning()` | — | Boolean | Возвращает текущее состояние признака выравнивания: `TRUE` — включено, `FALSE` — отключено. |
| `AddCameraAnimator()` | — | Empty | Добавляет к камере аниматор, управляющий её положением и ориентацией с помощью мыши и клавиатуры. Если аниматор уже был подключён ранее, он предварительно удаляется. Аниматор связывает цель и вращение камеры (`bindTargetAndRotation`) и использует курсор устройства Irrlicht. |

Метод `AddCameraAnimator` идемпотентен в системе EME.WMS: повторный вызов не накапливает аниматоры, а заменяет существующий. Это предотвращает множественные обработчики ввода на одной камере.

### Примеры использования IrrCamera в языке EME-L

Настройка точки зрения и поля зрения камеры:

```EME-L
'Получить активную камеру с корневого узла IrrNode'
objCam = objRoot.AddCamera();

'Направить взгляд на точку (10, 0, 5) в мировых координатах'
objCam.SetCameraTarget(10.0, 0.0, 5.0);

'Задать угол поля зрения ~60 градусов (в радианах)'
objCam.SetFOV(1.047);
```

Чтение текущей точки зрения камеры в массив:

```EME-L
'Массив из трёх элементов для приёма координат X, Y, Z'
arrTarget = is_array(3);

'Заполнить массив координатами target'
objCam.GetCameraTarget(arrTarget);

TargetX = arrTarget[0];
TargetY = arrTarget[1];
TargetZ = arrTarget[2];
```

Переключение на ортографическую проекцию и подключение управления мышью:

```EME-L
'Ортографическая рамка 800x600, near=1, far=1000'
objCam.SetOrthogonalProjection(800.0, 600.0, 1.0, 1000.0);

'Разрешить управление камерой мышью и клавиатурой'
objCam.AddCameraAnimator();
```
