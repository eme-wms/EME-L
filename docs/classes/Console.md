# Класс Console

> Синонимы для поиска: CIConsole Console класс EME-L, консоль вывода EME-L, окно консоли EME-L, PutText Show Hide SetMode SetCaption SetRect UpdateText, вывод сообщений EME-L, закладка консоли EME-L, SetConsole EME-L, CEMEConsole EME-L, интерфейсный поток консоль EME-L

Класс `Console` в языке EME-L управляет окном консоли вывода сообщений. Класс Console в языке EME-L предоставляет возможность создания, отображения и скрытия окна консоли, вывода и обновления текстовых строк на заданных закладках, а также настройки заголовка, размеров, положения окна и режима добавления перевода строки.

Консоль в языке EME-L доступна только в интерфейсном потоке выполнения. В неинтерфейсном (фоновом) потоке методы `Show`, `Hide`, `PutText`, `UpdateText` не выполняют никаких действий и возвращают пустое значение. При уничтожении объекта класса Console консоль автоматически закрывается, если она была открыта.

Объект класса `Console` в языке EME-L создаётся через `Object()` и обычно передаётся в движок методом `SetConsole` родительского объекта (например, `i_ERPEngine.SetConsole(m_Console)`), чтобы сообщения системы дублировались в окно консоли.

## Создание объекта класса Console

```EME-L
'Консоль вывода с размерами и положением по умолчанию'
m_Console = Object("Console");

'Консоль с явными размерами и положением (объект Rect)'
m_Console = Object("Console", r_Rect);

'Консоль с размерами и заголовком окна'
m_Console = Object("Console", r_Rect, "Журнал импорта");
```

Конструктор без аргументов класса Console в языке EME-L создаёт консоль с положением (200, 200), размерами 400×300 и заголовком «Консоль». Режим добавления перевода строки включён по умолчанию во всех перегрузках конструктора.

## Управление окном консоли

Методы управления окном класса Console в языке EME-L — `Show`, `Hide`, `SetRect`, `SetCaption`, `SetMode`.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Show()` | — | Integer | Открывает окно консоли. Возвращает предыдущее состояние видимости. В неинтерфейсном потоке ничего не делает. |
| `Show(tab)` | tab: Integer | Integer | Открывает окно класса Console в языке EME-L и переключает на указанную закладку. |
| `Hide()` | — | Empty | Закрывает окно консоли и освобождает ресурсы. В неинтерфейсном потоке ничего не делает. |
| `SetRect(rect)` | rect: Rect (по ссылке) | Empty | Задаёт положение и размеры окна на основе объекта `Rect`. |
| `SetCaption(caption)` | caption: String | Empty | Задаёт заголовок окна консоли. |
| `SetMode(mode)` | mode: Boolean | Boolean | Задаёт режим автоматического добавления перевода строки. TRUE — добавлять, FALSE — не добавлять. Возвращает предыдущее значение режима. |

## Вывод текста в консоль

Методы `PutText` класса Console в языке EME-L добавляют строку в конец вывода; методы `UpdateText` заменяют последнюю выведённую строку новой. Если включён режим добавления перевода строки (`SetMode(TRUE)`), к тексту автоматически добавляется символ перевода строки.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `PutText(text)` | text: String | Empty | Выводит строку на закладку 1. |
| `PutText(text, tab)` | text: String, tab: Integer | Empty | Выводит строку на указанную закладку класса Console в языке EME-L. |
| `UpdateText(text)` | text: String | Empty | Заменяет последнюю строку на текущей закладке (по умолчанию 1). |
| `UpdateText(text, tab)` | text: String, tab: Integer | Empty | Заменяет последнюю строку на указанной закладке. |

## Примеры использования класса Console

Создание консоли, привязка к движку и вывод хода выполнения процесса в языке EME-L:

```EME-L
'Создать консоль и передать в движок импорта для дублирования сообщений'
m_Console = Object("Console");
i_ERPEngine.SetConsole(m_Console);

m_Console.Show();
m_Console.PutText(tr("Соединяем с внешней базой данных..."));

Error = i_ERPEngine.Launch(i_ERPEngine.GetConnStr());
If (Error == "")
    m_Console.PutText(tr("Начинаем экспорт сообщений..."));
End If
```

Вывод на разные закладки и обновление последней строки без перевода строки в классе Console языка EME-L:

```EME-L
'Переключиться на закладку 2 и вывести прогресс'
m_Console.Show(2);
m_Console.PutText(tr("Обработка..."), 2);

'Заменить последнюю строку на закладке 2 без добавления новой'
m_Console.SetMode(FALSE);
m_Console.UpdateText(tr("Готово."), 2);
```

Вывод имени обрабатываемого файла в цикле (паттерн из класса `AmarCordDoctor` в системе EME.WMS):

```EME-L
Console = Object("Console");
Console.Show();
While (RetValue)
    RetValue = FileFind.FindNextFile();
    FileName = FileFind.GetFileName();
    Console.PutText(FileName);
    ProcessFile(ProjectType, SourceDir, FileName);
End While
```
