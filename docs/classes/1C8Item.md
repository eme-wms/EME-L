# Класс 1C8Item

> Синонимы для поиска: CI1C8Item 1C8Item класс EME-L, 1С Предприятие 8.0 EME-L, интеграция 1С 8.0 EME-L, объекты 1С 8.0 EME-L, справочник 1С EME-L, документ 1С EME-L, табличная часть 1С EME-L, Select SelectEx Next Attach GetAttrib SetAttrib GetAttribAsRef SetAttribAsRef GetRef GetThis IsEmpty SetWriteMode FindByCode FindByDescription FindByAttrib CreateItem CreateDocument Write SetNewNumber Posting FindDocumentByNumber Count Get IndexOf AddRow DeleteAllRows, 1C CreateObject 1C8Item, 1C8Item Attach ссылка, выборка справочника 1С 8.0 EME-L, проведение документа 1С EME-L

Класс `1C8Item` в языке EME-L — это интерфейс для чтения, записи и манипулирования объектами данных системы 1С:Предприятие версии 8.0. Класс 1C8Item в языке EME-L предоставляет методы для навигации по выборкам справочников и документов, получения и установки атрибутов объектов, поиска элементов по коду, наименованию и произвольным атрибутам, создания новых элементов справочников и документов, а также работы с табличными частями документов.

Объекты класса `1C8Item` в системе EME.WMS обычно получаются из методов класса `1C` (например, `1C.CreateObject`) или из других объектов `1C8Item` (`GetAttribAsRef`, `CreateItem`, `CreateDocument`, `Get`, `AddRow`), а не создаются напрямую через `Object()`. Вторая форма конструктора с параметром-ссылкой используется для явного связывания с уже созданным объектом 1С.

Класс 1C8Item предназначен для интеграции прикладной логики EME с учетными данными, хранящимися в базе 1С:Предприятие 8.0. Для работы с 1С версии 7.7 в языке EME-L используйте класс `1CItem`.

## Создание объекта класса 1C8Item

```EME-L
'Типичный путь — объект возвращает класс 1C'
objDoc = obj1C.CreateObject("Документы.ПриходныйОрдерНаТовары");
'objDoc — это объект 1C8Item, связанный с документом 1С'

'Прямое создание через Object() — используется редко'
objItem = Object("1C8Item");
objItem = Object("1C8Item", objRef);
```

Конструктор без аргументов класса `1C8Item` в языке EME-L создаёт несвязанный объект-обёртку — привязка к объекту 1С выполняется методом `Attach`. Конструктор с одним аргументом сразу связывается с переданной ссылкой, полученной методом `CreateObject` класса `1C`.

## Навигация по выборке

Методы `Select` и `SelectEx` класса 1C8Item в языке EME-L переводят объект в режим выборки, после чего перебор выполняется методом `Next`. Выборка применима к справочникам и документам.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Select()` | — | Boolean | Активирует режим выборки. TRUE — активирован, FALSE — неудача. |
| `SelectEx(arg0)` | arg0: String (начальная_дата) | Boolean | Выборка документов с указанной начальной даты. |
| `SelectEx(arg0, arg1)` | arg0: String (начальная_дата), arg1: String (конечная_дата) | Boolean | Выборка документов за период. |
| `SelectEx(arg0, arg1, arg2)` | + arg2: String (упорядочивание) | Boolean | То же с упорядочиванием по указанному полю. |
| `Next()` | — | Boolean | Переход к следующему объекту. Перед вызовом объект должен быть переведён в режим выборки. TRUE — переход выполнен, FALSE — достигнут конец выборки или объект не в режиме выборки. |

## Примеры

Создание документа 1С и заполнение табличной части — методы CreateObject, SetAttribAsRef, AddRow, Write, Posting класса 1C8Item в языке EME-L:

```EME-L
'Получить объект документа из класса 1C'
objDoc = obj1C.CreateObject("Документы.ПриходныйОрдерНаТовары");
objDoc.SetNewNumber("ПРТ");
objDoc.SetAttrib("ДатаДокумента", is_today());

'Найти склад и установить ссылочный атрибут'
objWh = obj1C.CreateObject("Справочники.Склады");
objWh.FindByDescription("Центральный склад");
objDoc.SetAttribAsRef("Склад", objWh.GetThis());

'Добавить строки табличной части'
objLines = objDoc.Get(0);
For (i = 0; i < 3; i = i + 1)
    objRow = objLines.AddRow();
    objRow.SetAttrib("Количество", 10.0 + i);
End For

objDoc.Write();
objDoc.Posting("Проведение");
```

Перебор выборки справочника 1С методами Select и Next класса 1C8Item в языке EME-L:

```EME-L
objGoods = obj1C.CreateObject("Справочники.Номенклатура");
If (objGoods.Select())
    While (objGoods.Next())
        Code = objGoods.GetAttrib("Код", "");
        Name = objGoods.GetAttrib("Наименование", "");
        'Code и Name — значения текущего элемента справочника 1С'
    End While
End If
```

Чтение ссылочного атрибута и переход в режим записи — методы GetAttribAsRef, SetWriteMode класса 1C8Item в языке EME-L:

```EME-L
'GetAttribAsRef возвращает новый объект 1C8Item'
objOwner = objItem.GetAttribAsRef("Владелец");
If (~objOwner.IsEmpty())
    objOwner.SetWriteMode();
    objOwner.SetAttrib("Описание", "Обновлено из EME");
    objOwner.Write();
End If
```
