# Класс 1C

> Синонимы для поиска: CI1C 1C класс EME-L, интеграция 1С EME-L, COM-соединение 1С EME-L, Connect Disconnect Is1CInstall IsConnect EvalExpr Execute CreateObject SetTimeout GetPostingWriteMode GetCurrentCurrencyRate GetCurrentCurrencyRatio, 1С Предприятие EME-L, 1CItem 1C8Item CreateObject, обмен данными с 1С EME-L

Класс `1C` в языке EME-L — это интерфейс для обмена данными с системой 1С:Предприятие версий 7.7 и 8.0. Класс 1C в языке EME-L предоставляет методы для установки и разрыва COM-соединения с 1С, выполнения выражений и операторов на встроенном языке 1С, создания объектов агрегатных типов данных, получения системных перечислений и курсов валют. Используется для интеграции системы EME с учетными системами на базе 1С:Предприятие.

Объект класса `1C` в языке EME-L создаётся через `Object()` и не имеет параметров конструктора — это самостоятельный класс-обёртка над COM-соединением с 1С:Предприятие. До вызова `Connect` объект не подключён к 1С и методы обмена данными возвращают пустые значения.

Классы `1CItem` и `1C8Item` в языке EME-L работают с объектами данных 1С версий 7.7 и 8.0 соответственно и принимают в конструкторе ссылку, полученную методом `CreateObject` класса `1C`.

## Создание объекта класса 1C

```EME-L
obj1C = Object("1C");
```

Конструктор без аргументов создаёт объект интерфейса CI1C в языке EME-L. До вызова `Connect` объект не подключён к 1С.

## Проверка установки и соединение

Методы управления жизненным циклом COM-соединения класса 1C в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Is1CInstall(arg0)` | arg0: Integer (версия 1С) | Boolean | TRUE — указанная версия 1С установлена, FALSE — не установлена. |
| `Connect(arg0, arg1)` | arg0: Integer (версия 1С), arg1: String (строка запуска) | Boolean | TRUE — соединение установлено, FALSE — ошибка подключения. |
| `Disconnect()` | — | Empty | Отсоединяется от 1С и закрывает COM-соединение. |
| `IsConnect()` | — | Boolean | TRUE — активное соединение установлено, FALSE — соединение отсутствует. |
| `SetTimeout(arg0)` | arg0: Integer (мс) | Empty | Таймаут операций с 1С. По умолчанию 15000 мс (15 секунд). |

Версия 1С передаётся целым числом: `0` — 1С:Предприятие 7.7, `8` — 1С:Предприятие 8.0. Строка запуска класса 1C в языке EME-L следует формату ключей командной строки 1С: `/d"<путь к базе>" /n<имя пользователя> /p<пароль>`.

## Выполнение кода 1С

Методы позволяют вычислять выражения и выполнять операторы на встроенном языке 1С:Предприятие.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `EvalExpr(arg0)` | arg0: String (выражение) | зависит от выражения | Вычисляет выражение на языке 1С. Применимо только для версии 7.7. При отсутствии соединения возвращает пустое значение. |
| `Execute(arg0)` | arg0: String (операторы) | зависит от операторов | Выполняет последовательность операторов на языке 1С. При отсутствии соединения возвращает пустое значение. |
| `CreateObject(arg0)` | arg0: String (имя типа 1С) | ссылка на объект 1С или Empty | Создаёт объект агрегатного типа данных 1С (например, `"Справочники.Склады"`). Возвращает `Empty`, если соединение не установлено или объект не создан. |

Ссылку, возвращённую `CreateObject` класса 1C в языке EME-L, принято передавать в конструктор классов `1CItem` (для 7.7) или `1C8Item` (для 8.0): `Object("1CItem", obj1C.CreateObject("Справочники.Склады"))`.

## Системные перечисления и курсы валют

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetPostingWriteMode()` | — | ссылка на перечисление | Значение `DocumentWriteMode.Posting`, необходимое для проведения документа в 1С версии 8.0. |
| `GetCurrentCurrencyRate(arg0, arg1)` | arg0: исходная валюта (String или ссылка), arg1: целевая валюта (String или ссылка) | Real | Текущий курс валюты. |
| `GetCurrentCurrencyRatio(arg0, arg1)` | arg0: исходная валюта (String или ссылка), arg1: целевая валюта (String или ссылка) | Real | Текущая кратность курса валюты. |

## Примеры

Подключение к базе 1С 7.7:

```EME-L
'Создать интерфейс и проверить установку 1С'
obj1C = Object("1C");
If (~obj1C.Is1CInstall(0))
    is_message("1C 7.7 не установлена");
    Return;
End If

'Сформировать строку запуска и подключиться'
PathToData = "C:\\DATABASE_1C";
UserName = "Иван Иванов";
Password = "";
InitString = "/d\"" + PathToData + "\" /n" + UserName + "/p" + Password;
If (~obj1C.Connect(0, InitString))
    is_message("Ошибка подключения к 1С");
End If
```

Создание объекта 1С и получение ссылки на справочник в языке EME-L:

```EME-L
'Создать объект справочника Склады через соединение'
objWarehouse1C = Object("1CItem",
    obj1C.CreateObject("Справочники.Склады"));
objWarehouse1C.FindByDescr("Наименование", "СВХ", 1);
```

Вычисление выражения на языке 1С 7.7:

```EME-L
'Получить значение периода расчёта'
Period = obj1C.EvalExpr("РасчетныйПериод");
```

Разрыв соединения:

```EME-L
'Закрыть COM-соединение с 1С'
obj1C.Disconnect();
```
