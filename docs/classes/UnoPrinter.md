# Класс UnoPrinter

Класс `UnoPrinter` в языке EME-L — это класс для управления струйным принтером Inc Jet UNO в системе EME.WMS. В системе EME.WMS он обеспечивает полный цикл работы с принтером: установку TCP/IP-соединения, передачу заданий печати, контроль состояния печати, получение журнальных сообщений по UDP и отключение от принтера.

В языке EME-L класс `UnoPrinter` (CI-имя `CIUnoPrinter`) предоставляет конструктор без аргументов и методы для настройки соединения, запуска и остановки печати, а также чтения статуса и счётчиков страниц. Объект класса должен быть размещён в классе-резиденте; перед использованием необходимо задать IP-адрес и имя Job-файла, после чего вызвать метод `Connect`.

> **Внимание:** в C++-исходнике `i_UnoPrinter.cpp` инициализация внутреннего рабочего компонента (`CRipInImagerWorker`) и запуск фонового потока сохранения серийных номеров находятся в конструкторе `CIUnoPrinter(int argn, COleVariant args[])`, тогда как в `BEGIN_IMPLEMENT` зарегистрирован только конструктор без аргументов (`IMPLEMENT_CTOR(CIUnoPrinter, 0, ...)`), и его тело пустое. В языке EME-L это означает расхождение между REGISTRY-контрактом и C++-реализацией; при использовании в конкретной сборке ядра проверяйте, что объект корректно инициализируется.

## Создание объекта

В языке EME-L класс `UnoPrinter` имеет один конструктор — без аргументов.

```EME-L
'Создать объект управления принтером UnoPrinter'
objUnoPrinter = Object("UnoPrinter");
```

Согласно контракту реестра, при создании объекта в языке EME-L инициализируется внутренний рабочий компонент, настраивается подписка на события принтера и запускается фоновый поток для сохранения напечатанных серийных номеров в базу данных.

## Подключение и настройка соединения в EME-L

В языке EME-L методы этой группы управляют TCP/IP-соединением с принтером Inc Jet UNO.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetIpAddress(ipAddress)` | ipAddress: String | Empty | Устанавливает IP-адрес принтера в языке EME-L. Адрес используется при последующем вызове `Connect`. |
| `SetJobName(jobName)` | jobName: String | Empty | Устанавливает название задачи печати (имя Job-файла), загруженного в принтер. Название должно быть задано перед вызовом `StartPrint`. |
| `Connect()` | — | Boolean | Устанавливает TCP/IP-соединение с принтером по ранее заданному IP-адресу. Если соединение уже установлено и принтер находится в состоянии "Подключен", возвращает `TRUE` без повторного подключения. `TRUE` — соединение установлено или уже было активно; `FALSE` — не удалось установить соединение. |
| `Disconnect()` | — | Empty | Разрывает TCP/IP-соединение с принтером и освобождает связанные ресурсы в системе EME.WMS. |
| `GetConnectionStatus()` | — | Integer | Возвращает текущее состояние TCP/IP-соединения: `0` — не подключен, `1` — выполняется подключение, `2` — подключен. |
| `IsPrinting()` | — | Boolean | `TRUE` — принтер выполняет печать; `FALSE` — принтер не печатает. |

## Управление печатью в EME-L

В языке EME-L методы этой группы запускают и останавливают задание печати на принтере Inc Jet UNO.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `StartPrint(serialNumbers)` | serialNumbers: Array (по ссылке) | Empty | Запускает задание печати серийных номеров в языке EME-L. Перед запуском должны быть заданы имя Job-файла (методом `SetJobName`) и IP-адрес принтера. Сбрасывает счётчик напечатанных страниц и передаёт принтеру массив серийных номеров (тип 252 — ссылки на записи `SerialNumber`). |
| `StopPrint()` | — | Empty | Останавливает текущее задание печати на принтере. Сбрасывает счётчики напечатанных и запланированных страниц в ноль. |

> **Примечание:** в реестре EME-L метод `StopPrint()` декларируется без аргументов, но в теле C++-метода проверка выполняется как `if (argn > 1) too_many();`. Фактически метод примет один аргумент, который будет проигнорирован. Для совместимости с документированным контрактом в языке EME-L вызывайте `StopPrint()` без аргументов.

## Счётчики страниц

В языке EME-L методы этой группы возвращают текущий прогресс печати задания на принтере UnoPrinter.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetNumberPrintedPage()` | — | Integer | Возвращает количество страниц, уже напечатанных в рамках текущего задания печати. Счётчик увеличивается по событию от принтера после печати каждой страницы. |
| `GetNumberPrintPage()` | — | Integer | Возвращает общее количество страниц для печати, установленное при запуске задания методом `StartPrint`. Равно размеру переданного массива серийных номеров. |

## Константы и детали реализации в системе EME.WMS

В системе EME.WMS класс `UnoPrinter` (CI-имя `CIUnoPrinter`) использует следующие сетевые параметры и механизмы:

- TCP/IP-порты управления принтером: **10000** (передача команд) и **10001** (приём ответов/событий).
- UDP-порт для получения журнальных сообщений: **20000**.
- Таймаут установки TCP-соединения: **1000 мс**.
- Фоновый поток сохранения напечатанных серийных номеров открывает серверную транзакцию и помечает записи `SerialNumber` как напечатанные (`SetSerialPrinted`).
- Массив, передаваемый в `StartPrint`, должен содержать ссылки на строки записей серийных номеров (тип 252), а не произвольные целые числа.

## Примеры в языке EME-L

В языке EME-L создание объекта и подключение к принтеру выглядят так:

```EME-L
'Создать объект UnoPrinter в резиденте'
objUnoPrinter = Object("UnoPrinter");
objUnoPrinter.SetIpAddress("192.168.1.10");
objUnoPrinter.SetJobName("Job1.job");

'Установить соединение с принтером'
If (objUnoPrinter.Connect())
    Status = objUnoPrinter.GetConnectionStatus();
End If
```

Запуск печати серийных номеров в системе EME.WMS:

```EME-L
'Собрать массив ссылок на серийные номера'
arrSerials = Object("Array");
Loop (r_SerialNumbers)
    arrSerials.Add(r_SerialNumbers.GetLine());
End Loop

'Отправить массив на печать'
objUnoPrinter = Object("UnoPrinter");
If (objUnoPrinter.Connect())
    objUnoPrinter.StartPrint(arrSerials);
End If
```

Контроль состояния по таймеру в диалоге:

```EME-L
'Обновить поле со статусом подключения'
Switch (objUnoPrinter.GetConnectionStatus())
Case 0:
    StatusTxt = tr("Принтер не подключен");
    Break;
Case 1:
    StatusTxt = tr("Подключение к принтеру...");
    Break;
Case 2:
    StatusTxt = tr("Принтер подключен");
    Break;
End Switch
ctrlStatePrinterProduction.Put(StatusTxt);
ctrlQtyPrinted.PutText(is_format("%d", objUnoPrinter.GetNumberPrintedPage()));
ctrlQtyAll.PutText(is_format("%d", objUnoPrinter.GetNumberPrintPage()));
```

Остановка печати в языке EME-L:

```EME-L
'Прервать текущее задание печати'
objUnoPrinter.StopPrint();
ctrlQtyPrinted.PutText("0");
ctrlQtyAll.PutText("0");
```
