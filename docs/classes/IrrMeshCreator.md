### IrrMeshCreator — класс конструктора полигональных сеток движка Irrlicht (язык EME-L)

Класс `IrrMeshCreator` в языке EME-L — это класс конструктора полигональных сеток (mesh) движка Irrlicht. Предназначен для программного создания и настройки трехмерных полигональных сеток из набора узлов, объединяемых в иерархическую структуру. В системе EME.WMS используется для формирования статических 3D-моделей, которые затем передаются в объект `IrrMesh` или `IrrOctree` для отображения в сцене Irrlicht.

Класс `IrrMeshCreator` поддерживает добавление плоскостей и кубов, задание масштаба, углов поворота и положения каждого узла относительно родительского, назначение текстуры и выделение на ней прямоугольных фрагментов, а также установку цвета отдельных углов полигонов. Сетка может быть сохранена в дерево свойств PropertyTree и загружена обратно. Для повторного использования фрагментов сетки поддерживаются параметризованные макросы.

### Создание объекта IrrMeshCreator

В блоке регистрации `BEGIN_IMPLEMENT(CIIrrMeshCreator)` в C++ исходнике зарегистрированы два конструктора: без аргументов и с одним аргументом. В текущей реализации конструктор с аргументом не используется для передачи параметров — все настройки сетки задаются последовательно через методы класса.

```EME-L
'Создать пустой конструктор сетки'
meshCreator = Object("IrrMeshCreator");
```

```EME-L
'Конструктор с аргументом — не используется в текущей реализации'
meshCreator = Object("IrrMeshCreator", arg0);
```

### Сериализация сетки IrrMeshCreator

Методы для сохранения и восстановления описания полигональной сетки через дерево свойств в языке EME-L. Сетка сохраняется в виде узлов и макросов, что позволяет полностью восстановить её состояние.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SaveMesh(arg0)` | arg0: PropertyTree (по ссылке) | Empty | Сохраняет полное описание полигональной сетки, включая узлы и макросы, в объект дерева свойств PropertyTree. |
| `LoadMesh(arg0)` | arg0: PropertyTree (по ссылке) | Empty | Загружает полное описание полигональной сетки, включая узлы и макросы, из объекта дерева свойств PropertyTree. |

### Текстура и материал IrrMeshCreator

Метод `SetMaterial` в языке EME-L задает путь к файлу текстуры и размеры изображения, используемые при построении сетки. Перед вызовом методов задания фрагментов текстуры необходимо вызвать `SetMaterial`, иначе при построении сетки возникнет ошибка.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetMaterial(arg0, arg1, arg2)` | arg0: String (путь к файлу текстуры), arg1: Integer (ширина изображения в пикселях), arg2: Integer (высота изображения в пикселях) | Empty | Задает путь к файлу текстуры и размеры изображения, используемые при построении сетки. |

### Трансформации узлов IrrMeshCreator

Методы `SetScale`, `SetRotation` и `SetPosition` в языке EME-L задают трансформации узла относительно родительского узла в трехмерном пространстве. Значения проверяются на корректность (отсутствие NAN и бесконечностей). При записи макроса аргументы `x`, `y`, `z` могут быть переданы строкой в виде `"[коэффициент*]имя_параметра"` для привязки к макропараметру.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetScale(arg0, arg1, arg2, arg3)` | arg0: Integer (node), arg1: Real (x), arg2: Real (y), arg3: Real (z) | Empty | Задает масштаб узла по осям X, Y и Z. |
| `SetRotation(arg0, arg1, arg2, arg3)` | arg0: Integer (node), arg1: Real (x), arg2: Real (y), arg3: Real (z) | Empty | Задает углы поворота узла в градусах по осям X, Y и Z. |
| `SetPosition(arg0, arg1, arg2, arg3)` | arg0: Integer (node), arg1: Real (x), arg2: Real (y), arg3: Real (z) | Empty | Задает смещение узла относительно родительского узла по осям X, Y и Z. |

### Фрагменты текстуры IrrMeshCreator

Методы в языке EME-L задают координаты фрагмента текстуры для отображения на полигонах узла. Координаты задаются в пикселях относительно размеров изображения, установленных методом `SetMaterial`. Значения `right` и `bottom` задают границу за пределами последнего пикселя. Перед вызовом необходимо задать размеры текстуры методом `SetMaterial`.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetLeftTop(arg0, arg1, arg2)` | arg0: Integer (node), arg1: Integer (u), arg2: Integer (v) | Empty | Задает координаты левого верхнего угла фрагмента текстуры в пикселях. |
| `SetRightTop(arg0, arg1, arg2)` | arg0: Integer (node), arg1: Integer (u), arg2: Integer (v) | Empty | Задает координаты правого верхнего угла фрагмента текстуры в пикселях. |
| `SetRightBottom(arg0, arg1, arg2)` | arg0: Integer (node), arg1: Integer (u), arg2: Integer (v) | Empty | Задает координаты правого нижнего угла фрагмента текстуры в пикселях. |
| `SetLeftBottom(arg0, arg1, arg2)` | arg0: Integer (node), arg1: Integer (u), arg2: Integer (v) | Empty | Задает координаты левого нижнего угла фрагмента текстуры в пикселях. |
| `SetImageRect(arg0, arg1)` | arg0: Integer (node), arg1: IrrIRect (по ссылке) | Empty | Задает прямоугольный фрагмент текстуры на основе объекта IrrIRect. |
| `SetImageRect(arg0, arg1, arg2)` | arg0: Integer (node), arg1: IrrIRect (по ссылке), arg2: Integer (start, 0..3, по умолчанию 0) | Empty | То же с указанием начальной вершины обхода углов прямоугольника. |
| `SetImageRect(arg0, arg1, arg2, arg3, arg4)` | arg0: Integer (node), arg1: Integer (left), arg2: Integer (top), arg3: Integer (right), arg4: Integer (bottom) | Empty | Задает прямоугольный фрагмент текстуры координатами в пикселях. |
| `SetImageRect(arg0, arg1, arg2, arg3, arg4, arg5)` | + arg5: Integer (start, 0..3) | Empty | То же с указанием начальной вершины обхода углов. |

### Цвет вершин IrrMeshCreator

| `SetLeftBottomColor(arg0, arg1)` | arg0: Integer (node), arg1: Integer (RGBA) | Empty | Задает RGBA-цвет левого нижнего угла узла. |

### Добавление узлов IrrMeshCreator

Методы в языке EME-L добавляют дочерние узлы в иерархию сетки. Все методы возвращают идентификатор добавленного узла (Integer). Узел с `parent = -1` является корневым.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AddNode(arg0, arg1)` | arg0: Integer (parentNode, -1 для корня), arg1: String (имя узла) | Integer | Добавляет дочерний узел с заданным именем. Возвращает идентификатор нового узла. |
| `AddPlane(arg0)` | arg0: Integer (parentNode) | Integer | Добавляет дочернюю прямоугольную грань размером 1x1. Возвращает идентификатор нового узла. |
| `AddCube(arg0)` | arg0: Integer (parentNode) | Integer | Добавляет дочерний куб размером 1x1x1. Возвращает идентификатор нового узла. |

### Макросы IrrMeshCreator

Макросы в языке EME-L позволяют создавать параметризованные фрагменты сетки для многократного использования. Запись макроса начинается методом `BeginMacro`, завершается `EndMacro`, а вставка экземпляра выполняется методом `CallMacro` с передачей числовых значений параметров.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `BeginMacro(arg0, ...)` | arg0: String (имя макроса), далее имена параметров (String) | Empty | Начинает запись макроса с указанным именем и списком параметров. Не допускается вложенная запись макроса и повторное определение макроса с тем же именем. |
| `EndMacro()` | — | Empty | Завершает запись макроса. Все последующие вызовы методов трансформаций и добавления узлов записываются в основную сетку. |
| `CallMacro(arg0, arg1, ...)` | arg0: Integer (parentNode), arg1: String (имя макроса), далее значения параметров (Real) | Integer | Вызывает макрос: создает узел с именем `"Macro:имя,значения..."`, который при построении сетки раскрывается в копию узлов макроса с подстановкой параметров. Возвращает идентификатор добавленного узла. |

### Примеры использования IrrMeshCreator в языке EME-L

Создание простой стены из двух плоскостей с текстурой:

```EME-L
'Создать конструктор сетки и назначить текстуру'
meshCreator = Object("IrrMeshCreator");
meshCreator.SetMaterial("3D\\brick.jpg", 200, 200);

'Добавить корневую плоскость'
Plane1Node = meshCreator.AddPlane(-1);
meshCreator.SetImageRect(Plane1Node, 0, 0, 199, 199);
meshCreator.SetPosition(Plane1Node, 0, -0.06, 0);

'Добавить вторую плоскость'
Plane2Node = meshCreator.AddPlane(-1);
meshCreator.SetImageRect(Plane2Node, 0, 0, 199, 199);
meshCreator.SetPosition(Plane2Node, 0, 0.06, 0);
```

Создание стеллажа с использованием макросов:

```EME-L
'Создать конструктор сетки для стеллажа'
meshCreator = Object("IrrMeshCreator");
meshCreator.SetMaterial("3D\\Rack\\rack.png", 500, 1000);

'Определить макрос коробки с параметром ширины'
meshCreator.BeginMacro("Box", "Width");
BoxNode = meshCreator.AddCube(-1);
meshCreator.SetScale(BoxNode, 1, 1, 1);
meshCreator.EndMacro();

'Вставить экземпляр макроса с шириной 2.0'
MacroNode = meshCreator.CallMacro(-1, "Box", 2.0);

'Сохранить сетку в PropertyTree для передачи в IrrMesh'
PropertyTree = Object("PropertyTree");
meshCreator.SaveMesh(PropertyTree);
```
