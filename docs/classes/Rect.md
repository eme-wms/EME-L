Rect CIRect

Класс `Rect` в языке EME-L — класс для работы с прямоугольной областью в двумерной плоскости. Представляет прямоугольник, заданный четырьмя целочисленными координатами: левой, верхней, правой и нижней. Поддерживает создание пустого прямоугольника, копирование из другого объекта Rect и инициализацию координатами. Предоставляет методы для получения и изменения каждой стороны, вычисления ширины и высоты, а также проверки принадлежности точки области. В языке EME-L используется для работы с координатами, размерами и областями экрана или элементов интерфейса.

## Создание объекта

В системе EME.WMS объект класса `Rect` создаётся через оператор `Object()`. Доступны три перегрузки конструктора.

```EME-L
'Пустой прямоугольник, все координаты равны 0'
rect = Object("Rect");
```

```EME-L
'Копия существующего прямоугольника'
rectCopy = Object("Rect", rect);
```

```EME-L
'Прямоугольник с явными координатами границ'
rect = Object("Rect", 100, 100, 500, 500);
```

Конструктор без аргументов инициализирует все границы нулём. Конструктор с одним аргументом копирует координаты из переданного объекта Rect. Конструктор с четырьмя аргументами задаёт левую, верхнюю, правую и нижнюю координаты в указанном порядке.

## Границы прямоугольника

Методы для чтения и изменения отдельных сторон прямоугольника в языке EME-L. Сеттеры возвращают предыдущее значение.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetLeft()` | — | Integer | Левая граница прямоугольника. |
| `SetLeft(arg0)` | arg0: Integer | Integer | Устанавливает левую границу. Возвращает предыдущее значение. |
| `GetTop()` | — | Integer | Верхняя граница прямоугольника. |
| `SetTop(arg0)` | arg0: Integer | Integer | Устанавливает верхнюю границу. Возвращает предыдущее значение. |
| `GetRight()` | — | Integer | Правая граница прямоугольника. |
| `SetRight(arg0)` | arg0: Integer | Integer | Устанавливает правую границу. Возвращает предыдущее значение. |
| `GetBottom()` | — | Integer | Нижняя граница прямоугольника. |
| `SetBottom(arg0)` | arg0: Integer | Integer | Устанавливает нижнюю границу. Возвращает предыдущее значение. |

## Размеры и проверка точки

Методы вычисления геометрических размеров и проверки принадлежности точки прямоугольнику в языке EME-L.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetWidth()` | — | Integer | Ширина прямоугольника (`right - left`). |
| `GetHeight()` | — | Integer | Высота прямоугольника (`bottom - top`). |
| `IsInside(arg0, arg1)` | arg0: Integer (x), arg1: Integer (y) | Boolean | `TRUE` — точка находится внутри прямоугольника. Левая и верхняя границы включаются, правая и нижняя — исключаются. |

## Примеры

Создание прямоугольника и получение его размеров в языке EME-L:

```EME-L
rect = Object("Rect", 100, 100, 500, 500);
width = rect.GetWidth();
height = rect.GetHeight();
```

Настройка области окна консоли в системе EME.WMS:

```EME-L
rect = Object("Rect");
rect.SetLeft(100);
rect.SetTop(100);
rect.SetRight(600);
rect.SetBottom(400);
Console.SetRect(rect);
```

Проверка попадания точки в область в языке EME-L:

```EME-L
rect = Object("Rect", 0, 0, 200, 100);
If (rect.IsInside(50, 50))
    'Точка внутри прямоугольника'
End If
```

Передача прямоугольника в метод `AddSprite` класса `Irrlicht` в системе EME.WMS (реальный пример из компонента 3D-курсора):

```EME-L
'Спрайт-прицел создаётся с прямоугольником 16×16'
CursorSpritePoint = m_Irrlicht.AddSprite("Cursor#point", Object("Rect", 0, 0, 16, 16));
```
