# Класс RegNo

> Синонимы для поиска: CIRegNo RegNo класс EME-L, генератор регистрационных номеров EME-L, учетный номер строки EME-L, нумерация строк записи EME-L, Init Get GetNext IsValid GetDialogGroup GetRecordGroup GetDialogIndex GetRecordIndex, throw_exceptions search_full_record search_last indent group_indent, превышена разрядность EME-L, циклический переход номера EME-WMS, DEFAULT_INDENT, регистрационный номер базы данных EME.WMS

Класс `RegNo` в языке EME-L — генератор регистрационных номеров строк записи (учетных номеров). Предназначен для автоматического формирования и валидации регистрационных номеров строк записей базы данных. Поддерживает два режима генерации номеров: поиск последнего использованного номера с инкрементом и поиск первого свободного номера в заданном диапазоне.

Класс RegNo в языке EME-L обеспечивает работу с группами номеров, проверку допустимости вручную введенных номеров, а также получение текущих групп и индексов из записи или диалога. При переполнении разрядности может либо генерировать исключение, либо выполнять циклический переход в начало диапазона в зависимости от настроек конструктора.

Класс RegNo в системе EME.WMS используется для организации нумерации документов, строк таблиц и других объектов, требующих уникальных регистрационных номеров в рамках группы.

Объект класса `RegNo` в языке EME-L создаётся через `Object()` и привязывается к объекту записи базы данных. Конструктор требует минимум один аргумент — ссылку на объект записи. Создание без аргументов генерирует ошибку.

## Создание объекта класса RegNo

Класс `RegNo` в языке EME-L имеет четыре перегрузки конструктора — от одного до четырёх аргументов.

```EME-L
'Минимальный конструктор — только объект записи'
objRegNo = Object("RegNo", r_Record);
```

```EME-L
'С объектом диалога и настройкой переполнения'
objRegNo = Object("RegNo", r_Record, r_Dialog, TRUE);
```

| Конструктор | Аргументы | Описание |
|-------------|-----------|----------|
| `RegNo(record)` | record: Record (ссылка) | Создаёт генератор на основе объекта записи. |
| `RegNo(record, dialog)` | record: Record (ссылка), dialog: Record (ссылка) | Дополнительно задаёт объект диалога для определения текущей группы номеров. |
| `RegNo(record, dialog, throw_exceptions)` | + throw_exceptions: Boolean | TRUE — генерировать исключение «Превышена разрядность» при переполнении. FALSE — циклический переход в начало диапазона. |
| `RegNo(record, dialog, throw_exceptions, search_full_record)` | + search_full_record: Boolean | TRUE — просматривать все строки записи для определения максимального индекса (включается при реанимации строк в записи, когда просмотра нескольких строк с конца недостаточно; использует блочное чтение `regno_GetLastIndex` класса RegNo в языке EME-L). FALSE — ограниченное число последних строк (по умолчанию). |

## Инициализация параметров поиска

Метод `Init` в языке EME-L задаёт режим поиска номеров и глубину просмотра строк записи. Вызов без аргументов недопустим — генерируется ошибка «Слишком мало аргументов».

| Перегрузка | Аргументы | Возвращает | Описание |
|------------|-----------|------------|----------|
| `Init(search_last)` | search_last: Boolean | Empty | Устанавливает только режим поиска. indent и group_indent получают значения по умолчанию (1000). |
| `Init(search_last, indent)` | search_last: Boolean, indent: Integer | Empty | Устанавливает режим поиска и число просматриваемых строк. Тот же `indent` используется для group_indent. |
| `Init(search_last, indent, group_indent)` | search_last: Boolean, indent: Integer, group_indent: Integer | Empty | Полная инициализация. Если `indent` передан как Empty, используется значение по умолчанию (1000). |

Параметры метода `Init` класса RegNo в языке EME-L:

- **search_last** — режим поиска: TRUE — искать последний использованный номер и возвращать следующий (по умолчанию). FALSE — искать первый свободный номер.
- **indent** — число просматриваемых последних строк записи при поиске последнего номера (по умолчанию 1000).
- **group_indent** — число просматриваемых последних строк текущей группы (по умолчанию 1000).

## Генерация и проверка номеров

Методы класса RegNo в языке EME-L для получения и валидации регистрационных номеров.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Get()` | — | String | Новый регистрационный номер. В режиме поиска последнего номера определяет максимальный индекс в текущей группе и возвращает следующий. В режиме свободного номера находит первый незанятый индекс в битовом буфере. При переполнении разрядности либо генерирует исключение, либо выполняет циклический переход. |
| `GetNext(reg_no)` | reg_no: String | String | Регистрационный номер, следующий за указанным. Увеличивает индекс на единицу. |
| `IsValid(reg_no)` | reg_no: String | Boolean | Проверяет допустимость: группа совпадает с текущей, индекс больше нуля и (если задана разрядность) не превышает максимум. TRUE — допустим, FALSE — недопустим. |

Методы класса RegNo в системе EME.WMS для получения групп и индексов текущей строки из диалога или записи.

| `GetRecordGroup()` | — | String | Группа номера текущей строки записи. |

| `GetRecordIndex()` | — | Integer | Индекс номера текущей строки записи. |

## Константы

Константы класса RegNo в языке EME-L, используемые по умолчанию:

| Константа | Значение | Описание |
|-----------|----------|----------|
| `DEFAULT_INDENT` | 1000 | Число просматриваемых последних строк по умолчанию для indent и group_indent. |

Тексты ошибок и исключений класса RegNo в языке EME-L:

- Переполнение разрядности: `«Превышена разрядность»`.
- Незаданная разрядность (режим свободного номера): `«Не задана разрядность»`.
- Вызов `GetDialogGroup` / `GetDialogIndex` без объекта диалога: `«Не задан диалог»`.

## Примеры

Генерация номера для нового документа с инкрементом от последней строки в языке EME-L (реальный паттерн из прикладных скриптов EME.WMS):

```EME-L
'Создать генератор, привязанный к записи документа'
objRegNo = Object("RegNo", r_Document);
'Режим поиска последнего номера; глубина просмотра — все строки документа'
objRegNo.Init(TRUE, r_Document.GetNoOfLines(TRUE));
'Получить следующий регистрационный номер и записать его в строку'
r_Document.PutRegNo(objRegNo.Get());
```

Проверка вручную введенного номера в системе EME.WMS:

```EME-L
'Проверить допустимость номера в текущей группе'
If (objRegNo.IsValid(sEnteredNo))
    'Номер допустим — использовать'
Else
    'Номер недопустим — предложить исправить'
End If
```

Получение номера от отдельной записи-перечисления без диалога в языке EME-L:

```EME-L
'Запись rDocEnum используется только для вычисления номера'
enum = Object("RegNo", rDocEnum);
csReg = enum.Get();
r_Document.PutRegNo(csReg);
```
