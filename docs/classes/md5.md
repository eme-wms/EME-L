# Класс MD5 — вычисление 128-битных хеш-значений по алгоритму MD5

Класс `MD5` в языке EME-L — класс для вычисления 128-битных хеш-значений по алгоритму MD5 (Message Digest 5). C++-имя класса — `CIMD5`. В системе EME.WMS класс MD5 предназначен для получения криптографических контрольных сумм (дайджестов сообщений) от строковых данных.

В языке EME-L класс MD5 вычисляет необратимый хеш фиксированной длины в виде 32-символьной шестнадцатеричной строки. Поддерживает передачу входных данных в виде строки или объекта `String` (`CIString`). Используется для проверки целостности данных, создания уникальных идентификаторов и других задач, требующих необратимого хеширования.

Объект класса `MD5` в языке EME-L создаётся без параметров; вычисление хеша выполняется методом `Calculate`.

## Создание объекта — конструкторы класса MD5

Класс `MD5` в языке EME-L имеет один доступный конструктор — без аргументов.

```EME-L
'Создание объекта MD5 без параметров в языке EME-L'
hashCalculator = Object("MD5");
```

Хотя в C++ классе `CIMD5` присутствует конструктор с аргументами `argn`/`args`, он не зарегистрирован в `IMPLEMENT_CTOR` и недоступен из языка EME-L. В скриптах EME-L используется только форма `Object("MD5")`.

## Методы класса MD5

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Calculate(arg0)` | arg0: String или ссылка на `String` / `CIString` | String | Вычисляет 128-битный хеш по алгоритму MD5 для переданных данных. Возвращает 32-символьную шестнадцатеричную строку. При передаче пустого значения (`VT_NULL`) в языке EME-L извлекает строку из объекта `CIString`; при передаче строки вычисляет хеш напрямую. |

## Примеры использования класса MD5 в языке EME-L

Вычисление хеша от строки:

```EME-L
'Вычислить MD5-хеш строки в языке EME-L'
hashCalculator = Object("MD5");
hash = hashCalculator.Calculate("Текст для хеширования");
'hash = 32-символьная шестнадцатеричная строка'
```

Проверка пароля пользователя:

```EME-L
'Проверить пароль пользователя, сравнив MD5-хеш с сохранённым значением'
Return Password == Object("MD5").Calculate(User.GetPassword());
```

Получение хеша JSON-запроса:

```EME-L
'Вычислить MD5-хеш JSON-тела запроса для контроля целостности в системе EME.WMS'
Hash = Object("MD5").Calculate(Request.GetJSON());
```
