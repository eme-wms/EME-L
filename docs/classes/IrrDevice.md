### IrrDevice — класс движка Irrlicht (корневой объект 3D-сцены) — описание на русском

Класс `IrrDevice` в языке EME-L представляет 3D-движок Irrlicht и управляет трёхмерной сценой в составе 3D-органа управления EME.WMS. Это корневой объект, через который скрипт EME-L создаёт и сбрасывает устройство Irrlicht, получает доступ к корневому узлу сцены (`IrrNode`) и активной камере (`IrrCamera`), регистрирует именованные узлы, работает со спрайтами (`IrrSprite`) и настраивает глобальные параметры рендеринга: рассеянное освещение (ambient light), цвет теней, глобальный масштаб и режим предзагрузки ресурсов.

В системе EME.WMS класс `IrrDevice` является точкой входа для построения интерактивной трёхмерной визуализации склада: от него получают дочерние узлы сцены, на которых строится всё 3D-отображение. Класс также позволяет получать размеры экрана и текущий FPS, а также запускать обновление и автоматическое обновление 3D-органа управления.

C++-класс реализации — `CIIrrDevice`. Скриптовое имя, используемое в `Object("...")`, — `IrrDevice`.

### Создание объекта в языке EME-L

Класс `IrrDevice` имеет один конструктор — без аргументов. Внутреннее устройство Irrlicht создаётся с типом драйвера `EDT_NULL` (без вывода в окно). Для привязки к 3D-органу управления EME.WMS объект обычно создаётся в обработчике `OnGetAnswer` и сохраняется в члене класса с префиксом `m_`.

```EME-L
'Создать 3D-устройство Irrlicht без привязки к окну'
m_Irrlicht = Object("IrrDevice");
```

### Управление сценой (язык EME-L, класс IrrDevice)

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Reset()` | — | Empty | Сбрасывает 3D-орган управления Irrlicht: удаляет вспомогательный узел клика, активную камеру и корневой узел сцены, очищает связи спрайтов с узлами, уничтожает все созданные спрайты и очищает реестры именованных узлов и спрайтов. |
| `Update()` | — | Empty | Принудительно обновляет 3D-орган управления, вызывая обновление окна-хоста. |
| `SetAutoUpdate(arg0)` | arg0: Boolean | Empty | Включает (`TRUE`) или отключает (`FALSE`) автоматическое обновление 3D-органа управления. |
| `GetRootNode()` | — | IrrNode | Корневой узел сцены 3D-органа управления. При первом обращении создаётся и кэшируется объект-обёртка `IrrNode`, связанный с внутренним корневым узлом Irrlicht. |
| `GetActiveCamera()` | — | IrrCamera или Empty | Активная камера сцены. При каждом вызове внутренний объект-обёртка `IrrCamera` обновляется данными текущей активной камеры Irrlicht. Возвращает `Empty`, если активная камера не задана. |
| `SetGlobalScale(arg0)` | arg0: Real | Empty | Устанавливает глобальный масштаб сцены. Влияет на масштаб и положение узлов, имеющих родителем корневой узел сцены Irrlicht. |

### Поиск узлов и спрайтов (в классе IrrDevice)

Реестры именованных узлов и спрайтов поддерживаются внутри `IrrDevice` и позволяют скрипту EME-L получать ранее созданные объекты по имени без повторного обхода сцены.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `FindNodeByName(arg0)` | arg0: String | IrrNode или Empty | Находит ранее зарегистрированный узел сцены Irrlicht по имени. Возвращает `Empty`, если узел не найден. |
| `FindSpriteByName(arg0)` | arg0: String | IrrSprite или Empty | Находит созданный спрайт Irrlicht по имени. Возвращает `Empty`, если спрайт не найден. |
| `GetNodeNames(arg0)` | arg0: Array (по ссылке) | Empty | Заполняет переданный массив именами всех зарегистрированных узлов сцены. |
| `GetSpriteNames(arg0)` | arg0: Array (по ссылке) | Empty | Заполняет переданный массив именами всех созданных спрайтов Irrlicht. |

### Создание спрайтов в системе EME.WMS

Метод `AddSprite` в языке EME-L перегружен по числу аргументов: форма с двумя аргументами создаёт спрайт по имени и прямоугольнику, форма с одним аргументом — по дереву свойств.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `AddSprite(arg0, arg1)` | arg0: String (имя), arg1: Rect (по ссылке) | IrrSprite | Создаёт спрайт Irrlicht с заданным именем и прямоугольником положения/размера. Если спрайт с таким именем уже существует, возвращает ранее созданный объект без создания дубликата. |
| `AddSprite(arg0)` | arg0: PropertyTree (по ссылке) | IrrSprite | Создаёт спрайт на основе параметров из дерева свойств `PropertyTree`. Используется при загрузке описания спрайта из конфигурации 3D-органа управления EME.WMS. |

### Параметры рендеринга (класс IrrDevice)

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetFPS()` | — | Integer | Текущее количество кадров в секунду (FPS), рассчитанное видеодрайвером Irrlicht. |
| `GetScreenWidth()` | — | Integer | Ширина экрана/окна вывода видеодрайвера Irrlicht, в пикселях. |
| `GetScreenHeight()` | — | Integer | Высота экрана/окна вывода видеодрайвера Irrlicht, в пикселях. |
| `SetAmbientLight(arg0)` | arg0: Integer (RGBA, формат `0xAARRGGBB`, компоненты 0..255) | Empty | Задаёт цвет рассеянного (окружающего) освещения сцены Irrlicht. |
| `SetShadowColor(arg0)` | arg0: Integer (RGBA, формат `0xAARRGGBB`, компоненты 0..255) | Empty | Задаёт цвет теней, используемый менеджером сцены Irrlicht. |
| `SetPreloadResources(arg0)` | arg0: Integer (0 — нет, любое ненулевое — да) | Empty | Задаёт признак необходимости предварительной загрузки ресурсов на сервере. Значение сохраняется в свойстве корневого узла сцены. |
| `IsPreloadResources()` | — | Integer | Возвращает `1`, если предварительная загрузка ресурсов включена, `0` — если отключена. Значение считывается из свойства корневого узла. |

### Примеры использования IrrDevice в языке EME-L

Создание 3D-устройства и базовая настройка сцены (по мотивам класса `ATWms3D`):

```EME-L
'Создать IrrDevice и задать масштаб и режим предзагрузки ресурсов'
m_Irrlicht = Object("IrrDevice");
m_Irrlicht.SetGlobalScale(10.0);
m_Irrlicht.SetPreloadResources(FALSE);
```

Настройка рассеянного освещения и цвета теней в системе EME.WMS (по мотивам `Wms3DLightComponent`):

```EME-L
'Рассеянный свет и цвет теней в формате 0xAARRGGBB'
m_Irrlicht.SetAmbientLight(is_RGBA(80, 80, 80, 255));
m_Irrlicht.SetShadowColor(is_RGBA(80, 80, 80, 150));
```

Поиск курсора по имени и создание спрайта-этикетки (по мотивам `Wms3DCursorComponent`):

```EME-L
'Найти зарегистрированный узел курсора'
CursorNode As "IrrMesh" = m_Irrlicht.FindNodeByName("Cursor");
If (is_empty(CursorNode))
    'Узел не найден — создать через корневой узел сцены'
    RootNode = m_Irrlicht.GetRootNode();
End If

'Создать спрайт-этикетку с привязкой к узлу курсора'
CursorSpriteLabel = m_Irrlicht.AddSprite("Cursor#label", Object("Rect", 0, 0, 200, 50));
CursorSpriteLabel.LinkToNode(CursorNode, 100);
```
