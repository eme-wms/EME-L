### MultiArray — класс двумерного динамического массива в языке EME-L

Класс `MultiArray` (C++ `CIMultiArray`) в языке EME-L — двумерный динамический массив для хранения табличных данных. Массив имеет фиксированное число столбцов (вторая размерность N, по умолчанию 2) и переменное число строк. На каждый основной (первый) индекс приходится N элементов, доступ к которым — по второму индексу. Элементы массива могут содержать значения любого типа данных EME-L.

Столбцам можно назначать строковые имена методом `SetColumnNames` и обращаться к элементам как по числовому индексу столбца, так и по имени. Нумерация строк и столбцов начинается с 0. Используется в прикладных скриптах EME-L для хранения и обработки табличных наборов данных, промежуточных результатов вычислений, параметров диалогов и других структурированных коллекций значений.

Объект класса создаётся через `Object("MultiArray", ...)`. В системе EME.WMS применяется для накопления данных перед транзакцией, передачи наборов строк между методами и формирования промежуточных выборок.

### Создание объекта MultiArray — конструкторы

Класс имеет четыре перегрузки конструктора, различающиеся по числу и типу переданных аргументов. Выбор перегрузки определяется по первому аргументу: если он строковый — срабатывает конструктор с именами столбцов; если числовой (Integer) — конструктор с размерами.

| Конструктор | Аргументы | Описание |
|-------------|-----------|----------|
| `MultiArray()` | — | Пустой массив 2×0: 2 столбца (значение N по умолчанию), 0 строк. Все элементы — Empty. |
| `MultiArray(имяСтолбца1[, ...[, имяСтолбцаN]])` | имена: String/Integer/Real | Массив с N столбцами и 0 строк. Integer и Real преобразуются в строку. Выбрасывает исключение при дублировании имён или пустой строке в качестве имени. |
| `MultiArray(числоСтолбцов)` | числоСтолбцов: Integer | Пустой массив с заданным числом столбцов и 0 строк. Если аргумент ≤ 0, используется значение по умолчанию 2. |
| `MultiArray(числоСтолбцов, числоСтрок)` | числоСтолбцов: Integer, числоСтрок: Integer | Массив заданных размеров. Если числоСтолбцов ≤ 0 — используется 2. При числоСтрок ≤ 0 строк нет. |

Примеры создания в языке EME-L:

```EME-L
'Без аргументов — 2 столбца, 0 строк (значение N по умолчанию)'
arr = Object("MultiArray");

'Имена столбцов — три строковых аргумента задают три столбца'
arr = Object("MultiArray", "code", "name", "qty");

'Один числовой аргумент — число столбцов'
arr = Object("MultiArray", 3);

'Два числовых аргумента — число столбцов и число строк'
arr = Object("MultiArray", 3, 5);
```

### Имена столбцов — SetColumnNames

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `SetColumnNames(имя1[, ...[, имяN]])` | имена: String/Integer/Real | Empty | Задаёт имена столбцов. Количество аргументов должно точно совпадать с числом столбцов массива. После этого к элементам можно обращаться по имени столбца. |

Метод `SetColumnNames` в языке EME-L выбрасывает исключение при: несовпадении количества имён и числа столбцов; дублировании имён столбцов; пустой строке в качестве имени; передаче аргумента в типе, отличном от String, Integer и Real.

### Доступ к элементам по индексам — GetAt, SetAt, SetAtGrow, Lookup

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetAt(индексСтроки, индексСтолбца)` | строка: Integer, столбец: Integer или String (имя) | Любой тип | Значение элемента. Выбрасывает исключение при выходе индексов за пределы, пустом массиве или несуществующем имени столбца. |
| `SetAt(индексСтроки, индексСтолбца, значение)` | строка: Integer, столбец: Integer/String, значение: Любой тип | Empty | Присваивает значение элементу. Исключения те же, что у `GetAt`. |
| `SetAtGrow(индексСтроки, индексСтолбца, значение)` | строка: Integer, столбец: Integer/String, значение: Любой тип | Empty | Присваивает значение элементу; при выходе индекса строки за пределы — автоматически расширяет массив, заполняя новые элементы Empty. |
| `Lookup(индексСтроки, индексСтолбца)` | строка: Integer, столбец: Integer/String | Любой тип или Empty | Безопасное чтение: при выходе индексов за пределы возвращает Empty без исключения. Выбрасывает исключение только при несуществующем имени столбца. |
| `Lookup(индексСтроки, индексСтолбца, значениеПоУмолчанию)` | + значениеПоУмолчанию: Любой тип | Любой тип | То же, но при выходе за границы возвращает `значениеПоУмолчанию`. |

`SetAtGrow` в языке EME-L выбрасывает исключение при отрицательном индексе строки, выходе индекса столбца за размерность или несуществующем имени столбца. Метод `Lookup` — единственный из группы, который не выбрасывает исключение при выходе индексов за границы: он возвращает Empty или заданное значение по умолчанию.

### Доступ через текущую строку — Get, Put

Методы работы с текущей строкой удобны при переборе массива или в контексте, где строка уже установлена навигационными методами.

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Get(индексСтолбца)` | столбец: Integer/String | Любой тип | Значение элемента в текущей строке. |
| `Put(индексСтолбца, значение)` | столбец: Integer/String, значение: Любой тип | Empty | Присваивает значение элементу текущей строки. |

Методы `Get` и `Put` в языке EME-L выбрасывают исключение при выходе индекса столбца за пределы, пустом массиве, недопустимой текущей строке или несуществующем имени столбца.

### Навигация по строкам — GetLine, SetLine, SetFirstLine, SetNextLine, IsValidLine, DeleteLine

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetLine()` | — | Integer | Номер текущей строки. Если текущая строка не установлена — возвращает -1. |
| `SetLine(номерСтроки)` | номерСтроки: Integer | Empty | Устанавливает текущую строку. |
| `SetFirstLine()` | — | Empty | Устанавливает текущей первую строку. |
| `SetNextLine()` | — | Empty | Переход к следующей строке. Обычно используется совместно с `SetFirstLine` и `IsValidLine` для перебора. |
| `IsValidLine()` | — | Boolean | TRUE — текущая строка установлена и в пределах массива; FALSE — не установлена или вышла за пределы. |
| `IsValidLine(номерСтроки)` | номерСтроки: Integer | Boolean | TRUE — строка с указанным номером существует. |
| `DeleteLine()` | — | Empty | Удаляет текущую строку. Текущая строка должна быть установлена. |

Для перебора строк в языке EME-L предпочтительно использовать цикл `Loop (arr)` — он автоматически вызывает `SetFirstLine()`, `IsValidLine()` и `SetNextLine()`, избавляя от ручного управления текущей строкой. Метод `DeleteLine` выбрасывает исключение при недопустимой текущей строке или пустом массиве.

### Размер и размерность — GetSize, GetDimension, IsEmpty, SetSize

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `GetSize()` | — | Integer | Текущее количество строк. |
| `GetDimension()` | — | Integer | Количество столбцов (размерность по второму измерению). |
| `IsEmpty()` | — | Boolean | TRUE — массив пуст (0 строк); FALSE — содержит хотя бы одну строку. |
| `SetSize(числоСтрок)` | числоСтрок: Integer | Empty | Задаёт количество строк. При увеличении новые элементы — Empty; при уменьшении лишние строки удаляются с конца. Число столбцов не меняется. |
| `SetSize(текущееЧислоСтрок, расширяемоДо)` | текущееЧислоСтрок: Integer, расширяемоДо: Integer | Empty | То же, но заранее резервирует память под `расширяемоДо` строк — ускоряет последующее расширение массива. |

### Добавление, вставка и удаление строк — Add, InsertAt, AddArray, Copy, Remove, RemoveAt, RemoveAll

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Add([значение1[, ...[, значениеN]]])` | значения: Любой тип | Integer | Добавляет строку в конец массива. Недостающие элементы заполняются Empty. Количество значений не может превышать число столбцов. Возвращает индекс добавленной строки и делает её текущей. |
| `InsertAt(индексСтроки[, значение1[, ...[, значениеN]]])` | индексСтроки: Integer, значения: Любой тип | Empty | Вставляет строку по индексу; существующие строки сдвигаются вниз. |
| `AddArray(массив)` | массив: MultiArray (по ссылке) | Empty | Добавляет все строки из другого MultiArray в конец текущего. Размерности должны совпадать. |
| `Copy(массив)` | массив: MultiArray (по ссылке) | Empty | Полностью перезаписывает текущий массив содержимым источника. Размерности должны совпадать. |
| `Remove()` | — | Empty | Удаляет последнюю строку. Если массив пуст — вызов игнорируется. |
| `RemoveAt(индексСтроки)` | индексСтроки: Integer | Empty | Удаляет строку по индексу. |
| `RemoveAll()` | — | Empty | Очищает массив, удаляя все строки. Размерность (число столбцов) сохраняется. |

Методы `Add` и `InsertAt` в языке EME-L выбрасывают исключение, если количество переданных значений превышает число столбцов массива. `InsertAt` также выбрасывает исключение при выходе индекса строки за пределы или пустом массиве. Методы `AddArray` и `Copy` выбрасывают исключение при несовпадении размерностей двух массивов или если аргумент не является объектом класса MultiArray.

### Поиск и отладка — Find, GetContent

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `Find(искомоеЗначение)` | искомоеЗначение: Любой тип | Integer | Индекс первого вхождения значения в первом столбце. Если не найдено — -1. |
| `Find(искомоеЗначение, индексСтолбца)` | + индексСтолбца: Integer/String | Integer | То же в указанном столбце. Выбрасывает исключение при выходе индекса столбца или несуществующем имени. |
| `GetContent()` | — | String | Строковое представление содержимого массива для отладки. Элементы разделяются `; `. |

### Примеры использования MultiArray в языке EME-L

Создание массива с именованными столбцами и заполнение:

```EME-L
'Создать массив товаров: код, наименование, количество'
arr = Object("MultiArray", "code", "name", "qty");
arr.Add("001", "Винт М6", 100);
arr.Add("002", "Гайка М6", 200);
arr.Add("003", "Шайба М6", 150);
'arr содержит 3 строки по 3 столбца'
```

Перебор строк и безопасное чтение:

```EME-L
arr.SetFirstLine();
While (arr.IsValidLine())
    csName = arr.Get("name");
    nQty = arr.Get("qty");
    'обработать csName и nQty'
    arr.SetNextLine();
End While;
```
