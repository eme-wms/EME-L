# Класс Hew — переменная выделения (битовый буфер) строк записей базы данных

Класс `Hew` в языке EME-L — класс для работы с переменной выделения (битовым буфером) строк записей базы данных. Внутреннее представление — битовый массив, где каждый бит соответствует одной строке записи; установленный бит означает, что строка выделена. C++-имя класса — `CIHew`.

Переменная выделения в системе EME.WMS — это тип `VARIABLE_HEW`, привязанный к переменной диалога, записи базы данных или фильтру строк. Класс `Hew` инкапсулирует операции над этим битовым буфером: установку и снятие пометок, проверку состояния, навигацию по выделенным строкам, инверсию, очистку и подсчёт.

Поддерживаемые сценарии в языке EME-L:

- Привязка к переменной выделения диалога, записи базы данных, фильтру строк или другому объекту Hew.
- Установка и снятие пометок на строках, проверка помечена ли конкретная строка.
- Навигация по выделенным строкам в прямом порядке — две параллельные системы итераторов: по прямым номерам строк битового буфера и по номерам строк записи базы данных.
- Сериализация состояния выделения в строку Base64 и восстановление из строки.
- Заполнение выделения из массива ссылок или объекта Array.
- Применение EME-L фильтров для инициализации выделения по алгоритму класса.
- Сравнение и слияние с другими объектами Hew.
- Управление автоматическим удалением битового буфера при уничтожении объекта.
- Простановка флага `BROWSE_BIT_SELECT_MARK` для переменных диалога, чтобы браузер отображал выбор из переменной выделения.

## Создание объекта — конструкторы класса Hew

Объект `Hew` создаётся через `Object()` в языке EME-L. Допустимы две формы вызова.

```EME-L
'Конструктор без аргументов — создание несвязанного объекта Hew'
objHew = Object("Hew");
```

Несвязанный объект не имеет внутреннего битового буфера. Для инициализации используйте методы `AttachToRecord`, `Attach`, `AttachToLineFilter`, `InitHew`, `InitFromArrayOfRefs` или `PutRefArray` — любой из них создаёт или подключает буфер.

```EME-L
'Конструктор с привязкой к переменной выделения диалога в языке EME-L'
objHew = Object("Hew", Dialog, "Заказы");
```

Вторая форма принимает объект `Dialog` (первый аргумент) и имя переменной выделения в диалоге (второй аргумент, строка). Конструктор эквивалентен последовательности `Object("Hew")` и `Attach(Dialog, "Заказы")`: объект сразу привязывается к существующей переменной выделения указанного диалога.

Один аргумент или более двух аргументов в `Object("Hew", ...)` не поддерживаются — это вызовет ошибку времени выполнения.

## Операции с пометками строк — битовый буфер класса Hew

Каждая строка в переменной выделения представлена битом в битовом буфере: установленный бит — строка выделена, сброшенный бит — не выделена. Адресация битов ведётся по номеру строки (0-based).

| Метод | Аргументы | Возвращает | Описание |
|-------|-----------|------------|----------|
| `MarkLine(arg0)` | arg0: Integer (номер строки) | Empty | Устанавливает пометку (бит) на строке в битовом буфере с проверкой корректности номера строки. |
| `UnmarkLine(arg0)` | arg0: Integer | Empty | Снимает пометку (бит) с указанной строки с проверкой корректности номера. |
| `IsMarkedLine(arg0)` | arg0: Integer | Boolean | TRUE — строка помечена; FALSE — строка не помечена или номер строки некорректен. |
| `IsCorrectLine(arg0)` | arg0: Integer | Boolean | TRUE — номер строки находится в допустимом диапазоне от 0 до максимального номера строки переменной выделения. |
| `UnmarkAll()` | — | Empty | Снимает пометки со всех строк битового буфера, оставляя сам буфер и привязку к записи неизменными. |
| `RemoveAll()` | — | Empty | Синоним `UnmarkAll` — снимает пометки со всех строк. |
| `Add(arg0)` | arg0: Integer | Empty | Устанавливает пометку на строке **без проверки корректности номера** — быстрее, но требует аккуратного использования. Не используйте для номеров за пределами допустимого диапазона. |
| `Remove(arg0)` | arg0: Integer | Empty | Снимает пометку со строки **без проверки корректности номера** — быстрее `UnmarkLine`, но без проверки границ. |
| `InvertAll()` | — | Boolean | Инвертирует все биты битового буфера: выделенные строки становятся невыделенными и наоборот. FALSE — объект Hew не инициализирован. |
| `GetSize()` | — | Integer | Возвращает количество выделенных (помеченных) строк в текущей переменной выделения. 0 — объект не инициализирован. |

| `IsNothingMarked()` | — | Boolean | TRUE — ни одна строка не помечена (инверсия `IsAnyMarked`). |

## Навигация по выделенным строкам — две системы итераторов класса Hew

У объекта Hew есть две **параллельные системы навигации** по выделенным строкам. Каждая хранит собственный итератор.

1. **По прямым номерам строк битового буфера** — методы без суффикса `Rec` (`SetFirstLine`, `SetNextLine`, `IsValidLine`, `GetLine`, `SetLine`).
2. **По номерам строк записи базы данных** — методы с суффиксом `Rec` (`SetFirstLineRec`, `SetNextLineRec`, `IsValidLineRec`, `GetLineRec`, `SetLineRec`).

Итераторы в языке EME-L независимы друг от друга: перемещение по одной системе не влияет на позицию итератора другой.

| `SetFirstLine()` | — | Empty | Устанавливает внутренний итератор на первую выделенную строку в контексте прямых номеров строк битового буфера. |
| `SetNextLine()` | — | Empty | Перемещает внутренний итератор на следующую выделенную строку. |
| `IsValidLine()` | — | Boolean | TRUE — текущая позиция итератора корректна (указывает на выделенную строку); FALSE — достигнут конец набора выделенных строк. |
| `GetLine()` | — | Integer | Номер текущей выделенной строки битового буфера; 0, если итератор не установлен или некорректен. |
| `SetLine(arg0)` | arg0: Integer (номер строки) | Empty | Устанавливает итератор на указанную строку. Если указанная строка не выделена — итератор перемещается на следующую выделенную строку после указанной. |

## Примеры использования класса Hew в языке EME-L

Создание объекта Hew с привязкой к переменной диалога и снятие всех пометок в системе EME.WMS:

```EME-L
'Привязка объекта Hew к переменной выделения «Заказы» диалога'
OrdersHew = Object("Hew", Dialog, "Заказы");
OrdersHew.UnmarkAll();
```

Инициализация выделения перебором ссылок и пометка строк:

```EME-L
'Создание объекта Hew и привязка к записи складов в языке EME-L'
WarehousesFilter = Object("Hew");
WarehousesFilter.AttachToRecord(r_Warehouses.GetRecord());

Prefix = "WarehouseRef";
Index = 0;
Ref = DialogParameters.GetLongParam(is_format("Filter.%s.%d", Prefix, Index), NULL_REF);
While (Ref >= 0)
    WarehousesFilter.MarkLine(Ref);
    Index = Index + 1;
    Ref = DialogParameters.GetLongParam(is_format("Filter.%s.%d", Prefix, Index), NULL_REF);
End While
```

Перебор выделенных строк с использованием итератора прямых номеров строк:

```EME-L
'Навигация по выделенным строкам битового буфера в языке EME-L'
objHew.SetFirstLine();
While (objHew.IsValidLine())
    LineNo = objHew.GetLine();
    'LineNo — номер выделенной строки битового буфера'
    objHew.SetNextLine();
End While
```
